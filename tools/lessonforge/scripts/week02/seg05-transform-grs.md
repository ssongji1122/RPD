## TTS 나레이션 스크립트

### [인트로 — 0:00]
이번 영상에서는 오브젝트를 직접 움직이고, 회전하고, 크기를 바꾸는
**Transform 기초**를 배웁니다.
단축키 세 개만 기억하면 됩니다. **G, R, S**. 이 세 글자예요.
G는 Grab(이동), R은 Rotate(회전), S는 Scale(크기).

### [선택 확인 — 0:30]
먼저 큐브를 **좌클릭**으로 선택합니다.
선택된 오브젝트는 주황색 테두리가 생깁니다.
블렌더에서 무언가를 조작하려면 항상 먼저 선택해야 해요.

### [G: Grab 이동 — 1:00]
**G키**를 누르면 오브젝트가 마우스에 붙어서 따라다닙니다.
마우스를 움직여서 원하는 위치로 이동한 뒤 **좌클릭**으로 확정하면 됩니다.
잘못 됐다면 **우클릭이나 ESC**로 취소할 수 있어요.

G를 누른 뒤 마우스를 움직여보세요. 큐브가 붙어서 움직이죠?
확정하려면 클릭, 취소하려면 우클릭이에요.

### [축 제한 — 2:00]
그런데 G를 누르면 모든 방향으로 자유롭게 이동해서, 정확한 위치에 놓기가 어려울 수 있어요.
이럴 때 **축 제한**을 씁니다.

- **G + X**: X축 방향으로만 이동
- **G + Y**: Y축 방향으로만 이동
- **G + Z**: Z축 방향으로만 이동 (위아래)

한번 해볼게요. G를 누르고, 바로 X를 누릅니다.
보이시죠? 이제 X축 방향으로만 움직입니다. 좌우로만 움직이는 거예요.
클릭으로 확정합니다.

### [수치 입력 — 3:00]
더 정확하게는 숫자를 직접 입력할 수도 있어요.
**G + X + 2 + Enter** 를 해보세요.
X축으로 정확히 2 유닛(미터) 이동됩니다.
즉, "G → 축 → 숫자 → Enter" 순서예요.

### [R: Rotate 회전 — 4:00]
다음은 **R키** 입니다. 회전이에요.
R을 누르면 오브젝트가 마우스 위치에 따라 회전합니다.
클릭으로 확정, 우클릭으로 취소.

축 제한도 동일하게 적용됩니다.
**R + Z + 45 + Enter** → Z축 기준 45도 회전.
R + X, R + Y도 마찬가지예요.

### [S: Scale 크기 — 5:30]
마지막으로 **S키** 는 스케일, 즉 크기입니다.
S를 누르면 오브젝트가 커지거나 작아집니다. 마우스를 가운데서 멀리 하면 커지고, 가까이 하면 작아져요.

**S + 0.5 + Enter** → 절반 크기로 축소
**S + X + 2 + Enter** → X축 방향으로만 2배 확대

이렇게 특정 축으로만 스케일을 조정하면 오브젝트를 납작하게 만들거나 늘릴 수 있어요.
로봇 몸체를 만들 때 많이 쓸 거예요.

### [Shift 축 제한 — 7:00]
팁 하나 더 드릴게요.
**Shift + 축키**를 누르면 그 축을 **제외한** 나머지 방향으로 이동합니다.
예를 들어 **G + Shift + Z**는 Z축을 제외한 XY 평면에서만 이동해요.
즉, 오브젝트를 바닥 위에서만 미끄러지듯 옮길 때 유용합니다.

### [종합 연습 — 7:30]
이제 직접 해볼게요.
Shift + A로 구체와 원기둥을 추가해서 세 가지 오브젝트를 배치해봅시다.
각각을 선택하고, G·R·S와 축 제한을 써서 원하는 위치에 놓아보세요.
일직선으로 배치해보거나, 피라미드처럼 쌓아보거나 자유롭게 해보세요.

### [취소·되돌리기 — 8:30]
잘못 됐을 때:
- **ESC 또는 우클릭**: Transform 중 취소 (오브젝트 원래 위치로)
- **Ctrl + Z**: 되돌리기 (언제든 쓸 수 있어요)
블렌더는 실행 취소 횟수가 넉넉하니까 마음껏 시험해보세요.

### [정리 — 9:00]
정리:
- **G** → 이동 (Grab)
- **R** → 회전 (Rotate)
- **S** → 크기 (Scale)
- **+ X/Y/Z** → 축 제한
- **+ 숫자 + Enter** → 정확한 수치 입력
- **+ Shift + 축** → 해당 축 제외 이동

### [아웃트로 — 9:30]
다음 영상은 마지막입니다.
**Apply Transform**과 **Origin 설정**이에요.
이게 왜 중요한지, 지금 당장은 모를 수 있지만 나중에 반드시 필요해집니다.
꼭 보고 넘어가세요.
