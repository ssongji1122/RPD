## TTS 나레이션 스크립트

### [인트로 — 0:00]
안녕하세요, 로봇 프러덕트 디자인 수업입니다.
이번 영상에서는 Blender 5.0을 설치하고, 처음으로 실행해보겠습니다.
설치는 딱 한 번만 하면 되니까, 천천히 따라와 주세요.

### [다운로드 — 0:20]
먼저 브라우저를 열고 주소창에 **blender.org** 를 입력합니다.
공식 사이트에서 상단 Download 메뉴로 들어가세요.
Blender는 완전 무료이고, 광고도 없습니다. 오픈소스 소프트웨어예요.

지금 보이는 버전이 **Blender 5.0** 입니다.
항상 페이지에서 가장 위에 있는 최신 안정 버전을 받으면 됩니다.

### [OS별 다운로드 — 1:00]
여러분의 운영체제에 맞는 버튼을 클릭하세요.
맥을 쓰신다면 macOS, 윈도우를 쓰신다면 Windows를 선택합니다.
Apple Silicon, 즉 M1 M2 M3 맥이라면 **macOS Apple Silicon** 버전을 받아야 합니다. 인텔 맥이라면 **macOS Intel** 버전으로요.

파일 크기가 약 250메가바이트 정도입니다. 학교 와이파이로 1~2분이면 완료됩니다.

### [설치 — Mac, 2:30]
다운로드가 끝나면 .dmg 파일을 더블클릭해서 마운트합니다.
나타나는 창에서 Blender 아이콘을 Applications 폴더로 드래그하면 설치 완료입니다.
딱 이것만 하면 돼요. 아주 간단합니다.

### [설치 — Windows, 3:00]
윈도우는 .msi 설치파일을 실행하고,
Next → Next → Install → Finish 순서로 진행합니다.
기본 설정을 바꿀 필요는 없습니다. 그냥 계속 클릭하면 됩니다.

### [첫 실행 — 4:00]
이제 Blender를 실행해볼게요.
처음 실행하면 이런 화면이 나타납니다. 이걸 **Splash Screen**이라고 해요.
버전 정보, 최근 파일, 그리고 새 파일 템플릿을 선택하는 화면입니다.

여기서 **General** 을 클릭하세요.
그러면 기본 씬이 열립니다. 가운데 정육면체, 카메라, 조명이 있는 이 화면이 Blender의 기본 상태예요.

### [첫 화면 — 5:00]
이제 블렌더가 열렸습니다.
처음 보면 복잡해 보이지만, 걱정하지 마세요.
다음 영상에서 이 화면 하나하나를 천천히 설명드릴게요.

지금은 그냥 "이게 블렌더구나" 하고 느껴보세요.
여러분이 이 프로그램으로 로봇을 만들 거예요.

### [아웃트로 — 5:30]
설치가 완료됐다면 다음 영상으로 넘어가세요.
다음 영상에서는 **기본 환경 설정** 을 잡아줄 거예요.
노트북 사용자에게 꼭 필요한 설정이 있으니 꼭 봐주세요.
