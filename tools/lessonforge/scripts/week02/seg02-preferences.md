## TTS 나레이션 스크립트

### [인트로 — 0:00]
이번 영상에서는 Blender를 제대로 쓰기 위한 **기본 환경 설정**을 잡겠습니다.
설치 후 이 설정을 한 번만 해두면, 수업 내내 훨씬 편하게 작업할 수 있어요.
특히 노트북 사용자라면 이 영상이 필수입니다.

### [Preferences 열기 — 0:30]
상단 메뉴에서 **Edit → Preferences** 를 클릭합니다.
단축키는 없고, 메뉴에서 직접 열어야 해요.
이 창이 Blender의 환경설정 패널입니다.

### [GPU 설정 — 1:00]
먼저 **System** 탭으로 이동합니다.
여기서 가장 중요한 설정이 **GPU Compute** 예요.

왜 중요하냐면, Blender의 렌더링은 CPU로도 할 수 있지만 GPU로 하면 10배에서 50배까지 빠릅니다.
여러분의 노트북이나 PC에 GPU가 있다면, 반드시 여기서 활성화해야 해요.

- **NVIDIA GPU** 사용자: CUDA 또는 OptiX 항목에서 GPU 체크
- **Apple Silicon (M1·M2·M3)** 사용자: Metal 항목에서 GPU 체크
- **Intel 내장 그래픽만** 있는 경우: 해당 항목이 없거나 비어 있을 수 있습니다. 그래도 괜찮아요.

설정하고 나면 렌더링 시 GPU가 사용됩니다. 이건 나중에 렌더링 영상에서 또 확인할게요.

### [Emulate Numpad — 2:30]
다음으로 **Input** 탭을 클릭합니다.
여기서 **Emulate Numpad** 를 체크하세요.

이게 왜 필요하냐면, Blender는 뷰포트 회전에 키보드 오른쪽 숫자패드(Numpad)를 많이 쓰거든요.
그런데 노트북에는 Numpad가 없잖아요.
이 옵션을 체크하면, 키보드 상단의 숫자 키 1·2·3·4·7이 Numpad처럼 작동합니다.

노트북이 아닌 데스크탑 사용자도 체크해두면 편합니다.

### [Emulate 3 Button Mouse — 3:30]
바로 아래에 **Emulate 3 Button Mouse** 옵션도 보입니다.
마우스 가운데 버튼, 즉 휠 클릭이 없는 분들은 이걸 체크하세요.
그러면 Alt + 좌클릭으로 뷰포트를 회전할 수 있어요.

마우스 휠 클릭이 잘 된다면 굳이 체크 안 해도 됩니다.

### [Interface — 4:30]
**Interface** 탭으로 이동합니다.
고해상도 모니터(Retina·4K)를 쓰신다면 **Resolution Scale** 을 1.0에서 1.2나 1.5로 올려보세요.
아이콘과 텍스트가 더 크게 보입니다.
일반 1080p 모니터라면 기본값 1.0으로 두면 됩니다.

### [저장 — 5:30]
설정을 다 했으면 **Save Preferences** 버튼을 꼭 클릭하세요.
왼쪽 하단에 있습니다. 이걸 눌러야 다음에 블렌더를 다시 열어도 설정이 유지됩니다.
깜빡하기 쉬우니 꼭 저장해주세요.

### [정리 — 6:00]
오늘 설정한 내용을 정리하면:
- GPU 활성화 → 렌더링 속도 향상
- Emulate Numpad → 노트북에서 뷰포트 조작 가능
- Emulate 3 Button Mouse → 2버튼 마우스 대응
- Resolution Scale → 모니터에 맞게 UI 크기 조정
- Save Preferences → 설정 영구 저장

### [아웃트로 — 7:00]
이 설정은 딱 한 번만 하면 됩니다.
다음 영상에서는 드디어 Blender 화면 구조를 하나씩 살펴볼게요.
각 영역이 어떤 역할을 하는지 이해하면, 나머지 수업이 훨씬 쉬워집니다.
