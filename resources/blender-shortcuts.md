## Navigation (뷰 조작)

| 단축키 | 기능 | 설명 |
|--------|------|------|
| `Middle Mouse` | Orbit | 뷰 회전 |
| `Shift + Middle Mouse` | Pan | 뷰 이동 (Pan) |
| `Scroll` | Zoom | 줌 인/아웃 |
| `Numpad 1 / 3 / 7` | Front / Right / Top View | Front/Right/Top 뷰 |
| `Numpad 5` | Perspective / Orthographic Toggle | 원근/직교 전환 |
| `Numpad 0` | Camera View | 카메라 뷰 |

| `Home` | View All | 전체 보기 |

## Blender 5.0 주요 변경사항

Blender 5.0에서 기존 버전 대비 달라진 사항을 정리합니다.

| 변경 항목 | 설명 |
|-----------|------|
| Collection 가시성 단축키 이동 | `1`, `2`, `3` 등의 Collection 가시성 단축키가 3D Viewport에서 Outliner로 이동 |
| 메뉴/팝업 동작 변경 | 메뉴/팝업이 클릭할 때까지 열린 상태 유지 (이전 버전: 자동 닫힘) |
| Workspace 통합 | Compositing/Rendering Workspace 제거, General Workspace로 통합 |
| 탭 UI 변경 | 탭이 pill 형태로 변경 (기능은 동일) |
