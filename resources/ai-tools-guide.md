| 항목 | 내용 |

## 2. AI 3D 모델 생성

### Meshy AI

| 항목 | 내용 |
|------|------|
| **개요** | 텍스트 또는 이미지로부터 3D 모델을 생성하는 AI 도구. 현존하는 AI 3D 생성 도구 중 가장 빠른 속도를 제공한다. |
| **접속 방법** | [https://meshy.ai](https://meshy.ai) |
| **무료 범위** | 월간 무료 크레딧이 제공된다. 무료 크레딧으로 기본적인 3D 모델 생성이 가능하며, 크레딧 소진 시 다음 달에 갱신된다. |
| **수업 활용** | Week 05 - AI 3D 모델 생성 실습의 주요 도구로 사용. 텍스트 프롬프트 또는 컨셉 이미지를 입력하여 3D 모델 초안을 빠르게 확보한다. |
| **주의사항** | 출력 포맷은 GLB, FBX, OBJ를 지원하며, Blender로 임포트하여 후처리 작업을 진행한다. 무료 크레딧을 아껴 사용하기 위해 프롬프트를 충분히 다듬은 후 생성할 것. 생성된 모델의 토폴로지가 불규칙할 수 있으므로 Blender에서 리토폴로지가 필요할 수 있다. |

## 7. 자동 리깅

### Mixamo

| 항목 | 내용 |
|------|------|
| **개요** | Adobe에서 제공하는 자동 리깅 및 애니메이션 라이브러리 서비스. 3D 캐릭터 모델을 업로드하면 자동으로 리깅(뼈대 설정)을 수행하고, 수천 개의 모션 캡처 애니메이션을 적용할 수 있다. |
| **접속 방법** | [https://mixamo.com](https://mixamo.com) |
| **무료 범위** | 완전 무료. Adobe 무료 계정만 생성하면 모든 리깅 기능과 애니메이션 라이브러리를 제한 없이 사용할 수 있다. |
| **수업 활용** | Week 12 - 자동 리깅 실습. Blender에서 제작하거나 AI로 생성한 캐릭터 모델을 Mixamo에 업로드하여 자동 리깅을 수행하고, 원하는 애니메이션을 적용한 후 FBX로 다운로드하여 Blender에서 임포트한다. |
| **주의사항** | 출력 포맷은 FBX이며, Blender로 임포트할 때 스케일과 회전값을 확인해야 한다. 인간형(Humanoid) 캐릭터에 최적화되어 있으므로, 비인간형 캐릭터는 리깅 품질이 낮을 수 있다. 모델 업로드 전 불필요한 메시를 정리하고, T-포즈 상태로 내보내는 것을 권장한다. |

## 8. AI 자동화

### Blender MCP + Claude

| 항목 | 내용 |
|------|------|
| **개요** | Claude AI가 Blender를 직접 제어하여 모델링, 머티리얼 설정, 조명, 렌더링 등의 작업을 자동화하는 도구. MCP(Model Context Protocol)를 통해 자연어 명령으로 Blender를 조작할 수 있다. |
| **접속 방법** | Blender MCP: [https://github.com/ahujasid/blender-mcp](https://github.com/ahujasid/blender-mcp) + Claude Desktop 앱 |
| **무료 범위** | 완전 무료. Blender MCP는 오픈소스이며, Claude Desktop도 무료로 사용할 수 있다. |
| **수업 활용** | Week 02 - Blender MCP 초기 설정 및 환경 구성. Week 05 - AI를 활용한 씬 생성 및 오브젝트 배치. Week 09 - 조명 연출 자동화. Week 13 - 렌더 설정 최적화. 학기 전반에 걸쳐 반복 작업 자동화에 활용한다. |
| **주의사항** | 초기 설정이 필요하며, 설치 방법은 `resources/blender-mcp-setup.md`를 참고할 것. Python 환경과 Node.js가 사전에 설치되어 있어야 한다. Claude Desktop과 Blender가 동시에 실행 중이어야 하며, MCP 서버가 정상적으로 연결되었는지 확인한 후 사용할 것. |
