# Week 02. Blender 인터페이스와 기초 모델링

> Blender를 처음 켜는 학생이 화면을 읽고, 오브젝트를 조작하고, 기본 모델링 도구를 직접 써보는 주차입니다.

3. Edit Mode에서 면을 선택하고 기본 모델링 도구를 적용할 수 있다.

## 오늘의 학습 흐름

1. 설치와 첫 실행
2. 인터페이스 구조 이해
3. 오브젝트 기본 조작
4. Edit Mode와 메시 편집
5. Extrude / Bevel / Loop Cut 실습

## 핵심 개념 한눈에 보기

### 1. 인터페이스

- `3D Viewport`: 오브젝트를 보고 조작하는 메인 작업 공간
- `Outliner`: 현재 씬의 오브젝트 목록을 확인하는 영역
- `Properties`: 선택한 오브젝트의 세부 설정을 조작하는 영역
- `Timeline`: 애니메이션이나 시간 흐름을 다루는 영역

### 2. 오브젝트 조작

- `G`: 이동
- `R`: 회전
- `S`: 스케일
- `G + X`, `R + Z`처럼 축을 제한하면 더 정확하게 조작할 수 있습니다.

### 3. 모드 전환

- `Object Mode`: 오브젝트 단위로 조작
- `Edit Mode`: 점, 선, 면을 직접 편집
- `1 / 2 / 3`: Vertex / Edge / Face 선택 모드 전환

### 4. 기본 모델링 도구

- `Extrude (E)`: 면을 밀어내어 새로운 형태 만들기
- `Inset (I)`: 면 안쪽에 새로운 면 만들기
- `Bevel (Ctrl + B)`: 모서리를 깎아 부드럽게 만들기
- `Loop Cut (Ctrl + R)`: 구조선을 추가해서 형태를 세밀하게 나누기

## 실습 순서

### Step 1. Blender 첫 화면 읽기

- Splash Screen과 기본 씬 구성을 확인합니다.
- 기본 씬에 포함된 `Camera`, `Cube`, `Light`를 찾습니다.
- 3D Viewport, Outliner, Properties, Timeline 위치를 확인합니다.

### Step 2. 인터페이스와 도구 익히기

- `T` 키로 Tool Shelf를 열고 주요 도구를 확인합니다.
- `N` 패널을 열어 위치, 회전, 스케일 값을 확인합니다.
- Transform Orientation, Pivot Point, Snap이 어디에 있는지 확인합니다.

### Step 3. 오브젝트 추가와 기본 조작

- `Shift + A`로 Cube, Sphere, Cylinder를 추가해봅니다.
- `G`, `R`, `S`로 오브젝트를 각각 이동, 회전, 크기 조절해봅니다.
- 축 제한과 숫자 입력으로 정밀 조작을 시도해봅니다.

### Step 4. Edit Mode 들어가기

- `Tab`으로 Edit Mode에 들어갑니다.
- `1 / 2 / 3` 키로 Vertex, Edge, Face 모드를 바꿔봅니다.
- 면 하나를 선택한 뒤 편집 대상이 오브젝트 전체가 아니라 메시 일부라는 점을 확인합니다.

### Step 5. 기본 모델링 도구 적용하기

- `E`로 Extrude를 해봅니다.
- `I`로 Inset을 적용해 안쪽 패널을 만들어봅니다.
- `Ctrl + B`로 모서리를 깎아봅니다.
- `Ctrl + R`로 구조선을 추가하고 위치를 조정해봅니다.

## 실습 중 꼭 기억할 포인트

- Object Mode와 Edit Mode를 헷갈리면 원하는 조작이 되지 않습니다.
- Extrude를 취소할 때는 `Esc` 후 중복 버텍스가 남을 수 있으니 필요하면 `Ctrl + Z`로 되돌립니다.
- 생성되는 오브젝트는 `3D Cursor` 위치를 기준으로 추가됩니다.
- Loop Cut은 메시 흐름에 따라 들어갈 수 있는 위치가 달라집니다.

## 자주 헷갈리는 질문

### 왜 오브젝트 전체가 움직이나요?

Edit Mode가 아니라 Object Mode에 있을 가능성이 큽니다.

### 왜 새 오브젝트가 이상한 위치에 생기나요?

`3D Cursor`가 원점이 아닌 다른 위치에 있을 수 있습니다.

### 왜 Loop Cut이 안 들어가나요?

현재 메시 구조상 연속된 Edge Loop를 만들기 어려운 면 구성이기 때문일 수 있습니다.
