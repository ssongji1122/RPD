# Week 01 과제: 무드보드 제작

## 학습 목표

- [ ] AI 도구(Mixboard 또는 나노바나나)를 사용하여 이미지를 생성할 수 있다
- [ ] 생성한 이미지를 조합하여 무드보드를 구성할 수 있다
- [ ] 자신만의 로봇/캐릭터 컨셉을 한 문장으로 정리할 수 있다

## 과제 내용

Mixboard 또는 나노바나나를 활용하여 자신의 로봇/캐릭터 무드보드를 제작한다.

### 제작 과정

1. **컨셉 구상:** 어떤 로봇 또는 캐릭터를 만들고 싶은지 키워드 정리
2. **AI 이미지 생성:** Mixboard 또는 나노바나나에서 다양한 프롬프트로 이미지 생성
3. **무드보드 구성:** 생성된 이미지 중 컨셉에 맞는 것을 선별하여 무드보드 완성
4. **컨셉 정리:** 캐릭터/로봇의 이름, 성격, 형태를 한줄로 요약

### 프롬프트 예시

- "cute companion robot, soft rounded body, warm pastel colors, friendly expression"
- "귀여운 미니 로봇, 동글동글한 형태, 파스텔 톤, 3D 렌더링 스타일"
- "retro-futuristic robot toy, vinyl figure style, collectible design"

## 제출 방법

- **제출처:** Discord #week01-assignment 채널
- **형식:** 이미지 2장 + 캐릭터/로봇 컨셉 한줄 설명
- **기한:** 다음 수업 전까지

## 평가 기준

| 항목 | 비율 | 세부 기준 |
|------|------|-----------|
| 창의성 | 40% | 독창적인 캐릭터/로봇 컨셉, 차별화된 아이디어 |
| AI 도구 활용 | 30% | 프롬프트 작성 능력, 도구의 효과적 사용 |
| 컨셉 명확성 | 30% | 캐릭터/로봇의 방향성이 명확하게 전달되는가 |
