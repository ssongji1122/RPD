# Week 02 과제: 기본 도형 배치 + MCP 테스트

## 과제 개요

Blender의 기본 뷰 조작과 Transform 기능을 활용하여 도형을 배치하고, Blender MCP 연결을 성공적으로 테스트합니다.

## 제출 방법

- **제출처:** 본인 학생 페이지
- **형식:** 스크린샷 2장 + 한줄 코멘트
- **기한:** 다음 수업 전까지

## 과제 내용

### 스크린샷 1: 기본 도형 배치

- 기본 도형(Cube, Sphere, Cylinder, Cone, Torus 등)을 **5개 이상** 사용
- 자유롭게 배치하여 하나의 씬을 구성
- Transform(이동, 회전, 스케일)을 활용하여 다양한 크기와 위치로 배치
- Shading 모드는 Solid 또는 Material Preview 권장

### 스크린샷 2: MCP 연결 테스트

- Blender MCP를 통해 Claude에게 명령을 전달한 결과 스크린샷
- 예시: "Create a red cube and a blue sphere" 등의 명령 실행 결과
- Claude의 응답과 Blender의 결과가 함께 보이도록 캡처

### 한줄 코멘트

- 작업 과정에서 느낀 점이나 어려웠던 부분을 한줄로 작성

## 평가 기준

| 항목 | 비율 | 설명 |
|------|------|------|
| 도구 활용 | 40% | Transform, 뷰 조작, MCP 연결을 올바르게 사용했는가 |
| 완성도 | 30% | 도형 배치가 의도를 가지고 구성되었는가 |
| 창의성 | 30% | 단순 배치를 넘어 재미있거나 독창적인 구성을 시도했는가 |

## 참고 사항

- Apply Transform(Ctrl + A)을 사용하여 Transform 값을 정리하는 것을 권장
- MCP 연결이 안 되는 경우, 콘솔 로그를 확인하고 수업 중 질문
