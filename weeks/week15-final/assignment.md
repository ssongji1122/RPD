# Week 15 기말 프로젝트: 캐릭터 로봇 완성 + 발표

## 학습 목표

- [ ] 한 학기 동안 배운 모든 기법을 종합하여 완성된 결과물을 제출할 수 있다
- [ ] 전체 제작 프로세스를 체계적으로 발표할 수 있다
- [ ] 다른 학생의 작품에 건설적인 피어 리뷰를 제공할 수 있다

## 과제 내용

Week 01에서 기획한 컨셉을 바탕으로, 한 학기 동안 배운 모델링, 텍스처, 리깅, 애니메이션, 조명, 렌더링, AI 도구 활용 기법을 총동원하여 완성된 캐릭터 로봇 프로덕트를 제출하고 발표합니다.

### 제출물

#### 필수

1. **렌더 이미지 5장 이상** - 다양한 조명/각도 (PNG 또는 JPG, 최소 1920x1080)
2. **.blend 파일** - 파일명: `final_학번_이름.blend`, Pack Resources 완료
3. **애니메이션 영상** - 5~10초, 캐릭터 움직임 (MP4 형식)

#### 선택 (가산점)

4. **프로모션 영상** - 30초~1분, AI 영상(Kling AI, Veo 등) + BGM(Suno AI 등) 활용 (MP4 형식)

### 제작 과정

1. **모델링 완성:** Edit Mode 도구 + Modifier 활용 (Week 03~05 기법)
2. **텍스처/재질 적용:** Principled BSDF, UV Unwrap, Image Texture (Week 06~07 기법)
3. **리깅:** Armature, Weight Painting, IK/FK 또는 Mixamo (Week 11~12 기법)
4. **애니메이션:** Keyframe, Graph Editor, NLA Editor (Week 10~12 기법)
5. **조명/카메라:** 3-Point Lighting, HDRI, Camera 설정 (Week 09 기법)
6. **렌더링:** Eevee 또는 Cycles, 색상 관리 (Week 13 기법)
7. **(선택) AI 영상/사운드:** Kling AI, Suno AI 등 활용 (Week 13 기법)
8. **Pack Resources:** File > External Data > Pack Resources 실행
9. **발표 자료 준비:** PPT, PDF, 또는 화면 공유 구성

## 제출 방법

- **제출처:** 본인 학생 페이지
- **형식:** 렌더 이미지 5장 + .blend 파일 + 애니메이션 MP4 + (선택) 프로모션 MP4
- **기한:** 발표일까지 (수업 시작 전 학생 페이지 업로드 필수)
- **발표:** 7분 발표 + 3분 Q&A (수업시간 발표 필수)

## 평가 기준

| 항목 | 비율 | 세부 기준 |
|------|------|-----------|
| 모델링 | 15% | 형태 완성도, Topology 품질, Modifier 활용도, 디테일 |
| 텍스처/재질 | 15% | Material 구성, UV 품질, 텍스처 매핑 정확도 |
| 리깅 | 10% | Armature 구조, Weight Painting 품질, 포즈 자연스러움 |
| 애니메이션 | 15% | 동작 자연스러움, 타이밍, 캐릭터성 표현 |
| 렌더링/조명 | 15% | 조명 구성, 렌더 품질, 카메라 구도, 색상 관리 |
| AI 도구 활용 | 10% | AI 도구의 효과적 활용, 워크플로우 통합 |
| 창의성 | 10% | 컨셉 독창성, 디자인 개성, 스토리텔링 |
| 발표 | 10% | 발표 구성, 과정 설명, Q&A 대응 |

> **배점:** 전체 성적의 **35%**
