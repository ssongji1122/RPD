# Week 14 과제: 최종 프로젝트 중간 체크

## 학습 목표

- [ ] 현재 프로젝트 진행 상황을 객관적으로 파악할 수 있다
- [ ] 남은 1주간의 작업 계획을 구체적으로 수립할 수 있다
- [ ] 기술적 문제를 인식하고 해결 방안을 모색할 수 있다

## 과제 내용

최종 프로젝트의 현재 진행 상황을 공유하고, 남은 기간 동안의 작업 계획을 구체적으로 제출합니다.

### 제출물 1: 진행 상황 스크린샷

아래 항목별로 현재 상태를 보여주는 스크린샷을 제출합니다. 아직 작업하지 못한 항목은 "미착수"로 표시합니다.

1. **모델링 상태** - Solid 모드에서 전체 모델 (최소 1장)
2. **텍스처/재질 상태** - Material Preview 또는 Rendered 모드 (최소 1장)
3. **리깅 상태** - Pose Mode에서 포즈를 취한 모습 (해당 시)
4. **애니메이션 상태** - Timeline 스크린샷 또는 애니메이션 미리보기 캡처 (해당 시)

### 제출물 2: 남은 작업 계획

아래 형식으로 남은 1주간의 작업 계획을 작성합니다.

```
[완료된 작업]
- (예: 모델링 100%, 텍스처 80%, 리깅 50%)

[이번 주 목표]
- (예: 리깅 완성, 애니메이션 5초 완성, 렌더 이미지 5장)

[일별 계획]
- Day 1~2: (구체적 작업 내용)
- Day 3~4: (구체적 작업 내용)
- Day 5~6: (구체적 작업 내용)
- Day 7: (최종 점검 및 발표 준비)

[현재 기술적 문제]
- (예: Weight Painting이 어깨 부분에서 깨짐, 해결 방안: __)
```

### 제작 과정

1. 체크리스트(lecture-note.md 참고)를 기반으로 현재 완성도 자가 점검
2. 미완성 항목을 파악하고 우선순위 설정
3. 기술적 문제가 있다면 FAQ(lecture-note.md 참고) 확인 또는 수업 중 질문
4. 스크린샷 촬영 및 작업 계획 작성
5. 본인 학생 페이지에 업로드

## 제출 방법

- **제출처:** 본인 학생 페이지
- **형식:** 스크린샷 (최소 2장) + 남은 작업 계획 (텍스트)
- **기한:** 다음 수업 전까지 (Week 15 발표일 전)

## 평가 기준

| 항목 | 비율 | 세부 기준 |
|------|------|-----------|
| 진행도 | 50% | 모델링, 텍스처, 리깅, 애니메이션 등 전체 진행 상황이 적절한 수준인가 |
| 계획 구체성 | 30% | 남은 작업 계획이 일별로 구체적이고 실현 가능한가 |
| 문제 인식 | 20% | 현재 기술적 문제를 정확히 파악하고 해결 방안을 모색하고 있는가 |
