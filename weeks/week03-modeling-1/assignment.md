# Week 03 과제: 로봇/캐릭터 기본 형태

## 과제 개요

Edit Mode의 핵심 도구(Extrude, Loop Cut, Inset, Bevel)를 활용하여 기본 도형에서 로봇 또는 캐릭터의 기본 형태를 만듭니다.

## 제출 방법

- **제출처:** Discord #week03-assignment 채널
- **형식:** 스크린샷 3장 + 한줄 코멘트
- **기한:** 다음 수업 전까지

## 과제 내용

### 스크린샷 3장

1. **Front 뷰 (Numpad 1):** 정면에서 본 로봇/캐릭터 형태
2. **Side 뷰 (Numpad 3):** 측면에서 본 로봇/캐릭터 형태
3. **Perspective 뷰:** 자유로운 각도에서 전체 형태를 보여주는 뷰

### 필수 요구사항

- Cube 또는 다른 Primitive에서 시작하여 Edit Mode로 형태를 만들 것
- 최소 3가지 이상의 도구를 사용할 것 (Extrude, Loop Cut, Inset, Bevel 중)
- 머리, 몸통, 팔, 다리 등 기본적인 신체 구조가 표현될 것

### 한줄 코멘트

- 어떤 도구를 주로 사용했는지, 작업 중 어려웠던 부분을 한줄로 작성

## 평가 기준

| 항목 | 비율 | 설명 |
|------|------|------|
| 도구 활용 | 40% | Extrude, Loop Cut, Inset, Bevel 등을 적절히 활용했는가 |
| 완성도 | 30% | 로봇/캐릭터의 기본 형태가 명확히 표현되었는가 |
| 창의성 | 30% | 자신만의 캐릭터 특성이나 독창적인 형태를 시도했는가 |

## 팁

- 무리하게 복잡한 형태를 만들기보다 깔끔한 기본 형태를 우선 완성할 것
- Numpad 1, 3, 7을 활용하여 정면/측면/상면에서 형태를 수시로 확인
- 좌우 대칭이 필요하면 한쪽만 작업하고 다음 주에 배울 Mirror Modifier를 적용할 준비
- Apply Transform(Ctrl + A)을 잊지 말 것
