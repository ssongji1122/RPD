# RPD Course OS Positioning

## Product Thesis

RPD는 범용 LMS가 아니라 `AI/3D 수업용 Course OS`로 포지셔닝합니다.

- 공개 영역: 학습자가 보는 정적 강의 사이트
- 비공개 영역: 교수자/운영자가 쓰는 편집, 업로드, publish 도구
- 자동화 영역: 커리큘럼 생성, Notion publish, 영상 제작 파이프라인

## Core Modules

1. 커리큘럼 발행
   - `weeks/site-data.json` 기반 canonical authoring
   - 공개 사이트 빌드
   - 변경 diff / 버전 추적

2. Show Me 위젯 라이브러리
   - 개념 이해용 인터랙티브 카드
   - week/library/shortcuts와 연결된 학습 흐름

3. 과제/진도 운영
   - 과제 템플릿
   - 로컬 또는 향후 인증 기반 진도 추적
   - 교수자 운영 대시보드 확장 여지

4. 수업 영상 제작 자동화
   - `tools/lessonforge`
   - 자막/아웃라인/요약 카드 생성
   - 배포형 학습 자료 제작 시간 단축

## Buyer Narrative

- 대학: 실습 중심 AI/3D 수업의 운영 표준화
- 학원: 강사별 콘텐츠 편차 축소
- 교수자 개인: 준비 시간 절감과 수업 품질 일관성 확보

## KPI

- 주차 활성 학습자
- step 완료율
- 과제 제출율
- 콘텐츠 수정 반영 시간
- 영상 제작 시간

## Demo Message

- “한 번의 커리큘럼 수정으로 공개 사이트, 교수자 운영 화면, Notion publish 흐름이 함께 맞춰집니다.”
- “Show Me 카드가 실습 중 막히는 개념을 짧은 상호작용으로 다시 설명합니다.”
- “영상 제작 파이프라인이 수업 자료 재가공 시간을 줄여줍니다.”
