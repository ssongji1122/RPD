# Chapter Overview Style Guide

`RPD` 영상의 `챕터 개요` / `목차 카드` / `오프닝 카드`를 위한 스타일 가이드.

## 목적

- 영상 시작 3~5초 안에 이번 챕터의 핵심 내용을 빠르게 이해시킨다.
- 텍스트 자막처럼 흐르지 않고, 정리된 인포그래픽 슬라이드처럼 보이게 만든다.
- 강의 카드, 목차, 비교 슬라이드, 핵심 요약 슬라이드에 동일한 룩앤필을 적용한다.

## 시각 방향

- 전체 톤: `블랙 베이스 + 화이트 타이포 + 네온 그린 포인트`
- 인상: `차분함`, `집중감`, `기술적`, `깔끔함`, `설명형`
- 지양: 그라데이션 남용, 장식 아이콘 남발, 과도한 박스 분할, 알록달록 팔레트

## 컬러 토큰

- Background: `#050505`
- Primary Text: `#F7F7F7`
- Secondary Text: `#969696`
- Divider / Stroke: `#5C5C5C`
- Low Contrast Divider: `#222222`
- Accent Green: `#00F58C`

## 타이포그래피

- 제목: 아주 굵은 산세리프, 중앙 정렬, 1~2줄
- 섹션 라벨: 작고 절제된 텍스트, 상단 중앙 배치
- 항목 제목: 굵은 화이트 텍스트
- 항목 설명: 한 단계 작은 그레이 텍스트
- 하단 요약 문장: 네온 그린, 굵게, 화면 하단 중앙
- 학교명, 과목명, 학기 표기 같은 footer 텍스트는 넣지 않는다

## 레이아웃 규칙

### 1. 챕터 개요 카드

- 상단 중앙: `챕터 개요`
- 그 아래 중앙: 챕터 제목
- 본문: `번호 원형 배지 + 항목 제목 + 보조 설명`
- 하단 중앙: 한 줄 핵심 요약
- 하단 보조 꼬리표는 생략

### 2. 목차 항목 수

- 기본 2~4개
- 최대 4개를 권장
- 5개 이상이면 의미상 묶거나 다음 카드로 분리

### 3. 항목 구조

- 제목: 명사형 또는 핵심 키워드 위주
- 설명: 1줄 보조 문장
- 예시:
  - `3D Viewport` / `메인 작업 공간`
  - `Outliner` / `씬 오브젝트 목록`
  - `Properties` / `선택 오브젝트의 세부 설정`

## 문구 작성 규칙

- 제목은 짧고 선명하게
- 설명은 기능을 바로 이해할 수 있게
- 하단 요약은 “이번 챕터의 핵심 한 문장”처럼 작성
- 한 카드 안에서 문장 길이를 비슷하게 유지
- 너무 교과서적인 문장보다, 바로 이해되는 설명형 표현 선호

## 카드별 사용 규칙

### 챕터 개요

- 오프닝 카드
- 본편 시작 전 3~5초 사용
- 강의 제목보다 “이번 파트에서 뭘 보는지”가 우선

### 목차 카드

- 세부 단계 안내용
- 실습 시작 전 또는 중간 전환 시 사용
- 항목 제목을 단계 중심으로 정리

### 핵심 요약 카드

- 하단 네온 그린 문장을 더 강조
- 항목 수는 줄이고 결론을 크게 보여준다

## 구현 원칙

- 카드 해상도는 원본 영상 해상도에 맞춰 생성
- 영상 오프닝 삽입 시 카드와 본편의 프레임 크기를 통일
- 카드 색상과 두께는 프로젝트 전체에서 고정
- 스타일 변경은 코드와 문서 둘 다 함께 갱신
