# week.html 리디자인 설계서

**날짜**: 2026-03-14
**범위**: `course-site/week.html` (index.html 제외)
**목표**: 수업 중 집중 + 나중에 빠른 참조, 인포그래픽 스타일

---

## 컨텍스트

- **사용 환경**: 듀얼 모니터 (Blender + 브라우저 풀스크린) + 모바일
- **사용 시점**: 수업 중 따라가기 + 수업 후 복습/참조
- **학생 수준**: 다양 — 단계 잠금 없이 자유 접근 필요

---

## 레이아웃: B형 (사이드바 + 스크롤)

```
┌─────────────────────────────────────────────────┐
│  Topbar: Week 03 · 모델링 기초   [≡ 목차] [진도] │
├──────────┬──────────────────────────────────────┤
│ SIDEBAR  │  MAIN CONTENT (스크롤)               │
│ (toggle) │                                      │
│          │  인포그래픽 스텝 카드들               │
│ 개요     │                                      │
│ 실습     │                                      │
│  ● Step1 │                                      │
│  ○ Step2 │                                      │
│ 단축키   │                                      │
│ 과제     │                                      │
└──────────┴──────────────────────────────────────┘
```

---

## 사이드바

- **기본 상태**: 숨김
- **토글**: Topbar의 `≡` 버튼
- **내용**: 섹션 목차 (개요 / 실습 Step1~N / 단축키 / 과제)
- **동작**: 스크롤 위치에 따라 현재 섹션 자동 하이라이트 (IntersectionObserver)
- **모바일**: 오버레이 드로어 (전체화면 가리지 않음)

---

## 스텝 카드 구조

```
┌─────────────────────────────────────────┐
│  ①  프리퍼런스 세팅               [완료] │
│  ─────────────────────────────────────  │
│  [이미지 (step.image)]                  │
│                                         │
│  step.copy (친절한 설명 텍스트)          │
│                                         │
│  □ task.label                           │
│     task.detail (small)                 │
│  □ task.label                           │
└─────────────────────────────────────────┘
```

**제거 항목**: `step.goal`, `step.done`, status-pill "진행 중/잠금", 단계 잠금 로직
**유지 항목**: `step.title`, `step.copy`, `step.image`, `task.label`, `task.detail`, 완료 체크박스
