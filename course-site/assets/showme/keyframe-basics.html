키프레임이란?

시간축(타임라인) 위의 특정 프레임에 값(위치, 회전, 크기 등)을 기록하는 것입니다. 두 키프레임 사이의 값은 Blender가 자동으로 계산(보간, Interpolation)하여 부드러운 움직임을 만듭니다.

만화 원화 — 핵심 장면(키프레임)만 그리면 중간 동작(인비트윈)은 자동으로 채워집니다

보간(Interpolation)이란?

두 키프레임 사이의 중간 값을 자동으로 계산하는 방법입니다. Linear(직선)는 일정한 속도, Bezier는 부드러운 가감속(Ease In/Out)을 만듭니다. Graph Editor에서 곡선을 직접 조절할 수 있습니다.

A지점에서 B지점까지 — Linear는 일정 속도, Bezier는 천천히 출발해서 천천히 도착

키프레임이 필요할 때

  • 오브젝트 이동/회전/크기 변화 애니메이션
  • 카메라 무빙
  • 재질 색상/투명도 변화
  • 조명 밝기/색상 변화

키프레임 외 다른 방법

  • 물리 시뮬레이션 (Rigid Body, Cloth 등)
  • Constraints 기반 애니메이션
  • Driver를 사용한 자동 연동
팁: I를 누르면 어떤 값을 키프레임할지 메뉴가 나타납니다. Location, Rotation, Scale을 자주 사용하며, LocRotScale을 선택하면 세 가지를 한 번에 키프레임합니다.
