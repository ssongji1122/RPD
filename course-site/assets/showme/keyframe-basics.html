키프레임이란? 기본

키프레임은 특정 시간(프레임)에 오브젝트의 상태(위치, 회전, 크기 등)를 기록하는 것입니다. 두 키프레임 사이의 중간 값은 Blender가 자동으로 계산(보간)합니다.

만화 원화가가 핵심 동작만 그리면, 사이 동작은 어시스턴트가 채우는 것과 같아요

키프레임 타입 핵심

I를 누르면 키프레임 삽입 메뉴가 나옵니다. 주요 타입은 다음과 같습니다:

  • Location: 위치 (X, Y, Z)
  • Rotation: 회전
  • Scale: 크기
  • LocRotScale: 위치+회전+크기 한번에

Timeline 인터페이스

Timeline은 화면 하단의 타임라인 에디터입니다. 키프레임은 마름모 모양으로 표시됩니다. 재생 헤드를 이동하여 현재 프레임을 변경할 수 있습니다.

  • 마름모: 키프레임 위치
  • 파란 선: 현재 프레임 (Playhead)
  • Start/End: 렌더 범위 설정

주요 단축키

I 키프레임 삽입 (Insert Keyframe)
Alt + I 키프레임 삭제
Space 애니메이션 재생/정지
← / → 이전/다음 프레임
↑ / ↓ 이전/다음 키프레임으로 이동
Shift + ← / → 처음/끝 프레임으로 이동

공이 좌에서 우로 이동하는 애니메이션입니다. Timeline과 키프레임을 확인하세요.

재생 버튼으로 애니메이션을 시작하고, Timeline에서 키프레임(마름모)을 확인하세요.

키프레임이 필요한 경우

  • 오브젝트 이동 애니메이션
  • 카메라 움직임
  • 라이트 밝기/색상 변화
  • 머터리얼 속성 변화

키프레임 대신 사용하는 것

  • 물리 시뮬레이션 (자동 계산)
  • Drivers (수식 기반 자동화)
  • Constraints (관계 기반)
  • Shape Keys (변형 블렌드)
팁: Auto Keying 기능을 켜면 오브젝트를 이동/회전/크기 변경할 때 자동으로 키프레임이 삽입됩니다. Timeline 상단의 녹화 버튼(빨간 원)을 클릭하세요. 단, 의도치 않은 키프레임이 생길 수 있으니 주의하세요.

애니메이션 워크플로우

Pose to Pose
핵심 포즈(키프레임)를 먼저 잡고, 중간 동작을 나중에 추가하는 방식입니다. 전체 흐름을 먼저 잡을 수 있어 효율적입니다.
Straight Ahead
프레임 1부터 순서대로 키프레임을 추가하는 방식입니다. 자연스러운 물리적 동작에 적합하지만 수정이 어렵습니다.
