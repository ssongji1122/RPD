핵심 G/R/S 삼총사

G = 잡아 이동(Grab), R = 회전(Rotate), S = 크기 변형(Scale). 이 세 가지만 알면 Blender에서 모든 변형의 70%를 커버할 수 있습니다. 키를 누른 뒤 마우스를 움직이고, 클릭으로 확정합니다.

카드 게임처럼 — G(잡기), R(돌리기), S(크기 바꾸기) 세 장의 카드로 모든 변형을 해요

필수 축 고정 (Axis Constraint)

G, R, S를 누른 뒤 X, Y, Z 키를 누르면 해당 축으로만 변형이 제한됩니다. 이중 축 제한도 가능합니다: G → Shift+Z = Z축을 제외한 XY 평면에서만 이동합니다.

레일 위의 기차처럼 — 축을 고정하면 정해진 방향으로만 움직여요

주요 단축키

G 이동 (Grab)
R 회전 (Rotate)
S 크기 변형 (Scale)
+ X / Y / Z 해당 축으로 고정
+ 숫자 입력 정확한 값 입력 (예: G → X → 2 = X축 2m 이동)
Shift + 축 해당 축 제외 (나머지 두 축에서 변형)

모드 선택 후 캔버스를 드래그 | 축 고정으로 제한된 변형 체험

G (이동) 활용

  • 오브젝트 배치 및 위치 조정
  • Edit Mode에서 정점 미세 조정
  • 장면 구성 및 레이아웃
  • 축 고정으로 정확한 배치

R / S 활용

  • R: 방향 맞추기, 기울이기, 각도 조정
  • S: 비율 조정, 찌그러트리기, 변형
  • R+Z+90: 정확한 90도 회전
  • S+특정축: 한 방향으로만 늘리기

자주 쓰는 조합

G → X → 2
X축으로 정확히 2m 이동합니다. 숫자를 타이핑하면 정밀한 배치가 가능합니다.
R → Z → 90
Z축 기준으로 정확히 90도 회전합니다. 오브젝트 방향을 깔끔하게 맞출 때 유용합니다.
팁: 숫자 입력이 핵심! G → X → 1.5 처럼 정확한 값을 넣으면 프로처럼 작업할 수 있어요.
