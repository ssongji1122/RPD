Subdivision Surface = 폴리곤을 수학적으로 세분화하여 부드럽게 만드는 모디파이어. Low-Poly 메쉬에서 고해상도 곡면을 생성합니다.

알고리즘 Catmull-Clark

각 폴리곤 페이스를 4개로 분할하고, 꼭짓점을 인근 점들의 가중 평균으로 이동시킵니다. 사각형 메쉬에 최적화된 방식입니다.

각진 종이접기를 수학적으로 부풀려 매끄러운 풍선으로 만드는 것

레벨 Viewport vs Render

Viewport는 작업 속도를 위해 낮게, Render는 최종 품질을 위해 높게 설정합니다.

  • Level 0: 원본 (변화 없음)
  • Level 1: 폴리곤 4배
  • Level 2: 폴리곤 16배
  • Level 3: 폴리곤 64배

기술 Crease 엣지

엣지를 선택 후 Shift+E로 Crease 값(0~1)을 설정합니다. 값이 1이면 Subdivision이 올라가도 그 모서리는 날카롭게 유지됩니다.

풍선에서 실로 꽉 묶인 부분만 날카롭게 남는 것
더보기
Subdivision은 폴리곤 수를 늘리고, Shade Smooth는 빛 반사를 부드럽게 만듭니다. 둘은 다른 기능입니다. 결과가 너무 둥글면 Crease를, 셰이딩만 거칠면 Shade Smooth by Angle을 먼저 확인하면 됩니다.

주요 단축키

Ctrl + 1 / 2 / 3 Subdivision 레벨 1/2/3 바로 적용 (오브젝트 모드)
Shift + E 선택한 엣지에 Crease 값 설정 (Edge 모드)
Ctrl + A → Apply Scale Subdivision 적용 전 스케일 초기화 필수

유기적 모델링

  • 캐릭터 얼굴 / 신체 모델링
  • 동물, 생물 형태
  • 부드러운 가구 / 쿠션
  • 자동차 차체 곡면

Hard Surface + Crease

  • 칼날, 메탈 엣지 유지
  • 로봇, 메카 부품의 날카로운 경계
  • 헬멧, 방패 등 Crease 혼합
  • Support Edge 대신 Crease 활용

렌더 품질 설정

  • Viewport: 1~2 레벨 (작업 속도)
  • Render: 2~3 레벨 (최종 품질)
  • 배경 오브젝트: Viewport 0, Render 1

Subdivision 안 쓰는 경우

  • 건축물 (날카로운 각형 형태)
  • 로우폴리 / 스타일라이즈드 아트
  • 이미 폴리곤이 많은 임포트 메쉬
  • 게임 최적화용 실시간 메쉬

자주 쓰는 조합

Mirror + Subdivision
Mirror가 위, Subdivision이 아래. 먼저 대칭을 잡고 나서 부드럽게 만드는 올바른 순서입니다.
Bevel + Subdivision
Bevel로 지지 엣지(support edge)를 만들어 Subdivision 형태를 날카롭게 제어합니다.

흔한 실수

실수 1: Apply 후 파일 용량 폭증
Level 3+ 적용 시 수백만 폴리곤이 생성됩니다. Apply 전 레벨을 반드시 낮추세요.
실수 2: 모서리가 다 둥글게 됨
Crease 없이 Subdivision 적용 시 모든 모서리가 부드러워집니다. Shift+E로 날카운 엣지를 미리 지정하세요.
팁: Subdivision 적용 전 반드시 Apply Scale (Ctrl+A)! 스케일이 (1,1,1)이 아니면 서브디비전 결과가 왜곡됩니다.
