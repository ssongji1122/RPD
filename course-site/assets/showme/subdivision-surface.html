Subdivision Surface란?

메쉬를 세분화하여 부드럽게 만드는 모디파이어입니다. 각 면을 4개로 나누고 꼭짓점을 평균화하여 곡면을 생성합니다. Catmull-Clark 알고리즘을 사용합니다.

각진 종이접기를 부드럽게 풍선처럼 부풀리는 것

Viewport vs Render 레벨

Viewport 레벨은 작업 중 프리뷰용으로 보통 1~2를 사용합니다. Render 레벨은 최종 출력용으로 보통 2~3을 사용합니다. 레벨이 높을수록 부드럽지만 무거워집니다.

  • 레벨 1: 면 수 4배
  • 레벨 2: 면 수 16배
  • 레벨 3: 면 수 64배

Crease (접기 강도)

Shift+E로 모서리에 Crease 값(0~1)을 설정합니다. 값이 1이면 해당 모서리가 날카롭게 유지됩니다. Edge Mean Crease로 Subdivision에서도 형태를 제어할 수 있습니다.

부풀어 오르는 풍선에서 실로 꽉 묶은 부분만 날카롭게 남는 것

주요 단축키

Ctrl + 1 / 2 / 3 Subdivision 레벨 1/2/3 바로 적용
Shift + E 선택한 모서리에 Crease 설정
Ctrl + E → Mark Sharp 날카로운 모서리 표시
📖 Blender 공식 문서 — Subdivision Surface Modifier

버튼을 눌러 Subdivision 레벨에 따른 메쉬 변화를 확인하세요. Crease가 적용된 모서리(주황색)는 날카롭게 유지됩니다.

각 레벨에서 면이 4배씩 늘어나며 점점 부드러워집니다. Crease=1인 모서리는 레벨이 올라가도 날카롭게 유지됩니다.

Subdivision이 필요할 때

  • 유기적 캐릭터
  • 부드러운 가구
  • 자동차 보디
  • 동물/생물

Subdivision을 안 쓰는 경우

  • 건축물 (각진 형태)
  • 저폴리 스타일
  • 이미 폴리곤이 많은 모델

자주 쓰는 조합

Subdivision + Mirror
대칭 캐릭터 모델링의 기본 조합. Mirror가 위, Subdivision이 아래 순서로 배치합니다.
Subdivision + Bevel
Bevel로 지지 엣지(support edge)를 넣어 Subdivision 형태를 정밀하게 제어합니다.
팁: Subdivision 전에 반드시 Apply Scale(Ctrl+A)! 스케일이 (1,1,1)이 아니면 왜곡됩니다.
