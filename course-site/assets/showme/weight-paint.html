Core Weight Paint란?

각 Bone이 메쉬의 어느 부분에 얼마나 영향을 주는지 결정하는 시스템입니다. Weight 값은 0(영향 없음)부터 1(완전한 영향)까지이며, 색상으로 시각화됩니다.

열화상 카메라처럼 — 빨간 부분이 뼈의 영향을 가장 많이 받는 곳이에요

색상 Weight 색상 스펙트럼

빨강(1.0) = 해당 Bone에 완전히 따라감. 노랑(0.5~0.7) = 중간 영향. 초록(0.3~0.5) = 약한 영향. 파랑(0.0) = 영향 없음. 이 그라데이션으로 자연스러운 변형을 만듭니다.

Skinning Automatic Weights

Ctrl+P → Armature Deform → With Automatic Weights를 선택하면 Blender가 각 Bone의 위치를 기반으로 자동 Weight를 계산합니다. 이후 Weight Paint 모드에서 수동 보정합니다.

브러시 Paint 도구

Add 브러시로 Weight를 높이고, Subtract(Ctrl)로 줄입니다. Blur 브러시로 경계를 부드럽게 만들어 자연스러운 변형을 유도합니다.

주요 단축키

Ctrl + Tab Weight Paint Mode 진입
F 브러시 크기 조절
Shift + F 브러시 강도(Strength) 조절
Ctrl Weight 줄이기 (Subtract 모드)

팔 메쉬 위에서 클릭/드래그하여 Weight를 칠해 보세요. 브러시 모드를 바꿔 Add/Subtract/Blur를 체험하세요.

메쉬 위를 클릭/드래그하여 Weight를 칠합니다. 색상 범례: 파랑(0) → 초록 → 노랑 → 빨강(1)

Weight Paint가 필요할 때

  • Automatic Weights 결과가 부자연스러울 때
  • 관절(팔꿈치, 무릎) 변형이 깨질 때
  • 옷이 몸을 뚫고 나올 때
  • 얼굴 리깅에서 섬세한 조절이 필요할 때

Weight Paint가 불필요한 경우

  • Automatic Weights로 충분한 간단한 모델
  • 로봇처럼 관절이 딱딱한 경우 (Weight 0 또는 1만)
  • 정적 오브젝트 (Armature 불필요)
팁: Weight Paint 작업 시 Pose Mode와 동시에 사용하면 실시간으로 변형을 확인할 수 있습니다. Armature를 선택하고 Ctrl+Tab으로 Pose Mode, 메쉬를 선택하고 Weight Paint Mode로 전환하세요.

자주 쓰는 조합

Weight Paint + Pose Mode
포즈를 잡으면서 실시간으로 Weight를 수정. 관절 변형이 자연스러운지 바로 확인할 수 있습니다.
Vertex Groups + Weight Paint
Properties에서 Vertex Group을 선택하면 해당 Bone의 Weight만 표시됩니다. Bone별로 정밀하게 작업할 수 있습니다.
