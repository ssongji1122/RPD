핵심 Object Mode vs Edit Mode

Object Mode는 가구 배치처럼 전체 오브젝트를 이동/회전/크기 변경합니다. Edit Mode는 목공 작업처럼 오브젝트의 형태 자체를 변경합니다. Tab 키로 두 모드를 전환합니다.

Object Mode는 방에서 가구를 배치하는 것, Edit Mode는 가구 자체를 깎고 다듬는 것

필수 Vertex / Edge / Face

메시의 3대 구성 요소입니다. 점(Vertex), 선(Edge), 면(Face). Edit Mode에서 1/2/3 키로 선택 모드를 전환하여 원하는 요소를 선택하고 편집합니다.

레고 블록의 점(꼭짓점), 테두리(모서리), 표면(면)을 각각 잡아서 변형하는 것과 같아요

주요 단축키

Tab Object Mode / Edit Mode 전환
1 / 2 / 3 Vertex / Edge / Face 선택 모드
A 전체 선택
Alt + A 선택 해제
Ctrl + Z 되돌리기 (Undo)

Object Mode에서 오브젝트는 주황색 아웃라인으로 선택됩니다. 오브젝트 전체가 하나의 단위로 이동/회전합니다.
Object Mode
Edit Mode

Object Mode

  • 여러 오브젝트 배치할 때
  • 전체 크기/위치 조정할 때
  • 렌더링 전 장면 구성할 때
  • Modifier 적용 전 확인할 때

Edit Mode

  • 형태를 바꿀 때 (Extrude, Bevel 등)
  • 디테일 추가할 때 (Loop Cut)
  • 메시 정리할 때 (Merge, Dissolve)
  • UV 작업을 위한 심(Seam) 표시할 때

자주 쓰는 조합

Tab (Edit) + 3 (Face) + E (Extrude)
Edit Mode로 전환 후 면 선택 모드에서 면을 선택하고 E로 뽑아내어 형태를 확장합니다.
Tab (Edit) + 2 (Edge) + Ctrl+B (Bevel)
Edit Mode에서 모서리를 선택하고 Ctrl+B로 베벨을 적용하여 모서리를 부드럽게 다듬습니다.
팁: Edit Mode에서 변경한 것은 Object Mode로 돌아가도 그대로 유지돼요. 걱정 말고 Tab을 누르세요!
