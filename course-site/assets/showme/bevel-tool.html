Ctrl+B Bevel Tool

모서리를 깎아 부드럽게 만드는 도구입니다. 실제 제품에서 안전과 미관을 위해 모서리를 둥글게 깎는 것(Chamfer)과 같습니다. 엣지를 선택하고 Ctrl+B를 누른 후 마우스를 이동하면 깎이는 양을 조절할 수 있습니다.

Segments & Profile

스크롤로 Segment 수를 늘려 더 부드럽게 만들 수 있습니다. Profile 값(P키)으로 깎인 형태를 볼록(1.0)하게 또는 오목(0.0)하게 조절할 수 있습니다. Segment 1은 Chamfer(평면 깎기), 2 이상은 둥근 모서리가 됩니다.

나무 책상 모서리를 사포로 둥글게 갈아내는 것, 그게 Bevel이에요

주요 단축키

Ctrl + B Bevel 시작 — 마우스 이동으로 깎기 양 조절
스크롤 Segment 수 조절 — 부드러움 정도
P (Bevel 중) Profile 조절 — 볼록/오목 형태 변경
Ctrl + Shift + B Vertex Bevel — 꼭짓점 깎기

Segments와 Profile 슬라이더를 조절하여 Bevel 효과를 비교하세요.

Segments 1 = Chamfer(평면 깎기), 2+ = 둥근 모서리. Profile 0.0 = 오목, 0.5 = 평평, 1.0 = 볼록

모서리 다듬기

  • 하드서피스 모델의 날카로운 엣지 정리
  • 제품 디자인의 필수 마무리
  • 리얼리즘 향상 (현실에 완벽한 직각은 없음)
  • 가구, 전자기기 등 산업 디자인

Subdivision 지원

  • SubD 적용 전 Bevel로 엣지 유지
  • Edge Loop 대신 Bevel로 형태 제어
  • 적은 지오메트리로 원하는 곡률 달성
  • Support Loop 역할 대체

자주 쓰는 조합

Ctrl+B + 스크롤 (Segments 3)
Bevel 후 스크롤로 Segment를 3으로 설정하면 자연스럽게 부드러운 모서리가 만들어집니다.
Edge 선택 + Ctrl+B + 이동
엣지를 선택하고 Bevel한 후 마우스를 이동하면 실시간으로 깎이는 양을 프리뷰할 수 있습니다.
주의: Bevel은 Ctrl+A → Scale 적용 후에 사용하세요! 스케일이 비균일(non-uniform)하면 양쪽 깎이는 양이 달라져서 비대칭 결과가 나옵니다.
