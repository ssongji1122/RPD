UV Editor란?

UV 좌표를 시각적으로 편집하는 전용 에디터입니다. 3D 모델의 표면이 2D 이미지 위에 어떻게 펼쳐져 있는지 보여주며, UV Island를 선택, 이동, 회전, 스케일 할 수 있습니다.

옷 패턴을 천 위에 배치하는 것 — 각 조각을 효율적으로 배열해야 합니다

UV Island란?

Seam으로 분리된 UV의 각 조각을 Island(섬)라고 합니다. Island끼리 겹치면 텍스처가 잘못 표시되므로, UV Editor에서 정리하여 겹침 없이 0~1 범위 안에 배치해야 합니다.

퍼즐 조각처럼 각 Island를 빈틈없이 맞춰 넣어야 해요

주요 단축키 (UV Editor)

L Island 선택 (Linked Select)
P Pin — 선택한 UV 정점 고정
A 전체 선택 / 해제
S+X/Y+0 X 또는 Y축 정렬 (좌표를 0으로)
G / R / S 이동 / 회전 / 스케일

UV Island를 드래그하여 이동하세요. 회전 버튼으로 선택된 Island를 회전할 수 있습니다.

Island를 클릭해 선택(파란 테두리)한 뒤 드래그하세요. 격자가 UV 0-1 범위입니다.

UV Editor 조작이 필요할 때

  • Unwrap 후 Island 겹침 정리
  • 텍스처에 맞게 Island 위치 조절
  • Island 크기 비율 맞추기 (Normalize)
  • 특정 부분에 텍스처 해상도 집중

UV Editor를 안 써도 되는 경우

  • Procedural 텍스처만 사용
  • Smart UV Project로 충분한 경우
  • 단순한 형태 (평면, 구 등)
팁: UV Editor에서 N 키로 사이드바를 열면 UV 좌표 수치를 직접 입력할 수 있습니다. 정확한 정렬이 필요할 때 유용합니다.

자주 쓰는 기능

Pack Islands
모든 Island를 0-1 범위 안에 자동으로 빈틈없이 배치합니다. UV 메뉴에서 실행합니다.
Average Islands Scale
모든 Island의 텍셀 밀도(해상도)를 균일하게 맞춥니다. 크기가 제각각인 Island를 정리할 때 사용합니다.
