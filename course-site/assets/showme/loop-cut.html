Ctrl+R Loop Cut

메시를 고리(loop) 형태로 한 바퀴 자르는 도구입니다. 디테일을 추가하는 가장 기본적인 방법이며, 모델링의 필수 기법입니다. Ctrl+R을 누르면 노란색 프리뷰 선이 나타나고, 마우스 휠로 분할 수를 조절할 수 있습니다.

GG Edge Slide

이미 있는 Edge Loop를 표면을 따라 미끄러지듯 이동시킵니다. 형상은 유지하면서 분할 위치만 조정할 때 사용합니다. G를 두 번 연속 누르면 Edge Slide 모드가 됩니다.

빵을 슬라이스하는 것처럼, Loop Cut은 메시를 깔끔하게 한 바퀴 잘라요

주요 단축키

Ctrl + R Loop Cut 시작 — 노란색 프리뷰 표시
스크롤 분할 수 조절 — 한 번에 여러 줄 추가
LMB 위치 확정 — 클릭한 곳에 배치
RMB 중앙 배치 — 정확히 가운데에 배치
G G Edge Slide — 표면을 따라 엣지 이동

디테일 추가

  • 모서리를 날카롭게 유지 (Subdivision과 함께)
  • 특정 영역만 세밀하게 나누기
  • Subdivision Surface 적용 시 형태 보존
  • 원하는 곳에만 정밀한 메시 밀도 추가

형태 제어

  • 창문이나 문 위치 잡기 (건축 모델링)
  • 캐릭터 관절 부위에 Edge Loop 추가
  • 벨트, 팔찌 등 장식 위치 설정
  • Extrude할 면 영역 구분

자주 쓰는 조합

Ctrl+R + Ctrl+B (Loop Cut + Bevel)
Loop 추가 후 해당 엣지를 Bevel로 다듬기. 날카로운 모서리를 부드럽게 만들 때 필수 조합입니다.
Ctrl+R + E (Loop Cut + Extrude)
새로 생긴 면을 선택해서 뽑기. 창문이나 선반 같은 돌출 형태를 만들 때 사용합니다.
팁: Loop Cut 중에 스크롤(마우스 휠)을 굴리면 분할 수가 늘어나요. 한 번에 여러 줄을 추가할 수 있어요! 확정 전 RMB를 누르면 정확히 중앙에 배치됩니다.
