Boolean이나 Extrude를 잘못 누르면 같은 자리에 버텍스가 두 개씩 쌓여요. 눈에는 멀쩡해 보여도 메쉬가 끊어진 상태라 Subdivision이 깨지거나 Smooth가 이상하게 적용돼요.

Merge by Distance는 설정한 거리 안에 있는 버텍스를 하나로 합쳐주는 청소 도구예요. 빨래 짤 때 물기 빼듯, 메쉬에서 중복 정보를 짜내는 작업.

  1. 1 Edit Mode 진입 Tab
  2. 2 전체 선택 A
  3. 3 Merge 메뉴 열기 M Mesh > Merge
  4. 4 By Distance 선택 By Distance
  5. 5 좌하단 패널에서 거리 조정 Merge Distance 0.0001m → 0.001m 순으로

언제 쓰나요

Boolean 직후, Mirror에 Clipping 없이 작업한 직후, 외부 메쉬(AI 생성·STL import)를 받았을 때 가장 먼저 돌려요. Subdivision이 갑자기 이상해지면 십중팔구 중복 버텍스가 원인.

흔한 실수

거리가 너무 크면(예: 1m) 분리되어 있어야 할 버텍스까지 합쳐져 메쉬가 뭉개져요. 기본값 0.0001m부터 시작하고, 안 합쳐지면 단계적으로 키워요. 작업 후에는 합쳐진 개수가 화면 하단에 표시돼요 — 'Removed 12 vertices' 같은 메시지.
