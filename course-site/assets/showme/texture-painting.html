핵심 원리

Texture Paint 모드는 3D 오브젝트 표면에 직접 붓질하는 모드입니다. 브러시로 칠하면 오브젝트에 연결된 UV 텍스처 이미지가 실시간으로 업데이트됩니다.

3D 오브젝트가 종이처럼 UV로 펼쳐져 있는데, 그 위에 수채화처럼 직접 그림을 그리는 것

두 가지 뷰

Texture Paint 모드에서는 두 곳에서 동시에 작업할 수 있습니다:

  • 3D 뷰포트 — 오브젝트 표면에 직접 칠하기
  • UV Editor — 펼쳐진 2D 이미지 위에 칠하기

두 뷰에서 칠한 내용은 실시간으로 동기화됩니다.

브러시 설정

  • F — 브러시 반경 조절 (드래그)
  • Shift+F — 브러시 강도 조절
  • X — 전경색 / 배경색 교체
  • S — Stencil 모드 토글
  • Ctrl+클릭 — 색상 피커 (스포이드)

Blend Mode

칠하는 방식(합성 모드)을 선택할 수 있습니다:

  • Mix — 기본 혼합
  • Add / Subtract — 밝게 / 어둡게
  • Multiply — 곱하기 합성
  • Lighten / Darken — 밝은 / 어두운 값 유지

Stencil 모드

이미지를 마스크로 사용해 반복되는 패턴을 오브젝트 표면에 찍어낼 수 있습니다. 문양이나 데칼 효과에 유용합니다.

준비 사항

Texture Paint 모드를 사용하기 전에 반드시 준비해야 할 것:

  • UV 언래핑 완료
  • 이미지 텍스처 노드 생성 (Shader Editor에서)
  • 저장: Image > Save As 또는 .blend에 Pack

단축키 요약

F 브러시 반경 조절 — 누른 채 좌우 드래그
Shift + F 브러시 강도(Strength) 조절
X 전경색 / 배경색 교체
S Stencil 모드 토글
Ctrl + 클릭 색상 피커 — 오브젝트 표면의 색상 샘플링
Numpad 0 카메라 뷰 전환

3D 구체 위를 클릭하거나 드래그해서 페인팅하세요 — UV 맵에도 동시에 반영됩니다

이럴 때 사용하세요

  • 오브젝트에 독특한 텍스처·색상 패턴을 직접 그릴 때
  • UV 맵이 복잡해서 2D 편집이 어려울 때 3D에서 직접 칠하기
  • 캐릭터 피부, 낡은 벽, 나무 결 등 유기적인 디테일 추가
  • 데칼·스탬프 이미지를 표면에 찍어낼 때

흔한 실수

  • 이미지 텍스처를 먼저 만들지 않고 칠하려 해서 오류 발생
  • UV 언래핑 없이 칠해서 패턴이 찌그러짐
  • 칠한 후 Image > Save As를 하지 않아 재시작 후 사라짐
  • Blend Mode를 Mix로 두지 않고 칠해서 예상치 못한 결과

작업 흐름 (Workflow)

1단계 — UV 언래핑
Edit Mode에서 UV 언래핑 완료. Smart UV Project 또는 수동 Seam 방식
2단계 — 이미지 생성
Shader Editor에서 Image Texture 노드 추가 후 New Image 생성
3단계 — 페인팅
Texture Paint 모드 진입 후 브러시로 칠하기. F로 크기, Shift+F로 강도 조절
4단계 — 저장
Image > Save As 로 외부 저장 또는 Image > Pack Into .blend
팁: 3D 뷰포트에서 칠한 결과를 바로 보면서 UV Editor를 함께 열면 작업 효율이 크게 올라갑니다. Area를 분리해서 두 뷰를 나란히 두세요.
