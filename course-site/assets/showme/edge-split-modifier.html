Edge Split이란?

특정 모서리에서 면을 분리하여 날카로운 셰이딩을 만드는 모디파이어입니다. Smooth Shading 상태에서 특정 모서리만 날카롭게 보이게 합니다.

부드러운 천에 접는 선을 만드는 것과 같아요

Split Angle

이 각도보다 큰 모서리는 자동으로 분리됩니다. 기본값은 30도입니다. 각도가 클수록 더 적은 모서리가 분리되고, 작을수록 더 많은 모서리가 분리됩니다.

수동 Sharp 지정

Sharp Edge 표시(Ctrl+E → Mark Sharp)로 수동 지정도 가능합니다. 각도 기반 자동 분리와 함께 사용할 수 있습니다.

Auto Smooth과의 관계

Blender 5.0에서는 Auto Smooth가 개선되어 Edge Split 대신 사용 가능한 경우가 많습니다. 노드 기반으로 더 유연한 제어가 가능합니다.

상태 전환 버튼으로 셰이딩 모드를 바꿔서 같은 실루엣이 어떻게 다르게 읽히는지 비교합니다.
Smooth는 모든 모서리를 하나의 덩어리처럼 이어 보이게 하고, Flat은 모든 면을 개별적으로 끊어 보이게 합니다. Edge Split은 필요한 모서리만 분리해 둘 사이를 조절하는 방식입니다.

All or Nothing

전부 Smooth면 기계 모서리까지 녹아 버리고, 전부 Flat이면 곡면까지 각져 보여서 형태 정보가 지나치게 손실됩니다.

Selective Sharp

Edge Split은 필요한 엣지만 날카롭게 남겨서, 하드서피스의 모서리와 곡면의 부드러움을 한 모델 안에서 함께 설명할 수 있어요.

모드를 전환하여 모서리 셰이딩이 어떻게 달라지는지 확인하세요.

Edge Split이 필요할 때

  • 기계 부품 (날카로운 모서리 + 부드러운 면 공존)
  • 건축 모델
  • 하드서페이스 모델링

Edge Split을 안 쓰는 경우

  • 유기적 모델 (캐릭터, 동물 등)
  • Auto Smooth으로 충분한 경우
  • 이미 Flat Shading이 적절한 경우

자주 쓰는 조합

Edge Split + Subdivision Surface
Edge Split으로 날카로운 모서리를 유지하면서 Subdivision으로 전체를 부드럽게 합니다.
Edge Split + Bevel
날카로운 모서리에 Bevel을 추가해 현실적인 하이라이트 반사를 만듭니다.
팁: Blender 5.0에서는 Auto Smooth 개선으로 Edge Split 대신 사용할 수 있는 경우가 많습니다. 노드 기반 Auto Smooth는 더 유연하고 비파괴적인 워크플로우를 제공합니다.
