Skin Modifier란?

꼭짓점과 모서리로 이루어진 뼈대(Skeleton)에 자동으로 살(Skin)을 씌우는 모디파이어입니다. 빠른 프로토타이핑에 최적화되어 있습니다.

철사로 뼈대를 만들고 점토로 감싸는 것과 같아요

Ctrl+A로 크기 조절

Edit Mode에서 꼭짓점을 선택한 뒤 Ctrl+A를 누르면 해당 지점의 Skin 반경을 조절할 수 있습니다. 팔은 가늘게, 몸통은 굵게 만들 수 있습니다.

Mark Root

어떤 꼭짓점을 루트(Root)로 지정할지 결정합니다. 루트 설정에 따라 Skin이 생성되는 방향과 형태가 달라집니다.

주요 단축키

Ctrl + A Skin 반경 조절 (Edit Mode)
Mark Root 버튼 Mesh → Skin → Mark Root (Edit Mode)

상태 전환 드래그는 각 지점의 Skin 반경을 직접 바꾸고, 버튼은 Subdivision 프리뷰와 초기화를 담당합니다.
Skin Modifier는 뼈대 점들을 선으로 잇고, 각 점의 반경만큼 살을 붙여 형태를 만드는 방식입니다. 반경을 어디서 키우고 줄이느냐가 몸통과 팔다리의 비율을 정합니다.

Uniform Radius

모든 지점의 반경을 비슷하게 두면 몸통과 관절의 차이가 사라져, 나무든 캐릭터든 단조로운 관 모양으로만 보이기 쉽습니다.

Readable Skeleton

몸통은 굵게, 가지와 팔다리는 가늘게 조절하면 적은 버텍스만으로도 구조와 무게중심이 읽히는 빠른 프로토타입을 만들 수 있어요.

꼭짓점의 원 테두리를 드래그하면 반경이 바뀝니다. Subdivision 토글로 부드러운 결과를 미리 볼 수 있습니다.

Skin이 필요할 때

  • 빠른 캐릭터 프로토타입
  • 나무 / 나뭇가지
  • 생물 / 촉수
  • 로봇 팔다리

Skin을 안 쓰는 경우

  • 정밀한 모델링
  • 하드서페이스 (Hard Surface)

자주 쓰는 조합

Skin + Subdivision Surface
Skin으로 기본 형태를 잡고 Subdivision으로 부드럽게. 유기적인 캐릭터에 필수 조합입니다.
Skin + Mirror
Mirror로 대칭 뼈대를 만들고 Skin으로 살을 씌워 대칭 캐릭터를 빠르게 프로토타이핑합니다.
팁: Skin으로 빠르게 형태를 잡고, Apply 후 Edit Mode에서 디테일을 추가하세요. 프로토타이핑의 왕도입니다!
