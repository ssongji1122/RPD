Apply Transform이란 Object Mode에서 설정한 Scale·Rotation 값을 메쉬 데이터에 실제로 "굽는(bake)" 작업입니다.

Apply Transform이란? 핵심

Object Mode의 Scale/Rotation은 메쉬에 실제 적용되지 않은 상태입니다. Ctrl+A → Apply Scale을 하면 현재 크기 모양은 유지되면서 Scale 값이 (1, 1, 1)로 초기화됩니다.

줄자의 눈금을 현재 위치에서 0으로 다시 세팅하는 것 — 물리적 위치는 그대로, 기준점만 이동

왜 Apply해야 하나요? 주의

Scale이 (1, 1, 1)이 아닌 상태에서 Modifier를 추가하면 Blender는 "메쉬 자체 크기"가 아닌 "변환 후 크기"를 기준으로 계산합니다.

  • Mirror Modifier — 비대칭으로 나옴
  • Bevel — 한쪽으로 찌그러짐
  • Subdivision Surface — 균일하지 않게 부풀음
  • Physics 시뮬레이션 — 크기·방향 오동작

주요 단축키

Ctrl+A → Rotation Apply Rotation만 적용 — Rotation을 (0,0,0)으로 리셋

반드시 Apply해야 할 때

  • Mirror Modifier 추가 전
  • Bevel / Array Modifier 추가 전
  • UV Unwrap 전
  • Physics (Cloth, Rigid Body) 적용 전
  • 다른 파일로 Export 전
  • Boolean 연산 전

흔한 실수 & 해결법

Mirror Modifier가 비대칭으로 나옴
Scale이 균일하지 않아서 발생. Ctrl+A → Apply Scale로 해결.
Cloth / Rigid Body가 이상하게 날아감
Scale이 1이 아니면 Physics가 실제 크기를 잘못 읽음. Apply Scale 후 시뮬레이션 재설정.
Bevel이 한쪽으로 찌그러짐
비균일 Scale 상태에서 Bevel 적용 시 발생. Apply Scale 필수.
Array 간격이 균일하지 않음
Scale이 1이 아닌 축 방향으로 Array하면 간격 비균일. Apply 후 재설정.
팁: 모델링 시작 전 습관적으로 Ctrl+A → All Transforms를 적용하세요. N-패널에서 Scale이 (1, 1, 1), Rotation이 (0, 0, 0)인지 항상 확인하세요.
