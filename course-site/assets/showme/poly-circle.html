핵심 원리 다각형 → 원

3D에서 진짜 "원"은 없어요. 모든 원형은 꼭짓점이 많은 다각형으로 흉내 낸 것입니다. 꼭짓점이 4개면 사각형, 8개면 8각형, 32개 이상이면 거의 완벽한 원처럼 보입니다.

픽셀로 그린 원처럼, 3D의 원도 충분히 많은 직선 조각으로 이루어져요

Loop Cut 면 분할의 의미

원통(Cylinder)에 Loop Cut(Ctrl+R)을 추가하면 가로 방향 면이 더 잘게 나뉩니다. 처음 원통을 만들 때 꼭짓점 수(Vertices)를 높게 설정하거나, Subdivision Surface와 함께 쓰면 훨씬 부드러운 원이 됩니다.

Loop Cut은 "가로줄"을 추가해서 세로 방향 조각의 수를 그대로 유지해요

R 값 Bevel Profile

Bevel 도구(Ctrl+B)의 Profile 값(R)은 베벨된 가장자리가 얼마나 둥근지 결정합니다.

  • 0.0 — 안쪽으로 오목하게
  • 0.5 — 평평한 기본값
  • 1.0 — 완전 원형 볼록 프로파일

세그먼트 수(S 입력)가 많을수록 R 값의 효과가 더 부드럽게 표현됩니다.

R 값이 1에 가까울수록 베벨이 파이처럼 부풀어 올라요

Crease 엣지 날카로움 제어

Crease 값은 Subdivision Surface 모디파이어와 함께 쓰여, 특정 엣지가 부드러워지지 않도록 "고정"하는 설정입니다.

  • 0.0 — 완전히 부드럽게 (Subdivision의 영향 받음)
  • 1.0 — 완전히 날카롭게 (Subdivision이 무시)

모서리는 부드럽게 만들되, 특정 경계선은 날카롭게 유지하고 싶을 때 사용합니다. 설정은 Edit Mode에서 엣지 선택 후 Sidebar(N) → Item 탭 → Edge Crease에서 조절하거나, Shift+E 단축키를 사용합니다.

Crease = "이 엣지만큼은 부드러워지지 마!" 라고 Subdivision에게 명령하는 것

주요 단축키 / 설정

Ctrl + R Loop Cut 추가
스크롤 휠 (Loop Cut 중) 루프컷 수 조절
Shift+A → Mesh → Circle 원 추가 시 꼭짓점 수 직접 지정 가능
Ctrl+B → S Bevel 후 S 입력으로 세그먼트 수 변경
Ctrl+B → P Bevel 후 P 입력으로 Profile(R) 값 변경
Shift + E 선택한 엣지의 Crease 값 설정 (0→날카로움, 1→완전 고정)
N → Item → Edge Crease Sidebar에서 선택 엣지의 Crease 수치 직접 입력

상태 전환 버튼이 꼭짓점 수 프리셋을 바꾸므로, 원형 품질과 폴리곤 비용의 균형을 단계별로 비교하기 쉽습니다.
4각형은 원을 흉내 낸 가장 거친 상태예요. 꼭짓점 수를 늘릴수록 원에 가까워지지만, 동시에 폴리곤 수와 작업 비용도 함께 늘어납니다.

Too Few Sides

폴리곤을 아끼려고 너무 적게 주면 원통이나 버튼이 각져 보여서, 하드서피스에서도 오히려 값싸 보이는 형태가 됩니다.

Right Balance

필요한 실루엣이 유지되는 최소 분할만 쓰면 형태는 충분히 둥글고, 이후 Bevel이나 Boolean도 더 가볍게 다룰 수 있어요.

원 면적 대비 커버율 (꼭짓점이 많을수록 100%에 가까워짐)
4각형 — 63.7%

버튼을 눌러 꼭짓점 수를 바꿔보세요. 주황 점선이 진짜 원입니다.

꼭짓점 수를 늘려야 할 때

  • 부드러운 원통/파이프
  • 구(Sphere) 형태 모델링
  • 렌더링 결과가 각져 보일 때
  • Bevel로 둥근 모서리 표현 시

낮은 꼭짓점 수가 적합할 때

  • 게임용 로우폴리 모델
  • 각진 디자인이 의도인 경우
  • Subdivision Surface로 나중에 부드럽게 할 때
  • 초기 블로킹 단계
꿀팁: 원통 추가 시 기본 꼭짓점 수는 32개입니다. 화면 왼쪽 아래 Add Cylinder 패널에서 즉시 변경 가능해요. 추가 후에는 변경이 불가능하므로, 처음에 정확히 설정하세요.

자주 쓰는 조합

낮은 N각형 + Subdivision Surface
적은 폴리곤으로 시작해 Subdivision으로 부드럽게 만드는 전통적인 방법. 편집은 단순하게, 결과는 부드럽게.
Loop Cut + Bevel (R=1.0)
Loop Cut으로 지지 엣지를 추가하고, Bevel Profile을 1로 설정해 완전한 원형 단면을 만드는 기법.
높은 N각형 + Decimate
충분한 꼭짓점으로 원형을 만든 후, 불필요한 폴리곤을 Decimate로 정리하는 방식.
Subdivision + Crease
Subdivision으로 전체를 부드럽게 만들고, 날카롭게 유지할 모서리에만 Crease=1을 적용. 딱딱한 제품 모델링의 핵심 기법.
Ctrl+B 베벨 + S 세그먼트
모서리를 베벨할 때 S로 세그먼트 수를 늘리면 자연스러운 둥근 모서리가 완성됩니다.
