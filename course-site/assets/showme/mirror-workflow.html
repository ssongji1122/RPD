Mirror Modifier는 대칭 모델링의 핵심 도구입니다. 아래 3단계를 순서대로 따라하면 완벽한 좌우 대칭을 만들 수 있어요.

Step 1 절반 삭제

대칭축(보통 X축)을 기준으로 한쪽 절반을 지웁니다. Edit Mode에서 삭제할 쪽의 버텍스를 모두 선택한 뒤 X → Vertices로 삭제합니다. 오브젝트의 Origin이 정확히 중심에 있어야 깔끔하게 나뉩니다.

Step 2 Mirror Modifier 추가

Properties → Modifier → Add Modifier → Mirror를 선택합니다. Axis를 X로 설정하면 남은 절반이 자동으로 반대쪽에 대칭 복사됩니다. 한쪽만 편집하면 반대쪽이 실시간으로 따라옵니다.

Step 3 Clipping 활성화

Mirror Modifier 설정에서 Clipping 체크박스를 켭니다. 이렇게 하면 중심선의 버텍스가 대칭축을 넘어가지 못하게 고정되어, 이음새 없는 매끄러운 대칭 형태를 유지할 수 있습니다.

팁: 중심선 버텍스가 살짝 어긋나 틈이 생겼다면? 해당 버텍스들을 선택하고 S + X + 0을 눌러 X축 스케일을 0으로 만들면 한 줄로 정렬됩니다.
Blender 공식 문서 — Mirror Modifier
상태 전환 버튼이 워크플로우 단계를 바꾸므로, 한 장면 안에서 모델링 순서를 차례대로 따라가며 설명하기 좋습니다.
대칭 모델링을 시작하기 전에는 보통 완성형을 한 번 떠올려요. 어떤 실루엣을 만들지 감을 잡아두면, 절반 삭제 이후에도 기준을 잃지 않습니다.

Before Workflow

양쪽을 따로 만들면 수정이 두 배로 늘고, 한쪽만 바뀌어도 좌우 균형이 금방 무너져요.

After Workflow

절반을 기준으로 Mirror를 걸어두면, 한쪽만 편집해도 반대쪽이 즉시 맞춰져 반복 수정이 크게 줄어듭니다.

각 단계 버튼을 클릭하여 Mirror Modifier 워크플로우를 단계별로 확인하세요.

캐릭터 모델링

얼굴, 몸통 등 좌우 대칭인 유기체 모델링에서 가장 많이 사용됩니다. 한쪽 눈, 볼, 귀만 만들면 반대쪽이 자동으로 완성됩니다.

로봇/메카 디자인

좌우 대칭인 파츠가 많은 메카닉 모델에서 작업 시간을 절반으로 줄여줍니다. 관절, 팔, 다리 등 반복 구조에 효과적입니다.

차량/비행기

자동차, 비행기, 선박 등 좌우 대칭 탈것의 외형을 모델링할 때 필수입니다. 한쪽 면만 다듬으면 전체 형태가 완성됩니다.

건축/가구

대칭 구조물인 건물 정면, 테이블, 의자 등을 모델링할 때 효율적입니다. 정밀한 대칭이 보장됩니다.

팁: Mirror는 모델링 초반에 추가하는 게 가장 효율적이에요. 나중에 추가하면 Origin 위치를 정확히 맞춰야 해요.
