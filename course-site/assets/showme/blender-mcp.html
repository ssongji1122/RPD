필요한 것 사전 준비물

  • Claude Desktop — claude.ai/download에서 설치
  • Blender 4.0+ — blender.org에서 최신 버전
  • Python 3.10+ — MCP 서버 실행용
  • uv — Python 패키지 실행 도구 (추천)
Claude Desktop = 지휘관, blender-mcp = 통역사, Blender = 실제 작업자. 세 가지가 다 있어야 해요

blender-mcp 핵심 구성

blender-mcp는 두 부분으로 이루어집니다:

  • Blender 애드온 — Blender 내부에서 소켓 서버 실행 (포트 9876)
  • MCP 서버 — Claude와 Blender 소켓 사이를 연결하는 Python 프로세스

Claude Desktop이 MCP 서버를 자동으로 시작하고, MCP 서버가 Blender와 통신해요.

설치 단계 5단계
1

blender-mcp 저장소 다운로드

GitHub에서 blender-mcp를 클론하거나 ZIP으로 받아요.

# 터미널에서 실행 git clone https://github.com/ahuja312/blender-mcp.git cd blender-mcp
2

Blender 애드온 설치

Blender → Edit → Preferences → Add-ons → Install… 에서 addon.py 파일을 선택하고 활성화해요.

# 애드온 파일 경로 blender-mcp/addon.py
3

Blender 소켓 서버 시작

Blender 사이드바(N키) → MCP 탭 → Start MCP Server 클릭. 포트 9876에서 수신 대기를 시작해요.

또는 View3D 상단 메뉴 → MCP → Start Server

4

Claude Desktop 설정 파일 수정

Claude Desktop의 claude_desktop_config.json에 MCP 서버를 등록해요.

// macOS: ~/Library/Application Support/Claude/claude_desktop_config.json // Windows: %APPDATA%\Claude\claude_desktop_config.json { "mcpServers": { "blender": { "command": "uvx", "args": ["blender-mcp"] } } }
5

Claude Desktop 재시작 → 연결 확인

Claude Desktop을 완전히 종료 후 재시작해요. 대화창에 🔌 아이콘이 보이면 MCP 연결 성공! 이제 "큐브 만들어줘" 같은 명령을 해보세요.

확인 방법: Claude 채팅창에서 "Blender에 현재 씬에 오브젝트가 몇 개야?" 라고 물어보세요. 답변이 나오면 연결 완료!

Blender에서 MCP 서버를 먼저 켜야 Claude와 연결돼요

이럴 때 쓰세요

  • 반복 배치·네이밍을 자동화하고 싶을 때
  • Python 스크립트 없이 복잡한 작업을 할 때
  • 씬 정보를 빠르게 파악하고 싶을 때
  • AI와 실시간으로 아이디어를 탐색할 때

주의사항

  • Blender MCP 서버 먼저 켜기 (N → MCP → Start)
  • Claude Desktop 재시작 후 연결 확인
  • 작업 전 .blend 파일 저장 권장
  • 복잡한 작업은 단계별로 나눠서 요청
⚠️ 주의: MCP로 실행한 명령은 Undo(Ctrl+Z)가 적용 안 될 수 있어요. 중요한 작업 전에 파일을 저장해두세요.
