Decimate란?

메쉬의 폴리곤 수를 줄이는 모디파이어입니다. 고폴리곤 모델을 저폴리곤으로 최적화할 때 사용합니다.

고해상도 사진을 압축하는 것 — 파일은 작아지지만 디테일이 줄어듦

3가지 모드

  • Collapse — 점진적으로 꼭짓점을 병합합니다. 가장 일반적으로 사용됩니다.
  • Un-Subdivide — Subdivision의 역연산. 균일하게 나뉜 메쉬를 되돌릴 때 유용합니다.
  • Planar — 같은 평면에 있는 면들을 병합합니다. 평평한 표면의 불필요한 면을 정리할 때 씁니다.

Ratio 핵심 설정

0.0~1.0 범위의 값으로 폴리곤 감소 비율을 조절합니다.

  • 1.0 = 원본 유지 (변화 없음)
  • 0.5 = 면 수 절반으로 감소
  • 0.1 = 90% 감소 (면의 10%만 남음)

결과는 Modifier 패널의 Face Count에서 확인할 수 있습니다.

Decimate가 필요할 때

  • 게임 에셋 최적화 (LOD)
  • 3D 프린팅 파일 크기 줄이기
  • AI 생성 모델 정리 (Meshy, Tripo 등)
  • Sculpt 후 폴리곤 감소

Decimate를 안 쓰는 경우

  • 정확한 토폴로지가 필요할 때
  • 애니메이션용 모델 (변형 시 꼬임 발생)
  • 깔끔한 Edge Flow가 필요할 때
팁: AI로 생성한 3D 모델(Meshy, Tripo)은 폴리곤이 매우 많습니다. Decimate로 적정 수준으로 줄인 뒤 작업하세요!

자주 쓰는 조합

Decimate + Remesh
Remesh로 메쉬를 균일하게 재구성한 뒤 Decimate로 폴리곤 수를 줄입니다. 깔끔한 토폴로지를 유지하면서 최적화할 때 유용합니다.
Sculpt → Decimate
스컬핑으로 디테일을 잡은 뒤 Decimate로 폴리곤을 최적화합니다. 고폴리 스컬프트를 게임이나 렌더용으로 변환할 때 필수입니다.
