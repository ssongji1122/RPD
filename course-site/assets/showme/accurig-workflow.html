AccuRIG 2.0

Reallusion이 2025년 7월에 출시한 완전 무료 standalone 자동 리깅 앱이에요. 캐릭터 메쉬만 있으면 약 1분 만에 본·웨이트가 들어간 리그를 만들어 줍니다.

Blender의 Rigify를 외부 앱으로 빼서 더 정확하고 더 빠르게 만든 버전 — 본 박는 작업을 AI에게 위임한다고 생각하면 돼요.

손가락 + 얼굴 자동

Mixamo와의 가장 큰 차이. AccuRIG는 손가락 관절과 얼굴 본까지 자동으로 잡아 줘요. 손 동작이나 표정 애니메이션이 필요한 캐릭터에 바로 써먹을 수 있습니다.

  • 손가락 15본 자동 배치
  • 얼굴 본은 ActorCore 모션과 호환

사족(Quadruped) 지원

인체형뿐 아니라 네 발 짐승, 새, beastmen까지 자동 리깅이 가능해요. Mixamo는 인체형만 지원하기 때문에, 동물 변형 캐릭터가 있다면 AccuRIG가 거의 유일한 자동 옵션입니다.

캐릭터 종류 메뉴에서 "Human / Quadruped / Bird"를 고르듯 골격 템플릿을 선택하는 구조예요.

ActorCore 모션 라이브러리

Reallusion이 자체적으로 운영하는 모션 캡처 애니메이션 라이브러리예요. AccuRIG로 리깅한 캐릭터에 바로 적용할 수 있게 골격이 호환됩니다.

  • 걷기·달리기·격투 등 수천 개 클립
  • 일부 무료, 고품질은 유료

Blender Rigify 변환

Reallusion이 공식 배포하는 무료 Blender 애드온으로, AccuRIG 본 구조를 Blender Rigify rig로 변환할 수 있어요. 변환 후에는 Rigify 컨트롤러로 자유롭게 포즈를 잡을 수 있습니다.

  • FBX import 후 한 번 클릭으로 변환
  • Rigify의 IK / FK 스위치 그대로 활용

기본 워크플로

Blender 작업 → FBX export → AccuRIG 업로드 → 마커 조정 → Auto Rig → FBX export → Blender import의 다섯 단계를 거쳐요. 메쉬는 Blender에서, 리깅만 AccuRIG에서 처리하는 구조입니다.

  • 메쉬는 T-pose 또는 A-pose가 가장 정확
  • 스케일은 미터 단위로 export

이럴 때 AccuRIG가 정답

  • 인체형 캐릭터에 정확한 자동 리깅이 필요할 때
  • Mixamo보다 손가락 / 얼굴 본까지 자동으로 원할 때
  • ActorCore 모션 캡처 라이브러리와 함께 작업할 때
  • 사족 동물·새·beastmen 같은 비인체형 골격이 필요할 때
  • 리깅 후 Blender Rigify 컨트롤러로 포즈 잡고 싶을 때
  • Reallusion 계정을 만들 수 있는 환경일 때

다른 도구가 나은 경우

  • 차량·식물·건축처럼 비유기체 오브젝트는 자동 리깅 불가 — 수동 Armature가 답
  • 인터넷 / 계정 / 앱 설치가 막히는 환경 (학교·회사 PC) — Mixamo 웹이 낫다
  • 아주 단순한 박스 캐릭터 — Rigify 또는 수동 본이 더 빠를 수 있음
  • 비표준 골격 (여러 팔·촉수·변형 생물) — 자동 감지 실패율 높음
  • 오프라인 작업이 필수일 때 — 클라우드 변환은 인터넷 필요
학생이 가장 많이 막히는 지점
1. 스케일 — Blender 미터 단위로 export 안 하면 AccuRIG에서 캐릭터가 점처럼 작게 보여요. Ctrl+A > Apply > Scale 후 export하세요.
2. 포즈 — 메쉬가 T-pose 또는 A-pose 상태여야 마커가 정확히 잡혀요. 다른 포즈면 Auto Rig 실패율이 올라갑니다.
3. 계정 — Reallusion 계정이 있어야 앱 실행 가능. 수업 전 미리 가입해 두는 게 좋아요.
