Edit Mode에서 메시를 변형하는 핵심 도구 5가지 계열을 알아봅니다. 각 도구의 역할과 변형을 이해하면 모델링 속도가 크게 빨라집니다.

1. Extrude 계열

핵심 Extrude Region

면/변/점을 법선 방향으로 밀어냅니다. Edit Mode에서 가장 많이 쓰는 기본 Extrude입니다.

점토를 손가락으로 눌러 밀어내는 것처럼, 선택한 면이 바깥으로 튀어나옵니다

핵심 Extrude Along Normals

각 면의 개별 법선(Normal) 방향으로 돌출합니다. 곡면에서 균일한 두께로 돌출할 때 유용합니다.

각 면이 자기만의 방향을 알고, 그 방향으로 정확히 밀려 나갑니다

Extrude Individual Faces

선택된 각 면이 개별적으로 돌출됩니다. 여러 면을 동시에 선택해도 각각 독립적으로 움직입니다.

Extrude Manifold

인접 면과 자동 병합되는 Extrude입니다. Boolean 없이도 깔끔하게 형태를 파고 들어갈 수 있습니다.

Extrude Edges / Vertices

모서리(Edge)만 또는 꼭짓점(Vertex)만 돌출합니다. 면이 아닌 선이나 점에서 새로운 지오메트리를 만듭니다.

Extrude to Cursor

마우스 위치로 직접 돌출합니다. 빠르게 점/면을 원하는 곳에 뻗어나갈 때 사용합니다.

Extrude Repeat / Spin

Repeat: 마지막 Extrude를 반복합니다.
Spin: 축을 기준으로 회전 돌출하여 선반(Lathe) 효과를 만듭니다.

Extrude Individual메뉴각 면이 독립적으로 돌출
Extrude Manifold메뉴인접 면 자동 병합

2. Loop Cut 계열

필수 Loop Cut & Slide

메시에 루프 컷을 삽입하고, 마우스로 위치를 슬라이드할 수 있습니다. 디테일 추가의 기본입니다.

빵을 칼로 한 줄 쭉 자르는 것처럼, 메시를 깔끔하게 분할합니다

Offset Edge Loop Cut

기존 에지 루프 양쪽에 오프셋된 루프 2개를 동시에 생성합니다. 하드서페이스 모델링에서 날카로운 모서리를 만들 때 유용합니다.

Edge Slide

기존 에지 루프를 모서리를 따라 슬라이드합니다. 토폴로지를 바꾸지 않으면서 루프 위치만 조정할 수 있습니다.

도구 단축키 핵심 차이
Loop Cut & SlideCtrl+R새 루프 삽입 + 슬라이드
Offset Edge Loop CutShift+Ctrl+R기존 루프 양쪽에 오프셋 루프 2개
Edge SlideGG기존 루프를 모서리 따라 이동
3. Knife 계열

다용도 Knife Tool

면 위에 자유롭게 선을 그어 커팅합니다. 마우스로 시작점과 끝점을 찍어 원하는 형태로 면을 분할합니다.

실제 칼로 종이를 자르듯, 원하는 경로를 자유롭게 그어 자를 수 있습니다

Knife Topology

기존 메시 구조(토폴로지)를 기반으로 커팅합니다. 기존 꼭짓점과 모서리를 따라 정확하게 분할합니다.

Knife Project

다른 오브젝트의 윤곽을 투영하여 커팅합니다. 글자나 로고 모양을 메시에 새길 때 유용합니다.

Bisect

평면으로 메시를 반분할합니다. 한쪽을 삭제하거나 채울 수 있어 정확한 절단에 유용합니다.

Intersect (Knife / Boolean)

Intersect Knife: 면의 교차 부분을 따라 커팅합니다.
Intersect Boolean: 면의 교차 부분에 Boolean 연산을 적용합니다.

도구 단축키 핵심 차이
KnifeK자유 경로로 커팅
Knife Topology메뉴기존 구조 기반 커팅
Knife Project메뉴다른 오브젝트 윤곽 투영
Bisect메뉴 / Toolbar평면으로 반분할
Intersect KnifeCtrl+F → K면 교차 부분 커팅
Intersect BooleanCtrl+F → B면 교차 부분 Boolean
4. Inset Faces

필수 Inset Faces

면 안쪽에 새로운 면을 삽입합니다. 창문 프레임, 패널 라인 등 면 내부에 디테일을 추가할 때 핵심 도구입니다. I를 누른 뒤 I를 한 번 더 누르면 Individual 모드로 전환되어 각 면이 개별적으로 Inset됩니다.

액자 틀처럼, 면 안쪽에 한 겹 더 작은 면이 생깁니다
5. Bevel Tool

필수 Bevel Tool

모서리 또는 꼭짓점을 깎아 둥근 모서리를 만듭니다. 현실 세계에서 완벽히 날카로운 모서리는 없으므로, 사실적인 모델링에 필수입니다. 스크롤 휠로 세그먼트 수를 조절할 수 있습니다.

  • Ctrl+B — 모서리(Edge) 베벨
  • Ctrl+Shift+B — 꼭짓점(Vertex) 베벨
각진 나무 모서리를 사포로 둥글게 다듬는 것과 같습니다
개념 이해 시각화
개별 꼭짓점(Vertex)을 선택하여 이동·변형합니다. 가장 세밀한 제어가 가능한 모드입니다.

버튼을 눌러 같은 시작 면에서 Extrude 방식이 어떻게 달라지는지 직접 확인하세요.

버튼을 눌러 Extrude 모드를 전환하세요.

이런 모양을 만들 때

  • 돌출 (Extrude) — 벽, 기둥, 돌출부
  • 분할 (Loop Cut / Knife) — 세부 디테일 추가
  • 삽입 (Inset) — 창문 프레임, 패널 라인
  • 깎기 (Bevel) — 부드러운 모서리, 하이라이트

도구 선택 가이드

  • 직선 컷 → Loop Cut Ctrl+R
  • 자유 컷 → Knife K
  • 면 돌출 → Extrude E
  • 면 삽입 → Inset I
  • 모서리 둥글게 → Bevel Ctrl+B

추천 도구 조합

Extrude + Loop Cut
기본 모델링 워크플로우. Extrude로 형태를 만들고 Loop Cut으로 디테일을 추가합니다.
Inset + Extrude
창문, 환기구 만들기. Inset으로 면을 삽입한 뒤 Extrude로 안쪽으로 밀어 구멍을 만듭니다.
Loop Cut + Bevel
하드서페이스 디테일. Loop Cut으로 에지를 추가하고 Bevel로 모서리를 다듬어 사실적인 형태를 만듭니다.
팁: Tab 키로 Object Mode와 Edit Mode를 전환합니다. Edit Mode에 진입해야 위 도구들을 사용할 수 있습니다. 메시 오브젝트를 선택한 상태에서 Tab을 누르세요.
