Core 컴포지팅이란?

렌더링된 이미지를 노드 기반으로 후처리(Post-processing)하는 과정입니다. 색상 보정, 효과 추가, 여러 렌더 레이어 합성 등을 수행합니다.

사진 촬영 후 포토샵에서 보정하는 것과 같아요

노드 노드 기반 워크플로우

Render Layers 노드(입력) → 효과 노드(처리) → Composite 노드(출력) 순서로 연결합니다. 각 노드는 독립적으로 ON/OFF 가능합니다.

효과 주요 컴포지팅 노드

  • Bright/Contrast: 밝기/대비 조절
  • Glare: 빛 번짐 효과 (Bloom)
  • Color Balance: 색상 톤 보정
  • Vignette (Lens Distortion): 가장자리 어둡게

주요 단축키 / 설정

Compositing Workspace 상단 Workspace 탭에서 전환
Use Nodes 체크 Compositing 에디터 상단 체크박스 활성화
Ctrl + Shift + Click Viewer Node 연결 (중간 결과 미리보기)

인터랙티브 데모 — 각 노드를 ON/OFF해 보세요:

각 버튼을 클릭하면 해당 노드를 ON/OFF합니다. 효과가 이미지에 쌓여가는 과정을 관찰하세요.

컴포지팅이 필요할 때

  • 렌더 후 색감/톤 보정
  • Bloom/Glare 빛번짐 효과 추가
  • 여러 렌더 패스 합성 (AO, Shadow 등)
  • 배경 합성 (크로마키 등)

컴포지팅이 불필요한 경우

  • 단순한 테스트 렌더
  • 외부 소프트웨어(After Effects 등)에서 후처리할 때
  • 게임 에셋처럼 후처리가 필요 없는 경우
팁: Ctrl+Shift+Click으로 Viewer Node를 노드에 연결하면 각 단계의 중간 결과를 확인할 수 있습니다. 효과를 하나씩 추가하면서 변화를 비교하세요.

자주 쓰는 노드 조합

Glare + Color Balance
빛 번짐으로 드라마틱한 분위기를 만들고, 색상 톤으로 영화 같은 느낌을 추가합니다.
Render Layers + Alpha Over
여러 렌더 레이어를 합성하여 캐릭터와 배경을 분리하거나 합치는 작업에 사용합니다.
