흐름 컴포지팅이란?

렌더링 결과물에 후처리(Post-Processing)를 적용하는 작업입니다. Blender의 Compositor는 노드 기반으로, 렌더 이미지를 입력받아 다양한 효과를 추가한 뒤 최종 출력합니다.

사진 편집에서 필터를 적용하는 것과 비슷해요. 촬영 후 보정하는 단계입니다

입력 Render Layers

렌더링된 이미지 데이터를 노드 그래프에 공급하는 입력 노드입니다. Image(색상), Alpha(투명도), Depth(깊이), Normal(법선) 등 다양한 Pass를 출력합니다.

  • Combined: 최종 합성 이미지
  • Z Depth: 카메라 거리 정보
  • Mist: 안개 효과용 거리 정보

효과 Glare / Color Balance

Glare는 밝은 부분에서 빛이 번지는 효과(Bloom, Streaks)를 추가합니다. Color Balance는 Lift(어두운 톤)/Gamma(중간 톤)/Gain(밝은 톤)으로 색상을 보정합니다.

영화 후반 작업에서 색보정(Color Grading)하는 과정이에요

출력 Composite / Viewer

Composite 노드는 최종 결과를 파일로 저장합니다. Viewer 노드는 Compositor 배경에서 실시간 프리뷰를 보여줍니다. 둘 다 연결해야 확인과 저장이 가능합니다.

노드를 드래그하여 이동하세요. Glare/Color Balance를 토글하면 연결이 바뀝니다.

컴포지팅이 필요할 때

  • 렌더 결과에 Bloom/Glare 추가
  • 색보정 (Color Grading)
  • 배경 합성 (Green Screen 키잉)
  • Depth of Field 후처리
  • 비네팅, 렌즈 왜곡 효과

컴포지팅이 불필요한 경우

  • 단순 테스트 렌더
  • 외부 편집기(After Effects 등)에서 합성
  • EEVEE Bloom으로 충분한 경우
팁: 컴포지팅은 렌더링 후에 적용되므로 재렌더 없이 효과를 조정할 수 있습니다. "Use Nodes" 체크박스를 반드시 켜야 Compositor가 활성화됩니다. Ctrl+Shift+클릭으로 중간 노드의 결과를 바로 확인하세요.

자주 쓰는 노드 조합

Render Layers → Glare → Composite
기본적인 Bloom 효과 추가. Glare 타입을 Fog Glow로 설정하면 자연스러운 빛 번짐을 만듭니다.
Render Layers → Color Balance → Composite
영화 같은 색감 보정. Lift로 그림자 색, Gain으로 하이라이트 색을 조정합니다.
