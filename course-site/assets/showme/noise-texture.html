Noise Texture란? 기초

수식으로 만드는 무작위 패턴입니다. 이미지 파일 없이도 자연스러운 얼룩, 구름, 나무결 같은 무늬를 만들 수 있어요. 해상도 제한이 없고 아무리 확대해도 선명합니다.

대리석 무늬를 생각해보세요 — 자연이 "수학적 규칙"으로 만든 랜덤 패턴. Noise Texture는 컴퓨터가 그 규칙을 흉내 냅니다

핵심 파라미터 필수

  • Scale — 패턴 크기. 작을수록 넓은 무늬, 클수록 촘촘한 무늬
  • Detail — 세밀도. 높을수록 자잘한 디테일 추가
  • Roughness — 거친 정도. 0=매끄러운 패턴, 1=거친 패턴
  • Distortion — 왜곡. 패턴을 비틀어 유기적 형태 생성

출력 소켓 2개 중요

  • Fac (흑백) — 0~1 float 값. Roughness, Bump 등에 연결
  • Color (컬러) — RGB 색상 출력. Base Color에 직접 연결 가능

보통 Fac → Color Ramp 조합이 가장 많이 쓰입니다.

Noise Texture 파라미터 탐색

Scale 10
Detail 4
Distortion 0.0

슬라이더를 조절해 Noise 패턴 변화를 확인하세요 (Perlin Noise 시뮬레이션)

팁: Scale 5~15 범위가 자연스러운 재질감에 적합합니다. Scale이 너무 크면 패턴이 점처럼 보이고, 너무 작으면 뭉개져요.

🪨 자연스러운 재질

  • Noise → Color Ramp → Base Color = 대리석/돌 표면
  • Noise → Roughness = 불균일한 광택
  • Noise → Bump = 미세한 표면 요철

🌫️ 환경 효과

  • Volume Scatter에 연결 → 안개/연기
  • Displacement → 지형 생성
  • Emission 강도 조절 → 불꽃 효과

🎲 절차적(Procedural)의 장점

  • 이미지 파일 불필요 → 가벼움
  • 해상도 무한 → 확대해도 선명
  • 파라미터만 바꾸면 무한 변형

🔗 핵심 연결 패턴

  • Noise → Color Ramp → Base Color
  • Noise (Fac) → Roughness 입력
  • Noise → Bump → Normal 입력
