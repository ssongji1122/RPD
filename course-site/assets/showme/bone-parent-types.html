Bone끼리 부모-자식을 맺을 때 두 가지 종류가 있어요. 어떤 걸 고르느냐에 따라 본의 위치와 회전 방식이 달라져요.

Connected 체인처럼 붙이기

자식 본의 Head가 부모 본의 Tail에 강제로 붙어요. 두 본이 한 줄로 이어진 체인을 만들어요.

소시지 두 개를 끈으로 매듭지어 이은 거예요. 매듭이 풀리지 않아요.
왜: 팔, 척추, 손가락처럼 관절이 항상 붙어 있는 부위는 분리되면 안 되니까요.

Free (Keep Offset) 자유롭게 떨어뜨리기

계층 관계는 유지하지만, 자식 본의 위치는 원래 자리 그대로예요. 부모와 자식 사이에 빈틈이 있어도 괜찮아요.

엄마 손을 잡고 걷는 아이처럼, 떨어져 있어도 같이 따라가요.
왜: 쇄골과 상완처럼 회전축이 어긋난 부위, 또는 컨트롤 본은 위치를 자유롭게 둬야 해요.

언제 뭐 쓰나 한 줄 판단

두 본이 한 직선 체인을 이루고, 관절이 항상 붙어 있어야 한다면 Connected. 떨어져 있어야 자연스러우면 Free예요.

  • 팔, 척추, 손가락, 꼬리 → Connected
  • 쇄골-상완, 골반-허벅지, 컨트롤 본 → Free
  • 로봇 파츠처럼 독립 회전이 목적 → Free

주요 단축키 (Edit Mode 기준)

Ctrl+P → Connected: 자식 본의 Head가 부모 Tail로 강제 이동
Ctrl+P → Keep Offset: 부모-자식만 묶고 위치는 그대로
Alt+P → Clear Parent: 부모-자식 관계 완전 해제
Alt+P → Disconnect Bone: 부모 관계는 유지, Connected만 해제 (붙어 있던 Head를 떼어냄)
핵심: 먼저 자식 본을 선택하고, 마지막에 부모 본을 Shift 추가 선택해요. 그 다음 Ctrl+P를 눌러요. 순서가 바뀌면 부모-자식이 거꾸로 잡혀요.

슬라이더를 움직이면 부모/자식 본이 같이 회전해요. Connected와 Free 모드의 가장 큰 차이는 "자식 Head 위치"예요.

Connected 추천

  • 팔 체인 (Upper Arm → Forearm → Hand)
  • 척추 (Spine.001 → Spine.002 → Spine.003)
  • 손가락 마디 (Finger.01 → .02 → .03)
  • 꼬리, 안테나, 머리카락 가닥
  • 관절이 절대 떨어지면 안 되는 부위

Free 추천

  • 쇄골 → 상완 (회전축이 어긋남)
  • 골반 → 허벅지 (좌우로 벌어진 시작점)
  • 척추 → 쇄골 / 머리
  • IK Target, Pole Target 같은 컨트롤 본
  • 로봇 파츠처럼 독립적으로 움직이는 부위
로봇 리깅 팁: 로봇은 파츠가 독립적으로 회전하는 게 자연스러워요. 대부분 Ctrl+P → Keep Offset으로 묶고, "계층 정확성"만 신경 쓰면 돼요. Connected는 한 직선 체인일 때만 써요.
자주 하는 실수: 쇄골을 상완 부모로 만들 때 Connected를 골라버리는 거예요. 그러면 상완 Head가 쇄골 Tail로 끌려가서 어깨 위치가 망가져요. 이럴 땐 무조건 Keep Offset이에요.
실수 회피 체크리스트

리깅 직전에 확인

  • 자식 본을 먼저 선택했나? (마지막이 부모)
  • 두 본이 한 직선 체인인가? → Connected
  • 두 본의 Head가 떨어져 있는가? → Free
  • 모드는 Edit Mode인가? (Object Mode 아님)

잘못 연결됐을 때

  • 완전 해제: Alt+P → Clear Parent
  • 관계는 두고 떼기만: Alt+P → Disconnect Bone
  • 다시 묶기: 자식 → 부모 순서로 선택 후 Ctrl+P
