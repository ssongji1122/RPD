Blender에는 위치를 나타내는 두 가지 기준점이 있습니다. 헷갈리기 쉽지만 역할이 완전히 다릅니다.

Origin 오리진

오브젝트 자체에 붙어있는 기준점.
오브젝트를 이동하면 같이 따라다닙니다.

회전·스케일 변환의 중심이 되고, 오브젝트의 위치 좌표(X, Y, Z)가 이 점을 기준으로 표시됩니다.

3D Cursor 커서

뷰포트에 독립적으로 떠있는 커서.
어떤 오브젝트에도 속하지 않습니다.

새 오브젝트 추가 위치, 스냅 기준점, 피벗 포인트 등으로 활용됩니다.

Origin 단축키

  • 우클릭 → Set Origin
  • Ctrl+Alt+Shift+C
  • Object > Set Origin 메뉴

3D Cursor 단축키

  • Shift+RMB — 클릭 위치로 이동
  • Shift+S — Snap 메뉴
  • Alt+G on cursor — 원점 리셋

상태 전환 버튼으로 기준점을 바꿔 보면서 Origin과 3D Cursor가 각각 어떤 역할을 맡는지 장면별로 분리해 읽습니다.
Origin은 오브젝트에 붙어 다니는 손잡이예요. 큐브를 움직이면 주황 Origin도 함께 따라오고, 회전과 스케일의 중심도 이 점을 기준으로 잡힙니다.

Confused

Origin과 Cursor를 같은 점으로 생각하면, 왜 생성 위치와 회전 중심이 다르게 반응하는지 계속 헷갈리게 됩니다.

Separated

Origin은 오브젝트 기준, 3D Cursor는 장면 기준이라고 나누면 새 오브젝트 생성과 피벗 설정이 훨씬 예측 가능해져요.

좌클릭: 큐브 선택·이동 | 우클릭(터치: 빈 곳 탭): 3D Cursor 이동

Origin을 쓰는 경우

  • 오브젝트 기준점을 특정 위치로 변경할 때
  • 회전 중심을 바꾸고 싶을 때
  • 여러 오브젝트를 Origin 기준으로 정렬
  • 부모-자식 관계(Parenting) 설정
  • 애니메이션 키프레임 기준 설정

3D Cursor를 쓰는 경우

  • 새 오브젝트를 원하는 위치에 추가
  • Pivot Point를 3D Cursor로 설정해 회전
  • 스냅 기준점으로 사용
  • Origin을 Cursor 위치로 이동
  • 정확한 좌표에 커서를 놓고 작업
