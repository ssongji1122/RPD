핵심 Dope Sheet란?

애니메이션에서 키프레임의 타이밍과 위치를 편집하는 에디터예요. 여러 오브젝트와 채널의 키프레임을 한눈에 볼 수 있고, 드래그로 타이밍을 빠르게 조정할 수 있어요.

영상 편집의 타임라인 트랙과 같아요. 키프레임이 클립처럼 나열돼 있고, 드래그로 위치를 바꿀 수 있어요.

비교 Graph Editor와 차이

Dope Sheet는 키프레임이 언제 있는지 — 타이밍과 위치 편집에 집중해요.
Graph Editor는 키프레임 사이의 보간 곡선 — 속도와 가속도 편집에 집중해요.

  • 타이밍 조정 → Dope Sheet
  • 이징·속도 조정 → Graph Editor

뷰 모드 채널 보기

Summary, Object, Action 등 다양한 뷰 모드로 원하는 채널만 볼 수 있어요. 채널이 너무 많으면 Filter로 오브젝트를 한정하세요.

  • Summary — 전체 요약
  • Object — 오브젝트별 키프레임
  • Action — 액션 데이터 편집

흔한 실수 방지: Graph Editor와 헷갈리기 쉬워요. Dope Sheet는 타이밍(언제), Graph Editor는 속도·곡선(어떻게) 편집이라고 기억하세요. 채널이 너무 많이 보이면 헤더의 Filter 버튼으로 오브젝트를 한정할 수 있어요.

Dope Sheet를 직접 조작해 보세요. 키프레임 다이아몬드를 드래그해서 타이밍을 바꾸거나, 빨간 플레이헤드를 드래그해 보세요.

키프레임(◆)을 드래그해 타이밍을 조정하세요 · 플레이헤드(빨간 선)도 드래그 가능

이럴 때 사용해요

  • 애니메이션 타이밍을 빠르게 조정할 때
  • 여러 오브젝트의 키프레임을 동시에 맞춰야 할 때
  • 이징/보간보다 키프레임 위치 자체를 바꿀 때
  • 특정 프레임에서 동작 시작점을 앞뒤로 밀 때

이럴 때는 다른 에디터를

  • 속도 곡선·이징 편집 → Graph Editor
  • NLA 액션 믹싱 → NLA Editor
  • Shape Key 애니메이션 → Shape Key Editor
  • 마스크 애니메이션 → Mask Editor
