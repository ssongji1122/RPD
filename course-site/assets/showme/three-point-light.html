Key Key Light (주광)

장면의 주된 빛. 가장 밝고 강한 조명으로, 피사체의 형태와 그림자를 결정합니다. 보통 피사체의 45도 위, 좌측 또는 우측에 배치합니다.

태양처럼 장면의 '주인공' 조명이에요

Fill Fill Light (보조광)

Key Light가 만든 그림자를 부드럽게 채워주는 보조 조명. Key Light보다 약하며(보통 50~70% 밝기), Key Light 반대편에 배치합니다.

그늘진 곳을 살짝 밝혀주는 반사판 역할이에요

Rim Rim/Back Light (역광)

피사체 뒤에서 비추는 빛. 윤곽선(Edge)을 만들어 배경과 분리합니다. 피사체 뒤쪽 위에서 아래로 비춥니다.

역광으로 머리카락이 빛나는 효과, 실루엣을 만드는 빛이에요

주요 단축키

Shift + A Light 추가 (Add → Light)
G 조명 이동 (Grab)
R 조명 회전 (Rotate)
📖 Blender 공식 문서 — Light Object
Key Light는 주광원으로 가장 밝고 강한 빛입니다. 피사체의 형태를 규정하고 주 그림자를 만듭니다.

적용 후

3점 조명이 필요할 때

  • 캐릭터/인물 렌더링
  • 제품 비주얼라이제이션
  • 포트폴리오용 프레젠테이션 렌더
  • 단편 애니메이션 장면

다른 조명법을 쓰는 경우

  • HDRI 환경 조명 (야외 장면)
  • 단일 Point Light (게임용 간단 조명)
  • 스튜디오 다중 조명 (광고, 패션)
팁: Blender에서 Shift+A → Light → Area Light를 추가하면 면적 조명으로 더 자연스러운 그림자를 만들 수 있습니다. Key Light의 밝기를 기준으로 Fill은 50~70%, Rim은 80~120% 정도로 설정하세요.

조명 유형 비교

Point Light
모든 방향으로 빛 발산. 전구처럼 동작. 부드러운 Fill Light에 적합합니다.
Area Light
면적 광원. 부드러운 그림자. Key Light에 가장 많이 사용됩니다.
Spot Light
원뿔 형태로 빛 집중. 무대 조명처럼 특정 부분을 강조할 때 좋습니다.
Sun Light
평행광. 야외 태양 표현에 적합. 위치와 상관없이 방향만 영향을 줍니다.
