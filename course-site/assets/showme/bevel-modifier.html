Core Bevel Modifier란?

모든 모서리 또는 선택한 모서리를 자동으로 깎아 부드러운 경사면을 만드는 모디파이어입니다. Bevel Tool(Ctrl+B)과 달리 비파괴적으로 동작하여, 언제든 값을 수정하거나 제거할 수 있습니다.

나무 테이블 모서리를 사포로 둥글게 깎는 것과 같아요

Key Width vs Segments

Width: 베벨 너비(깎는 폭). Segments: 분할 수(많을수록 부드러운 곡면).

  • Segments = 1 → 45도 chamfer (평면 깎기)
  • Segments ≥ 3 → 부드러운 라운드 곡면

Limit Limit Method

어떤 모서리에 베벨을 적용할지 결정합니다.

  • None — 전체 모서리
  • Angle — 각도 기준 자동 선택
  • Weight — Bevel Weight 지정 모서리만
  • Vertex Group — 버텍스 그룹 기반

Fix Weighted Normal

Bevel 후 Normal 방향이 꼬여 셰이딩이 이상해지는 문제를 해결합니다. Weighted Normal 모디파이어를 Bevel 아래에 추가하면 깔끔한 셰이딩을 유지할 수 있습니다.

더보기
입문 단계에서는 `Segments`, `Width`, `Weighted Normal` 세 가지만 먼저 익혀도 충분합니다. 선택 적용이 필요할 때만 `Limit Method = Weight`까지 확장하면 됩니다.

Bevel Modifier가 필요할 때

  • 하드서페이스 모델 (금속, 기계, 전자제품)
  • 현실적인 가구 모서리
  • 건축 디테일

Bevel Modifier를 안 쓰는 경우

  • 유기적 형태 (캐릭터, 생물)
  • 이미 Subdivision으로 부드러운 모델
  • 유기체 모델링에는 Subdivision Surface가 더 적합
  • 모서리 하나만 깎을 때는 Bevel Tool(Ctrl+B)이 더 빠름

→
결과원하는 모서리만 깔끔하게 Bevel + 음영 정리

흔한 실수 & 해결법

모든 모서리에 균일 Bevel이 걸림
Limit Method를 Weight로 변경 → Ctrl+Shift+E로 원하는 엣지에 가중치 부여.
Bevel 후 음영이 지저분함
Bevel 아래에 Weighted Normal Modifier를 추가하세요. 스택 순서가 중요합니다.
Bevel Tool과 Modifier 혼동
Tool(Ctrl+B)은 즉시 적용, Modifier는 비파괴적. 수정 가능성이 있으면 Modifier.
비균일 Scale에서 Bevel 찌그러짐
Ctrl+A → Apply Scale 후 Bevel을 적용하세요.
팁: Bevel Modifier의 Limit Method를 'Weight'로 설정하고, 원하는 모서리에만 Ctrl+Shift+E로 Weight를 주면 선택적으로 베벨을 적용할 수 있어요. Weighted Normal 모디파이어를 항상 함께 사용하세요!
