Build Modifier란?

메쉬의 면을 순서대로 나타나게 하거나 사라지게 하는 애니메이션 모디파이어입니다. 건물이 지어지듯 면이 하나씩 추가되어 완성됩니다.

레고 블록을 하나씩 쌓아 올리는 타임랩스

Start / Length

Start: 애니메이션이 시작되는 프레임입니다. Length: 전체 면이 나타나는 데 걸리는 프레임 수입니다. Reversed 옵션을 켜면 면이 하나씩 사라지는 분해 효과를 만들 수 있습니다.

Seed

면이 나타나는 순서를 랜덤화합니다. 같은 Seed 값을 사용하면 항상 같은 순서로 면이 나타납니다.

참고 사항

Space 타임라인 재생 — Build 효과를 확인하는 기본 방법
Properties 패널 Modifier 탭에서 Start, Length, Seed 값을 설정
📖 Blender 공식 문서 — Build Modifier
상태 전환 버튼은 조립, 분해, 랜덤 순서처럼 모션 상태를 바꾸고, 같은 메시가 어떤 내러티브로 읽히는지 비교하게 합니다.
Build Modifier는 메쉬를 만드는 게 아니라 보이는 순서를 제어합니다. 같은 모델도 정방향 조립인지, 역방향 분해인지, 랜덤 시드인지에 따라 완전히 다른 모션 문법이 됩니다.

Flat Reveal

순서와 방향을 의식하지 않으면 단순히 켜졌다 꺼지는 것처럼 보여서, 조립이나 분해라는 서사가 잘 읽히지 않습니다.

Readable Motion

정방향, 역방향, 랜덤 순서를 분리해서 보면 같은 메시도 조립 애니메이션, 파괴 연출, 데이터 리빌처럼 다른 역할을 맡길 수 있어요.

재생 버튼을 눌러 Build Modifier 효과를 확인하세요. 랜덤 순서로 Seed 변경 효과를 볼 수 있습니다.

Build가 필요할 때

  • 모션 그래픽 (건물 조립 애니메이션)
  • 타이틀 시퀀스
  • 데이터 시각화
  • 파괴 효과 (Reversed)

Build를 안 쓰는 경우

  • 정적 모델 (애니메이션 불필요)
  • 실시간 렌더링
  • 정밀한 면 단위 제어가 필요할 때
팁: Build는 애니메이션 모디파이어입니다. 타임라인 재생(Space)으로 효과를 확인하세요!
