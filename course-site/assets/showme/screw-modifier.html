Screw Modifier란?

프로파일(단면)을 축 기준으로 회전시켜 3D 형태를 만드는 모디파이어입니다. 선반(Lathe) 가공과 동일한 원리로, 병, 꽃병, 나사 등을 만들 때 사용합니다.

도자기 물레 위에서 흙을 돌려 형태를 만드는 것

Axis & Angle

Axis: 회전 축(X/Y/Z)을 선택합니다.

Angle: 회전 각도입니다. 360도는 완전한 회전체, 180도는 반쪽만 만들어집니다.

Steps: 회전의 분할 수입니다. 높을수록 부드러운 곡면이 됩니다.

Screw (나사산)

Screw 값을 0이 아닌 값으로 설정하면, 프로파일이 회전하면서 동시에 축 방향으로 이동합니다. 이렇게 나사산이나 스프링 형태를 만들 수 있습니다.

연속 조절 슬라이더는 Angle과 Screw 값을 연속적으로 바꾸고, 초기화 버튼은 현재 실험 상태를 기준값으로 되돌립니다.
Screw Modifier는 단면을 돌려서 입체를 만드는 과정입니다. Angle은 얼마나 감쌀지, Screw는 회전하면서 축 방향으로 얼마나 이동할지를 정해 순수 회전체와 나사형 구조를 가릅니다.

Axis Mistake

회전 각도와 축 이동의 차이를 헷갈리면 꽃병을 만들려다 열린 쉘이 나오거나, 원통을 만들려다 엉뚱한 나사 형태가 생깁니다.

Lathe to Thread

Angle은 회전체의 닫힘 정도, Screw는 축 이동량이라고 나눠 보면 선반 가공형 병부터 스프링과 볼트까지 같은 원리로 설명할 수 있어요.

Angle 360°
Screw 0

프로파일이 축을 기준으로 회전하여 3D 형태가 만들어지는 과정을 관찰해 보세요.

Screw가 필요할 때

  • 꽃병 / 병 / 컵 (선반 가공)
  • 나사 / 볼트 / 스프링
  • 기둥 / 파이프
  • 나선 계단

Screw를 안 쓰는 경우

  • 비대칭 형태
  • 복잡한 형태 (직접 모델링이 나음)
  • 유기적이고 불규칙한 모양
팁: 프로파일은 반드시 축 한쪽에만 있어야 합니다! 중심에서 한쪽으로만 그려야 올바른 결과가 나옵니다. 프로파일이 축을 걸쳐 있으면 겹치는 면이 생깁니다.

자주 쓰는 조합

Screw + Subdivision Surface
Screw로 기본 형태를 만들고 Subdivision으로 부드러운 표면을 만듭니다. Steps를 낮추고 Subdivision으로 보완하면 효율적입니다.
Screw + Array
나사산 한 바퀴를 Screw로 만들고 Array로 반복하여 긴 나사를 만듭니다.
