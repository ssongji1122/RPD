Draw 기본

크레파스로 칠하듯이 표면을 올려요. X로 빠른 선택, Ctrl 누르면 파내기.

크레파스로 종이 위를 칠하는 것처럼 — 누르는 만큼 솟아오르는 기본 브러시예요

Clay Strips 덧붙이기

찰흙 띠를 붙이듯이 넓은 면에 층층이 쌓아요.

납작하게 민 찰흙 띠를 표면에 하나씩 붙이는 느낌이에요

Clay 밀어붙이기

손바닥으로 찰흙을 누르듯이 밀면서 쌓아요. Draw보다 평탄한 결과.

손바닥으로 찰흙을 꾹 누르는 것 — Draw보다 면이 고르게 쌓여요

Inflate 부풀리기

풍선처럼 바깥으로 부풀려요. 볼이나 근육 강조.

표면 아래에서 공기를 불어넣는 것처럼 — 균일하게 부풀어 오르는 느낌이에요

Crease 홈 파기

송곳으로 긋듯이 깊고 날카로운 주름. Shift+C로 빠른 선택.

손톱으로 찰흙 표면을 꾹 긁듯이 — 날카롭고 깊은 선이 생겨요

Grab 잡아 당기기

찰흙을 잡고 끌어당겨요. G로 빠른 선택.

반죽을 엄지와 검지로 잡고 당기는 것처럼 — 형태 전체를 크게 변형해요

Snake Hook 끌어내기

찰흙 끝을 잡고 길게 늘여요. K로 빠른 선택.

뱀처럼 늘어나는 느낌 — 뿔이나 촉수를 뽑아낼 때 써요

Smooth 정리

손가락으로 쓸듯이 정리. Shift 누르고 있으면 임시 Smooth.

손가락으로 표면을 부드럽게 문지르는 것처럼 — 울퉁불퉁한 부분을 다듬어요

얼굴 만들기

1. Grab
전체 실루엣을 잡아요. 코, 광대뼈, 턱 라인을 큰 덩어리로 먼저 잡아당깁니다.
2. Clay Strips
볼과 이마에 볼륨을 층층이 쌓아요. 자연스러운 살집 표현에 최적입니다.
3. Crease
눈 라인, 입술 경계, 콧방울 주름처럼 날카로운 경계를 새겨요.
4. Smooth
마무리로 울퉁불퉁한 부분을 정리해요. Shift 누르고 빠르게 다듬습니다.

근육/관절 표현

1. Inflate
근육 덩어리가 될 부분을 풍선처럼 부풀려요. 이두근, 허벅지 등에 씁니다.
2. Crease
근육 사이 경계선과 관절 주름을 새겨요. 힘줄 라인 표현에도 효과적입니다.
3. Clay
근육 위에 미세한 볼륨을 더 쌓아요. 평탄하게 쌓이므로 고른 볼륨 추가에 적합합니다.
4. Smooth
전체를 부드럽게 다듬어 자연스러운 근육 표면을 완성합니다.

뿔/촉수 만들기

1. Snake Hook
표면에서 끌어내어 뿔이나 촉수의 기본 형태를 뽑아냅니다. K로 빠르게 선택하세요.
2. Draw
뿔 표면에 디테일과 굴곡을 추가해요. 울퉁불퉁한 질감을 만들 때 씁니다.
3. Crease
뿔 가장자리에 날카로운 엣지와 세로 홈을 새겨 뿔다운 질감을 만들어요.
4. Smooth
마무리로 전체를 정리해요. 너무 많이 쓰면 형태가 무뎌지므로 살짝만 씁니다.
