Curve to Tube란? Blender 5.0

커브 오브젝트를 튜브(파이프) 형태의 메쉬로 변환하는 Blender 5.0 신규 모디파이어입니다. 기존의 Bevel+Fill 방식보다 훨씬 직관적이고, 메쉬 모디파이어 스택과 자연스럽게 조합됩니다.

실(커브)에 빨대(튜브)를 씌우는 것 — 커브가 경로, 튜브가 결과물

Radius & Resolution

Radius: 튜브의 반지름을 조절합니다. 값이 클수록 굵은 파이프가 만들어집니다.

Resolution: 원형 단면의 분할 수입니다. 높을수록 단면이 둥글어지고, 낮으면 다각형 형태가 됩니다.

Fill Caps: 켜면 튜브 양 끝을 막아줍니다.

기존 방식과 비교

이전에는 Curve의 Bevel Depth나 Bevel Object로 튜브를 만들었습니다. Curve to Tube는 메쉬 결과를 바로 보여주고, 다른 메쉬 모디파이어(Array, Subdivision 등)와 직접 조합할 수 있다는 점이 큰 차이입니다.

사용 방법

커브 오브젝트 선택 Bezier 또는 Nurbs 커브를 선택
Modifier 추가 Properties → Modifier tab → Add Modifier → Curve to Tube
파라미터 조절 Radius, Resolution, Fill Caps 설정
📖 Blender 공식 문서 — Curve to Tube Modifier
연속 조절 상태 전환 Radius와 Resolution은 슬라이더로 연속 조절하고, Fill Caps는 버튼으로 열린 관인지 막힌 튜브인지를 전환합니다.
곡선을 튜브로 바꾼다는 건, 선 하나에 두께와 단면을 입혀 실제 파이프처럼 만드는 일이에요. Radius는 두께, Resolution은 단면 분할, Fill Caps는 끝이 막힌 파이프인지 열린 관인지 정합니다.

Open or Faceted

Resolution이 너무 낮거나 끝을 비워 두면, 튜브가 부드러운 호스가 아니라 각진 파이프나 비어 있는 껍데기처럼 보일 수 있어요.

Readable Tube

두께, 분할, 캡을 목적에 맞게 잡으면 케이블, 배관, 프레임처럼 선 기반 형태를 빠르게 입체 구조로 바꿀 수 있습니다.

Curve to Tube가 필요할 때

  • 파이프 / 배관
  • 전선 / 케이블
  • 식물 줄기 / 덩굴
  • 난간
  • 로프 / 밧줄

Curve to Tube를 안 쓰는 경우

  • 복잡한 비원형 단면이 필요할 때
  • 별, 사각형 등 커스텀 프로파일 → 기존 Bevel Object 방식 사용
팁: Curve to Tube는 Blender 5.0 신규 기능입니다! 기존의 Curve Bevel보다 메쉬 모디파이어 스택과 자연스럽게 조합됩니다.

자주 쓰는 조합

Curve to Tube + Array
반복 패턴 파이프를 만들 때. Array로 커브를 따라 반복 배치하면 복잡한 배관 시스템을 구성할 수 있습니다.
Curve to Tube + Subdivision
더 부드러운 튜브를 만들 때. Resolution 대신 Subdivision으로 부드러움을 추가하면 폴리곤 수를 세밀하게 제어할 수 있습니다.
