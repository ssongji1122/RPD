Array Modifier란?

하나의 오브젝트를 여러 개 복제해서 일정한 간격으로 배열하는 Modifier입니다. 울타리, 계단, 체인 같은 반복 패턴을 한 번에 만들 수 있어요.

복사기에 원본을 넣고 "5장 복사" 버튼을 누르는 것처럼, 하나를 수정하면 전부 바뀌어요

Offset 종류

복제본 사이의 간격을 정하는 3가지 방법이 있습니다.

  • Constant: 절대 거리 (m 단위)
  • Relative: 오브젝트 크기 기준 비율
  • Object: 빈 오브젝트를 기준으로 배치 (원형 배열 가능)
더보기
Relative는 일렬 반복, Object는 원형 반복, Constant는 절대 거리 기준 반복으로 기억하면 됩니다.

Array가 필요할 때

  • 울타리 / 난간 기둥
  • 계단 / 나선 계단
  • 체인 / 톱니바퀴
  • 건물 층 반복
  • 타일 / 벽돌 패턴

Array보다 다른 방법이 나을 때

  • 불규칙한 배치 → Particle System
  • 자연 요소 (풀, 나무) → Scatter on Surface
  • 각각 다른 형태 → 수동 복제
팁: 원형 배열을 만들려면 Empty 오브젝트를 추가하고 Z축 회전값을 360/개수 로 설정한 뒤, Array의 Object Offset에 연결하세요. 예: 8개 원형 → Empty Z회전 = 45도

자주 쓰는 조합

Array + Curve Modifier
Curve를 따라 오브젝트를 반복 배치. 울타리, 도로 가드레일, 체인 등에 활용
Array + Mirror
한쪽 방향으로 반복 후 대칭. 대칭적인 반복 구조물에 유용
