기본 개념 Pivot Point란?

R(회전)이나 S(크기)를 할 때 '어느 점을 기준으로 돌릴지' 정하는 설정이에요. 같은 R키를 눌러도 Pivot이 오브젝트 자체냐 중앙이냐에 따라 결과가 완전히 달라요.

회전목마를 중심축으로 돌리는지, 의자 자체를 뱅글뱅글 돌리는지의 차이예요

종류 5가지 Pivot 종류

  • Median Point: 선택된 것들의 평균 중심 (기본값)
  • Individual Origins: 각각 자기 원점을 기준으로 회전/크기
  • Active Element: 마지막으로 선택한 것이 기준
  • 3D Cursor: 3D Cursor 위치를 기준으로
  • Bounding Box Center: 선택 영역 전체 바운딩 박스 중심

주의 Individual Origins 주의

여러 오브젝트를 동시에 S(크기)로 줄이면, Median Point에서는 서로 모이고 Individual Origins에서는 각자 제자리에서 줄어요. 의도에 맞게 선택해야 해요.

Median: 여러 풍선을 한 점으로 모아서 동시에 줄이기 / Individual: 각 풍선이 자기 위치에서 줄기

주요 단축키

. Pivot Point 메뉴 열기 (마침표)
Header Dropdown 3D Viewport 상단 점 아이콘 클릭
Shift + S 3D Cursor를 원점/선택 위치로 이동 (Pivot용 위치 설정)

회전 기준점이 원으로 표시됩니다. 두 모드를 번갈아 눌러 차이를 확인하세요.

Pivot이 필요할 때

  • 여러 오브젝트를 각자 자기 위치에서 크기 조절 (Individual Origins)
  • 특정 지점 기준으로 배열/회전 (3D Cursor)
  • 오브젝트 그룹의 중심으로 돌리기 (Median Point)

잘 안 쓰는 경우

  • 단일 오브젝트 변형 (큰 차이 없음)
  • 기본 이동 작업

자주 쓰는 조합

3D Cursor + Pivot
Shift+S → Cursor to World Origin 후 Pivot을 3D Cursor로 → 원점 기준 회전/크기
Individual Origins + Scale
Edit Mode에서 여러 면을 선택하고 각자 법선 방향으로 Inset하기
팁: 단축키 .(마침표)와 ,(쉼표)가 헷갈려요. .은 Pivot Point, ,는 Transform Orientation이에요. 마침표(.)는 기준점(Pivot), 쉼표(,)는 방향(Orientation)으로 외우세요.
