Pivot Point는 회전(R)과 스케일(S) 변환의 '기준점'이에요. 어디를 중심으로 돌리느냐에 따라 완전히 다른 결과가 나옵니다.

. Pivot Point 파이 메뉴 열기 (마침표)

기본 개념 Pivot Point란?

회전(R)이나 스케일(S) 변환을 할 때 '어느 지점을 중심으로 움직일지' 결정하는 기준점입니다.

마치 문이 열릴 때 경첩이 기준인 것처럼, Pivot Point를 어디에 두느냐에 따라 오브젝트의 움직임이 완전히 달라집니다.

6가지 옵션

  • Bounding Box Center — 선택 영역 전체 박스의 중심
  • 3D Cursor — 커서 위치를 기준점으로 사용
  • Individual Origins — 각 오브젝트의 Origin 기준
  • Median Point — 선택 오브젝트들의 평균 중심
  • Active Element — 마지막 선택 오브젝트 기준
  • Each Object — 편집 모드에서 각 오브젝트 개별

BEFORE
AFTER (회전)

Individual Origins — 문 여닫기

  • 여러 문을 선택 후 R 로 회전
  • 각 문이 자신의 경첩(Origin) 기준 회전
  • Origin을 경첩 위치에 미리 배치해두는 게 핵심

3D Cursor — 행성 공전

  • 3D Cursor를 태양 위치에 배치 (Shift+RMB)
  • 행성 선택 → Pivot: 3D Cursor → R
  • 행성이 태양 주위를 공전하는 효과

Median Point — 그룹 회전

  • 여러 오브젝트 선택 후 R
  • 선택 군집 전체가 공통 중간점 기준으로 함께 회전
  • 오브젝트들의 상대 위치가 유지됨

Active Element — 기준 오브젝트 설정

  • 기준이 될 오브젝트를 마지막에 선택 (밝게 빛남)
  • Pivot: Active Element → R
  • 나머지 오브젝트들이 Active 주위를 공전

상황여러 오브젝트를 같이 회전했는데 이상하게 뭉쳐짐

단축키 조합 팁
. + R

파이 메뉴에서 Pivot 선택 후 즉시 회전 — 작업 흐름 단축

. + S

Pivot 기준 스케일 변환 — 3D Cursor 방향으로 수축/확장

Shift + C

3D Cursor를 월드 원점으로 리셋 — 날아간 오브젝트 대응

Shift + RMB

3D Cursor를 클릭 위치에 배치 — 공전 기준점 설정

팁: 작업 중 .을 누르는 습관을 들이세요. Pivot 파이 메뉴는 블렌더에서 가장 자주 쓰는 단축키 중 하나입니다. 특히 여러 오브젝트를 다룰 때 Pivot 설정 하나로 작업 결과가 완전히 달라집니다.
