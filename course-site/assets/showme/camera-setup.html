렌즈 Focal Length (초점 거리)

렌즈의 초점 거리(mm)로 화각을 결정합니다. 낮을수록 광각(더 넓은 시야), 높을수록 망원(좁고 압축된 원근). 기본값은 50mm입니다.

  • 24mm 이하: 광각 — 공간감, 왜곡 효과
  • 50mm: 표준 — 눈에 자연스러운 원근
  • 85~200mm: 망원 — 배경 압축, 인물 촬영
DSLR에서 광각 렌즈와 망원 렌즈를 교환하는 것과 완전히 같은 원리예요

심도 Depth of Field (DOF)

피사계 심도. Focus Distance와 f-Stop으로 조절합니다. f-Stop 값이 낮을수록 아웃포커싱이 강하게 적용됩니다.

  • f/1.4~2.8: 얕은 심도 — 배경 강한 흐림
  • f/5.6~8: 중간 심도 — 자연스러운 흐림
  • f/11~22: 깊은 심도 — 전체 선명
카메라 조리개(f-Stop)를 열수록 배경이 흐려지는 효과와 동일해요

센서 Sensor Size

카메라 센서 크기. Full Frame(36mm)이 기준. 센서가 클수록 얕은 심도와 넓은 화각을 얻을 수 있습니다.

Properties → Camera (카메라 아이콘) → Lens 탭에서 조정합니다.

같은 렌즈라도 풀프레임 카메라에서 더 넓게 찍히는 것과 같아요

뷰 Camera View

Numpad 0으로 카메라 뷰로 진입하면 실제 렌더링될 프레임을 미리 볼 수 있습니다. 테두리 안이 렌더 영역입니다.

  • Numpad 0: 카메라 뷰 진입/나가기
  • Ctrl+Alt+Numpad 0: 현재 시점을 카메라로 설정
  • G+Z+Z: 카메라 뷰에서 앞뒤 이동 (dolly)

주요 단축키

Numpad 0 카메라 뷰 진입 / 나가기
Ctrl + Alt + Numpad 0 현재 시점을 카메라로 설정
G + Z + Z 카메라 뷰에서 앞뒤 이동 (dolly)
N 속성 패널 (카메라 뷰에서 Lock Camera to View)
Ctrl + B 카메라 렌더 경계 설정

카메라 세팅이 필요한 때

  • 렌더링 시점을 결정할 때 (항상 필요)
  • 광각으로 공간감 표현 (건축, 인테리어)
  • 망원으로 배경 압축 효과 (인물, 제품)
  • 배경 아웃포커싱(심도) 효과 추가

자주 하는 실수

  • 카메라 뷰(Numpad 0) 없이 렌더해서 예상치 못한 화각
  • f-Stop을 너무 낮게 설정해서 전체가 흐려짐
  • Ctrl+Alt+Numpad 0을 몰라서 수동으로 카메라 위치 조정
  • Sensor Size를 바꾸지 않아 화각이 틀어짐
팁: 카메라를 원하는 시점으로 이동시키기 어려울 때는 뷰포트에서 원하는 각도를 잡은 다음 Ctrl+Alt+Numpad 0을 누르면 현재 뷰를 바로 카메라 시점으로 설정할 수 있어요. N 패널 → View → Lock Camera to View를 체크하면 카메라 뷰에서 자유롭게 이동하며 구도를 잡을 수도 있습니다.

Focal Length 범위별 용도

10~24mm (초광각)
건축 외관, 광활한 환경, 왜곡 효과. 공간을 과장되게 넓어 보이게 합니다.
35~50mm (표준)
사람 눈에 가장 자연스러운 원근감. 일반적인 장면과 제품 렌더링에 적합합니다.
85~135mm (단렌즈)
인물 촬영에 이상적. 배경 압축 효과와 얕은 심도 연출이 자연스럽습니다.
200mm+ (망원)
강한 배경 압축. 원거리 피사체 강조, 원근 왜곡 최소화에 사용합니다.
