Protocol MCP란?

MCP(Model Context Protocol)는 AI 모델이 외부 도구·서비스와 대화하는 표준 언어입니다. Anthropic이 2024년 공개했으며, Claude가 Blender·GitHub·DB 등 어떤 앱이든 일관된 방식으로 조작할 수 있게 해줍니다.

USB처럼, MCP 이전에는 AI마다 각자 다른 케이블(커스텀 API)이 필요했어요. MCP는 하나의 표준 포트로 어떤 기기든 연결하게 해줘요

Architecture 어떻게 동작하나요?

MCP는 Host(Claude Desktop) ↔ Client ↔ Server(각 앱) 구조로 동작합니다. Claude가 명령을 보내면 MCP 서버가 실제 앱을 조작하고 결과를 돌려줍니다.

Claude
AI 모델
⟷
MCP
표준 프로토콜
⟷
Blender
실제 앱

Why 왜 중요한가요?

MCP 이전에는 AI가 앱을 조작하려면 각 앱마다 별도 코드가 필요했어요. MCP는 한 번 만들면 어디서든 재사용할 수 있는 표준을 제공합니다. Blender용 MCP 서버를 만들면 Claude든 다른 AI든 바로 사용 가능합니다.

레고 블록처럼, MCP 서버만 있으면 어느 AI와도 연결돼요. AI를 바꿔도 서버를 다시 만들 필요가 없어요

STEP 4

실행 결과가 Claude에게 돌아옴

MCP가 필요할 때

  • Claude에게 Blender를 자연어로 지시하고 싶을 때
  • 반복적인 씬 세팅을 자동화하고 싶을 때
  • AI와 실시간으로 3D 작업을 협업하고 싶을 때
  • 여러 앱(GitHub, Notion 등)을 Claude로 통합 관리할 때

없어도 되는 경우

  • Blender를 직접 수작업으로 조작할 때
  • Python 스크립트를 직접 실행할 줄 알 때
  • 단순한 단발성 작업일 때
  • Claude Desktop이 설치되어 있지 않을 때

MCP 활용 예시

자동 씬 생성
"로봇 3개를 삼각형으로 배치해줘" → Claude가 좌표 계산 + 오브젝트 배치
재질 자동화
"모든 금속 파츠에 Metallic 1.0 적용해줘" → 씬 순회 + Material 일괄 설정
렌더 파이프라인
"카메라 세팅하고 4K로 렌더해줘" → 카메라 설정 + 렌더 설정 + 실행
씬 분석
"이 씬에 폴리곤이 몇 개야?" → 씬 데이터 읽어 통계 계산
핵심 포인트: MCP는 "AI가 앱을 직접 조작"하는 표준입니다. blender-mcp 서버만 설치하면 Claude가 Blender의 Python API를 대신 실행해줘요. 코딩을 몰라도 자연어로 복잡한 3D 작업을 자동화할 수 있어요.
