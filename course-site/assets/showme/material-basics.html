Material이란? 기초

오브젝트에 색상, 질감, 반사 등의 시각적 속성을 부여하는 시스템입니다. 하나의 오브젝트에 여러 Material을 할당할 수 있으며, Material Slot으로 관리합니다.

물체에 페인트를 칠하는 것 — 색상뿐 아니라 광택, 거칠기까지 한 번에 설정해요

핵심 속성 3가지 필수

  • Base Color — 물체의 기본 색상
  • Metallic — 0 = 비금속(플라스틱) / 1 = 금속
  • Roughness — 0 = 매끄러움(거울) / 1 = 거침(분필)

이 3가지만으로 대부분의 기본 재질을 표현할 수 있습니다.

Material Slot 핵심

하나의 오브젝트에 여러 재질을 적용하는 구조입니다. Edit Mode에서 면을 선택한 뒤 Assign으로 각 면에 다른 Material을 할당합니다.

색칠 놀이 — 팔은 피부색, 옷은 파랑, 신발은 검정으로 각각 칠하는 것과 같아요

Material이 필요한 경우

  • 렌더링 시 색상·질감·반사 표현
  • 하나의 오브젝트에 여러 색상 적용
  • 게임 엔진(Unity/Unreal)으로 Export
  • Viewport에서 실시간 미리보기

Material Slot 활용 예시

  • 캐릭터: 피부·옷·눈·머리카락 분리
  • 건물: 벽·창문·지붕·문 각각 다른 재질
  • 차량: 차체(금속)·타이어(고무)·유리
  • 소품: 손잡이(나무)·금속 부분 분리

Material 관련 개념 구분

Material vs Texture
Material = 전체 재질 속성(색상+반사+거칠기), Texture = 이미지 패턴. Texture는 Material의 일부로 사용됨
Material vs Shader
Shader = 빛·표면 상호작용 계산 프로그램, Material = Shader에 전달되는 파라미터 세트. Shader Editor에서 노드로 조합
Principled BSDF
Blender의 기본 만능 Shader. Base Color, Metallic, Roughness 3개 파라미터만으로 대부분의 재질 표현 가능
Material + UV
Texture를 붙이려면 UV 맵이 필요. Material은 "어떤 재질인지", UV는 "어디에 이미지를 붙일지" 결정
팁: Z → Material Preview 모드에서 Material 결과를 실시간으로 확인하세요. HDRI 환경을 바꾸면 다양한 조명에서 미리 볼 수 있습니다.
주의: Metallic 값은 0 또는 1만 사용하세요. 현실에서 물체는 금속이거나 비금속이지 중간은 없습니다. 0.5 같은 중간 값은 비현실적 결과를 만듭니다.
