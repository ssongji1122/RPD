Core Weighted Normal이란?

면의 크기(면적)를 기준으로 Normal(법선) 방향을 다시 계산하는 Modifier입니다. 큰 면이 더 많은 영향을 주어서 평평한 면이 더 깨끗하게 보입니다.

Why 왜 필요한가?

Shade Smooth만 쓰면 작은 면과 큰 면의 Normal이 동일하게 섞여서 큰 면에 울퉁불퉁한 음영이 생깁니다. Weighted Normal로 큰 면에 우선권을 주면 평평한 면이 깔끔해집니다.

Auto Smooth와의 관계

Shade Smooth는 모든 면을 부드럽게 만들지만, Auto Smooth는 각도 기준으로 날카로운 엣지를 보존해요.

Weighted Normal은 여기서 한 단계 더 나아가서, 면 크기(Face Area) 기반으로 Normal을 재계산해요. 큰 면이 주변 작은 면의 음영 방향을 이끌어서 하드서페이스에 적합한 깨끗한 음영을 만들어요.

적용 순서: Shade Smooth → Auto Smooth 활성화 → Weighted Normal Modifier 추가

큰 거울은 빛을 균일하게 반사하고, 작은 거울 조각은 빛을 산란시켜요. Weighted Normal은 큰 거울이 이기게 해요

적용 방법

Right-click → Shade Smooth (Object Mode에서 오브젝트 우클릭)
Properties → Object Data → Normals → Auto Smooth 활성화
Properties → Modifier → Add Modifier → Weighted Normal

직접 비교: Shade Smooth만 vs Weighted Normal 추가

Shade Smooth만 적용하면 윗면에 울퉁불퉁한 음영이 생기지만, Weighted Normal을 추가하면 깨끗해집니다.

Weighted Normal 필요할 때

  • 하드서피스 모델의 평평한 면이 울퉁불퉁할 때
  • Bevel Modifier와 함께 쓸 때
  • 제품/기계/로봇 모델링

안 써도 되는 경우

  • 유기적 모델 (캐릭터, 동물)
  • 이미 음영이 깨끗한 경우
  • 정밀한 Normal 제어가 필요한 경우
원인Shade Smooth만 적용, WN 없음
→
결과평면에 울퉁불퉁한 음영 왜곡 발생
원인WN이 Bevel 위에 위치 (스택 순서 오류)
→
결과Bevel이 만든 면에 WN 효과가 적용되지 않음

→
결과WN은 Smooth 기반이라 Flat에서는 효과 없음

→
결과Bevel 면까지 깨끗한 음영 — 하드서페이스 마무리 완성

흔한 실수 & 해결법

Modifier 스택 순서가 잘못됨
Weighted Normal은 반드시 Bevel 아래에 놓으세요. 가장 마지막 Modifier여야 합니다.
Shade Smooth를 빼먹음
WN은 Smooth Shading 기반이에요. Object Mode에서 Right-click → Shade Smooth 필수.
Auto Smooth 미활성화
Properties → Object Data → Normals에서 Auto Smooth를 켜야 WN이 제대로 작동합니다.
Bevel 조합 시 Face Influence 미사용
WN Modifier에서 Face Influence 체크를 켜면 Bevel 면까지 깨끗한 음영이 적용됩니다.
팁: Weighted Normal은 항상 Modifier Stack의 맨 아래에 두세요! Bevel → Weighted Normal 순서가 정석이에요.
