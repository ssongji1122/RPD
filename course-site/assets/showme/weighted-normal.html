Core Weighted Normal이란?

면의 크기(면적)를 기준으로 Normal(법선) 방향을 다시 계산하는 Modifier입니다. 큰 면이 더 많은 영향을 주어서 평평한 면이 더 깨끗하게 보입니다.

Why 왜 필요한가?

Shade Smooth만 쓰면 작은 면과 큰 면의 Normal이 동일하게 섞여서 큰 면에 울퉁불퉁한 음영이 생깁니다. Weighted Normal로 큰 면에 우선권을 주면 평평한 면이 깔끔해집니다.

큰 거울은 빛을 균일하게 반사하고, 작은 거울 조각은 빛을 산란시켜요. Weighted Normal은 큰 거울이 이기게 해요

적용 방법

Right-click Shade Smooth 적용
Properties → Normals Auto Smooth 설정 (Object Data Properties에서)
Ctrl + 1~5 Weighted Normal은 Modifier 탭에서 Add Modifier로 추가
📖 Blender 공식 문서 — Weighted Normal Modifier

왼쪽은 Shade Smooth만 적용한 상태, 오른쪽은 Weighted Normal을 추가한 상태입니다. 토글 버튼으로 비교해 보세요.

Shade Smooth만 적용하면 윗면에 울퉁불퉁한 음영이 생기지만, Weighted Normal을 추가하면 깨끗해집니다.

Weighted Normal 필요할 때

  • 하드서피스 모델의 평평한 면이 울퉁불퉁할 때
  • Bevel Modifier와 함께 쓸 때
  • 제품/기계/로봇 모델링

안 써도 되는 경우

  • 유기적 모델 (캐릭터, 동물)
  • 이미 음영이 깨끗한 경우
  • 정밀한 Normal 제어가 필요한 경우

자주 쓰는 조합

Bevel Modifier + Weighted Normal
하드서피스 마무리 정석. Bevel로 모서리를 깎고 Weighted Normal로 셰이딩을 정리합니다.
Auto Smooth + Weighted Normal
날카로운 모서리는 유지하면서 평평한 면의 셰이딩을 깨끗하게 만듭니다.
팁: Weighted Normal은 항상 Modifier Stack의 맨 아래에 두세요! Bevel → Weighted Normal 순서가 정석이에요.
