Mirror Modifier는 Origin(원점)을 기준으로 반전합니다. Origin의 위치와 모드(Edit/Object)에 따라 결과가 달라지는 이유를 알아봅시다.

Origin Mirror의 거울선

Mirror Modifier는 오브젝트의 Origin(주황 점)을 중심으로 메시를 반전합니다. Origin이 어디에 있느냐에 따라 미러된 반쪽의 위치가 완전히 달라집니다.

거울의 위치 = Origin의 위치. 거울을 옮기면 반사되는 모습도 달라지듯, Origin을 옮기면 미러 결과도 달라져요.

핵심 차이 Edit vs Object Mode

Object Mode에서 이동: Origin도 함께 이동 → Mirror 기준선이 바뀜
Edit Mode에서 이동: Origin은 그대로, 메시만 이동 → Mirror 기준선 유지

Object Mode = 거울째로 옮기기, Edit Mode = 거울은 두고 물건만 옮기기

Mirror Object 다른 오브젝트 기준

Mirror Modifier의 Mirror Object 옵션에 Empty를 지정하면, 자기 자신의 Origin 대신 그 Empty의 위치·회전을 기준으로 미러합니다. 복잡한 배치에 유용합니다.

내 거울을 쓰는 대신, 다른 사람의 거울을 빌려 쓰는 것과 같아요.
Object Mode 이동 Edit Mode 이동
Origin 오브젝트와 함께 이동 제자리에 고정
메시(Mesh) 함께 이동 메시만 이동
Mirror 결과 정상 유지 (Origin과 메시 관계 불변) 메시가 Origin에서 멀어지며 미러 위치 변경
주 용도 완성된 오브젝트 배치 대칭 모델링 작업

핵심 단축키

S X 0 선택 정점의 X좌표를 0으로 → 중심선 정렬
Ctrl+A Apply Transform → 현재 변환을 기본값으로 적용

상태 전환 버튼으로 Mode 자체를 바꿔서, 같은 드래그라도 Origin과 Mirror 기준선이 어떻게 달라지는지 비교합니다.
Object Mode에서는 오브젝트와 Origin이 함께 움직여요. 그래서 Mirror 기준선도 같이 따라가고, 거울 자체를 옮긴 것처럼 보입니다.

Wrong Mode

메시만 다듬고 싶은데 Object Mode에서 움직이면 Origin까지 따라가서 Mirror 기준선이 통째로 바뀌어 버려요.

Right Mode

Edit Mode에서 버텍스를 움직이면 Origin은 제자리에 남아 있어서, 대칭 중심을 유지한 채 메시만 안정적으로 조정할 수 있습니다.

오브젝트를 드래그해 보세요. 모드에 따라 Origin(주황)과 Mirror(초록)가 어떻게 달라지는지 비교하세요.

Edit Mode에서 모델링해야 할 때

  • 대칭 모델 작업 중 (캐릭터, 자동차 등)
  • Mirror Modifier가 적용된 상태에서 형태 수정
  • Origin 위치를 유지하면서 메시만 조정할 때
  • 중심선(Origin) 기준 정렬이 중요한 작업

Object Mode에서 이동해야 할 때

  • 완성된 대칭 오브젝트를 씬에 배치
  • 여러 오브젝트를 정렬·배열할 때
  • Origin과 메시를 함께 옮겨야 할 때
  • Apply 후 최종 위치 조정
자주 하는 실수: Object Mode에서 Scale/Rotate를 적용하지 않고 Mirror Modifier를 추가하면, 미러된 반쪽이 예상과 다른 위치·크기로 나타납니다. 반드시 Ctrl+A → Apply All Transforms 후 Mirror를 추가하세요.

흔한 실수 → 해결법

미러가 엉뚱한 곳에?
Origin이 메시 중심에 없는 경우. 우클릭 → Set Origin → Origin to Geometry로 Origin을 메시 중심에 놓으세요.
중심선 틈이 생겨요
중심 정점이 정확히 X=0이 아닌 경우. Edit Mode에서 중심 정점 선택 후 S → X → 0으로 정렬하세요.
Edit Mode에서 이동했더니 미러가 어긋나요
Edit Mode 이동은 Origin을 움직이지 않으므로 메시와 Origin 사이 거리가 바뀝니다. 의도한 것이 아니면 Object Mode에서 이동하세요.
다른 오브젝트 기준으로 미러하고 싶어요
Mirror Modifier의 Mirror Object에 Empty를 지정하면, 그 Empty의 위치·회전 기준으로 미러됩니다.
