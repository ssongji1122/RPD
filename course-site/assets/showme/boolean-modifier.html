Boolean이란?

두 오브젝트의 겹치는 부분을 이용해 새 형태를 만드는 모디파이어입니다. 3가지 연산을 지원합니다: Union(합), Intersect(교), Difference(차).

쿠키 커터로 반죽을 찍어내는 것 (Difference), 반죽 두 덩어리를 합치는 것 (Union)

3가지 연산

  • Difference (차) — 커터 오브젝트 모양을 잘라냄 가장 많이 사용
  • Union (합) — 두 오브젝트를 하나로 합침
  • Intersect (교) — 겹치는 부분만 남김

Solver 중요

Fast vs Exact 두 가지 솔버를 선택할 수 있습니다. Exact가 더 정확하지만 느립니다. Blender 5.0에서는 Exact가 기본값입니다.

더보기
학생 단계에서는 `Difference = 구멍`, `Union = 합치기`, `Intersect = 겹친 부분만` 이 세 가지로 먼저 기억하면 충분합니다. 결과가 깨지면 Solver를 `Exact`로 다시 확인하세요.

Boolean이 필요할 때

  • 구멍 뚫기 (볼트홀, 창문)
  • 복잡한 형태 합치기
  • 교차 부분 추출
  • 하드서페이스 디테일

Boolean을 피해야 할 때

  • 유기적 형태 (토폴로지 꼬임)
  • 지나치게 복잡한 메쉬
  • 정확한 토폴로지가 필요할 때
팁: Boolean 후 토폴로지가 지저분할 수 있습니다. 중요한 모델이라면 Remesh나 수동 리토폴로지로 정리하세요. Exact Solver를 사용하면 더 깨끗한 결과를 얻을 수 있습니다.

자주 쓰는 조합

Boolean + Array
반복 구멍 패턴을 만들 때 유용합니다. 환기구처럼 일정 간격으로 동일한 구멍을 뚫을 수 있습니다.
Boolean + Solidify
두께가 있는 구멍을 만들 때 사용합니다. Solidify로 두께를 주고 Boolean으로 구멍을 뚫습니다.
