E Extrude Region

선택한 면을 뽑아내어 새로운 지오메트리를 만드는 도구입니다. 점토를 손가락으로 잡아 당기는 것과 같습니다. 기본 형태에서 복잡한 구조를 만들어 나가는 가장 핵심적인 모델링 기법이에요.

Alt+E Normal 방향 Extrude

Alt+E로 면의 법선(Normal) 방향을 따라 균일하게 돌출합니다. 여러 면을 동시에 선택했을 때, 각 면이 자신의 법선 방향으로 개별적으로 뽑혀 나옵니다. 곡면에 균일한 두께를 줄 때 특히 유용합니다.

Manifold Extrude Manifold

돌출 후 내부에 남는 면(interior face)을 자동으로 제거해 메시를 항상 매니폴드(2-매니폴드) 상태로 유지합니다. 기존 지오메트리 안으로 뽑거나 인접한 메시를 합칠 때 Region 대신 사용하면 위상 오류 없이 깨끗한 결과를 얻을 수 있어요.

Ctrl+RMB Extrude to Cursor

3D 뷰포트에서 마우스를 클릭한 위치까지 선택된 지오메트리를 곧장 뽑아냅니다. Blender의 3D Cursor 오브젝트(Shift+RMB로 배치하는 다트)가 아니라, 클릭 순간 마우스 포인터가 가리키는 뷰포트 위치가 끝점이 됩니다.

찰흙에서 손가락으로 잡아당기면 새로운 형태가 늘어나오는 것, 그게 Extrude예요

주요 단축키

E Extrude Region — 선택한 면/엣지/점을 뽑기
Alt + E Extrude Along Normals — 법선 방향으로 균일 돌출
Ctrl + RMB Extrude to Cursor — 커서 위치까지 뽑기
Alt + E → Manifold Extrude Manifold — 내부 면 자동 제거로 클린 메시 유지
E → ESC 뽑고 제자리 — 분리된 면 생성 (주의!)

세 면의 기울기를 주목하세요 — 모드별로 돌출 방향이 어떻게 달라지는지 드래그로 확인하세요.

자동으로 애니메이션됩니다 — 드래그로 직접 조작할 수도 있어요.

Region (E) — 영역 돌출

선택된 모든 면이 하나의 덩어리로 묶여 전체가 같은 방향(전역 위쪽)으로 돌출됩니다.

기울어진 Face A·C도 위쪽으로 뽑히므로 연결된 하나의 박스 형태로 결과가 나옵니다. 면 간 경계가 합쳐집니다.

Along Normals (Alt+E) — 법선 방향 돌출

각 면이 자신의 법선(수직 방향)으로 개별 돌출됩니다. Region과의 핵심 차이입니다!

→ Face A는 왼쪽 위, Face B는 바로 위, Face C는 오른쪽 위로 뽑힙니다. 곡면·갑옷에 균일한 두께를 줄 때 필수입니다.

Individual Faces — 개별 면 돌출

각 면이 완전히 독립적으로 자신의 법선 방향으로 돌출됩니다. 공유 엣지가 분리되어 면 사이에 틈이 생깁니다.

Along Normals와 방향은 같지만 면들이 서로 연결되지 않고 별개 형태로 분리되어 뽑혀 나옵니다.

Extrude Manifold — 내부 면 자동 제거

위로 뽑힌 뒤 원본 면이 자동으로 제거됩니다(빨간 점선). 내부 면이 남지 않아 항상 매니폴드 메시를 유지합니다.

Region은 원본 면이 그대로 남아 내부 면(Interior Face)이 생기지만, Manifold는 이를 자동으로 제거해 렌더링 오류와 위상 문제를 방지합니다.

To Cursor (Ctrl+RMB) — 마우스 클릭 위치로 돌출

마우스를 클릭한 위치까지 선택 면이 일직선으로 뻗어나갑니다. Blender 3D Cursor 오브젝트가 아니라 클릭 순간 마우스 포인터 위치가 끝점입니다.

화면에서 Face B(파란 면)가 동심원 클릭 표시 방향으로 돌출되는 걸 확인하세요. 수치 없이 클릭 한 번으로 길이를 지정할 수 있어 파이프·케이블 배치에 유용합니다.

Region Extrude (E)

  • 건물 층 쌓기
  • 캐릭터 팔다리 만들기
  • 기본 형태에서 복잡한 형태로 발전
  • 파이프, 터널 형태 생성

Along Normals (Alt+E)

  • 갑옷 두께 만들기
  • 곡면의 디테일 추가
  • 선택한 면 모두 균일하게 돌출
  • 벽체 두께 부여

Extrude Manifold

  • 인접 메시 안으로 뚫고 들어가기
  • Boolean 없이 두 메시 합치기
  • 내부 면 오류 없는 클린 모델링
  • 게임 에셋 최적화 토폴로지 유지

To Cursor (Ctrl+RMB)

  • 파이프·케이블 경로 빠르게 배치
  • 수치 없이 클릭만으로 길이 지정
  • 건축 모델 정확한 끝점 연결
  • 스냅과 함께 그리드 정렬 돌출

자주 쓰는 조합

E + S (Extrude + Scale)
뽑은 후 바로 크기 조절. 지붕을 뾰족하게 만들거나 점점 좁아지는 형태를 만들 때 사용합니다.
E + 0 + S (제자리 + Scale)
면을 복제한 뒤 안쪽으로 줄여 Inset 효과를 낼 수 있습니다. Inset Faces(I) 대용으로 유용합니다.
주의: E를 누르고 ESC를 누르면 겉보기엔 아무 일도 안 일어난 것 같지만, 실제로는 겹친 면이 생겨요! 반드시 Ctrl+Z로 취소하세요. 겹친 면은 렌더링 아티팩트와 법선 오류의 원인이 됩니다.
