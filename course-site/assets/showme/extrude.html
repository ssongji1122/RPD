E Extrude Region

선택한 면을 뽑아내어 새로운 지오메트리를 만드는 도구입니다. 점토를 손가락으로 잡아 당기는 것과 같습니다. 기본 형태에서 복잡한 구조를 만들어 나가는 가장 핵심적인 모델링 기법이에요.

Alt+E Normal 방향 Extrude

Alt+E로 면의 법선(Normal) 방향을 따라 균일하게 돌출합니다. 여러 면을 동시에 선택했을 때, 각 면이 자신의 법선 방향으로 개별적으로 뽑혀 나옵니다. 곡면에 균일한 두께를 줄 때 특히 유용합니다.

Manifold Extrude Manifold

돌출 후 내부에 남는 면(interior face)을 자동으로 제거해 메시를 항상 매니폴드(2-매니폴드) 상태로 유지합니다. 기존 지오메트리 안으로 뽑거나 인접한 메시를 합칠 때 Region 대신 사용하면 위상 오류 없이 깨끗한 결과를 얻을 수 있어요.

Ctrl+RMB Extrude to Cursor

3D 뷰포트에서 마우스를 클릭한 위치까지 선택된 지오메트리를 곧장 뽑아냅니다. Blender의 3D Cursor 오브젝트(Shift+RMB로 배치하는 다트)가 아니라, 클릭 순간 마우스 포인터가 가리키는 뷰포트 위치가 끝점이 됩니다.

Region Extrude (E)

  • 건물 층 쌓기
  • 캐릭터 팔다리 만들기
  • 기본 형태에서 복잡한 형태로 발전
  • 파이프, 터널 형태 생성

Along Normals (Alt+E)

  • 갑옷 두께 만들기
  • 곡면의 디테일 추가
  • 선택한 면 모두 균일하게 돌출
  • 벽체 두께 부여

Extrude Manifold

  • 인접 메시 안으로 뚫고 들어가기
  • Boolean 없이 두 메시 합치기
  • 내부 면 오류 없는 클린 모델링
  • 게임 에셋 최적화 토폴로지 유지

To Cursor (Ctrl+RMB)

  • 파이프·케이블 경로 빠르게 배치
  • 수치 없이 클릭만으로 길이 지정
  • 건축 모델 정확한 끝점 연결
  • 스냅과 함께 그리드 정렬 돌출

자주 쓰는 조합

E + S (Extrude + Scale)
뽑은 후 바로 크기 조절. 지붕을 뾰족하게 만들거나 점점 좁아지는 형태를 만들 때 사용합니다.
E + 0 + S (제자리 + Scale)
면을 복제한 뒤 안쪽으로 줄여 Inset 효과를 낼 수 있습니다. Inset Faces(I) 대용으로 유용합니다.
주의: E를 누르고 ESC를 누르면 겉보기엔 아무 일도 안 일어난 것 같지만, 실제로는 겹친 면이 생겨요! 반드시 Ctrl+Z로 취소하세요. 겹친 면은 렌더링 아티팩트와 법선 오류의 원인이 됩니다.
