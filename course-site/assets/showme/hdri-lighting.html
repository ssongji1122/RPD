HDRI란? 기본

HDRI(High Dynamic Range Image)는 360도 파노라마 이미지로, 실제 환경의 빛 정보를 담고 있습니다. 이를 환경 조명으로 사용하면 사실적인 반사와 조명을 손쉽게 구현할 수 있습니다.

실제 장소를 360도 촬영한 사진 안에 3D 오브젝트를 놓는 것과 같아요

Environment Texture 연결 노드

World Properties에서 Environment Texture 노드를 Background에 연결하면 HDRI가 적용됩니다. Shader Editor(World 모드)에서 설정합니다.

  • World Properties → Surface → Color → Environment Texture
  • Strength로 밝기 조절
  • Mapping 노드로 회전 조절

Poly Haven 리소스

Poly Haven(polyhaven.com)은 무료 CC0 라이선스 HDRI를 제공하는 사이트입니다. 실내, 야외, 스튜디오 등 다양한 환경을 다운로드하여 사용할 수 있습니다.

  • 1K~16K 해상도 제공
  • 상업적 사용 가능 (CC0)
  • 수백 개의 무료 HDRI

HDRI 설정 경로

World Properties Properties 패널 → 지구 아이콘
Surface → Color 노란 점 클릭 → Environment Texture 선택
Open → HDRI 파일 .hdr 또는 .exr 파일 불러오기
Z → Rendered 렌더 프리뷰에서 HDRI 확인
📖 Blender 공식 문서 — World Lighting

구체에 다른 HDRI 환경(실내/야외/스튜디오)이 반사되는 모습을 비교하세요.

각 환경을 선택하면 구체의 반사와 조명이 달라집니다.

HDRI가 필요할 때

  • 사실적인 환경 반사가 필요한 제품 렌더
  • 야외 건축 시각화
  • 자동차, 금속 오브젝트 렌더링
  • 빠른 조명 세팅이 필요할 때

HDRI만으로 부족한 경우

  • 강한 직접 조명이 필요한 장면
  • 특정 방향의 그림자가 필요할 때
  • HDRI에 없는 환경을 만들 때
  • 스타일라이즈드(비사실적) 렌더
팁: HDRI는 보통 Strength 1.0으로 시작하세요. 너무 밝으면 0.5~0.8로, 어두우면 1.5~2.0으로 조절합니다. Mapping 노드의 Z Rotation으로 태양 방향을 회전시킬 수 있습니다.

HDRI 활용 조합

HDRI + Sun Light
HDRI로 환경 반사를 주고 Sun Light로 직접 조명과 그림자를 강화합니다. 야외 장면의 표준 세팅입니다.
HDRI + Light Path (Is Camera Ray)
카메라에만 HDRI 배경을 보여주고, 조명은 다른 색으로 설정하는 고급 기법입니다.
