AI 3D 생성이란? 자동화

텍스트 프롬프트를 넣으면 AI가 3D 메쉬를 자동으로 생성해요. 완벽하지 않지만, 아이디어를 빠르게 3D로 확인하는 데 유용해요.

AI = 초벌 장인 — 큰 덩어리를 잡아주면 우리가 다듬기만 하면 돼요

Meshy vs Tripo 도구 비교

  • Meshy — 텍스트→3D + 이미지→3D 모두 지원. 무료 크레딧 있음
  • Tripo — 빠른 생성 속도. 이미지→3D가 강점
  • 둘 다 .glb 파일로 다운로드 → Blender에서 Import

AI 3D 생성 워크플로우

1
프롬프트 입력 & 생성
Meshy/Tripo 사이트에서 텍스트 또는 이미지로 3D 생성
2
.glb 다운로드
생성 완료 후 .glb 형식으로 다운로드
3
Blender에서 Import
File → Import → glTF (.glb/.gltf)
4
메쉬 정리
Mesh Cleaner 2 → Decimate → Scale 적용

Import 후 체크리스트 필수

  • 크기: S키로 적절한 크기로 맞추고 Ctrl+A → Scale 적용
  • 원점: Right-click → Set Origin → Origin to Geometry
  • 폴리곤 수: Viewport Overlay → Statistics로 확인. 수만 개 이상이면 Decimate 필요
  • 노멀: Mesh Cleaner 2로 뒤집힌 노멀 자동 수정

애드온 설치법 설정

  • Edit → Preferences → Add-ons → Install
  • 다운로드한 .zip 파일 선택 → 체크박스 활성화
  • Mesh Cleaner: N패널 → Mesh Cleaner 탭에서 실행
  • QRemeshify: Properties → Object Data → QRemeshify 패널
