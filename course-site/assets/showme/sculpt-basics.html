Sculpt Mode란? 직관적

찰흙을 빚듯이 메쉬를 직접 밀고, 당기고, 다듬는 모드입니다. Edit Mode처럼 정점을 하나씩 선택하는 대신, 브러시로 넓은 영역을 한번에 조작합니다.

디지털 찰흙 — 브러시로 표면을 밀고 당겨서 형태를 만들어요

주요 브러시 6가지

  • Draw — 표면을 올리거나 내림 (기본)
  • Clay Strips — 찰흙 덩어리를 붙이는 느낌
  • Smooth — 울퉁불퉁한 표면을 평탄화
  • Grab — 표면을 잡아서 이동
  • Inflate — 표면을 부풀림
  • Crease — 날카로운 주름/홈 생성

주요 단축키

Ctrl + Tab Sculpt Mode 전환
F 브러시 크기 조절
Shift + F 브러시 강도 조절
Ctrl 브러시 반전 (올리기 ↔ 내리기)
Shift 임시 Smooth 브러시
📖 Blender 공식 문서 — Sculpt Mode

캔버스 위에서 마우스를 드래그하여 각 브러시 효과를 체험해 보세요.

브러시를 선택한 뒤 캔버스를 드래그하세요. 높이맵 시뮬레이션으로 브러시 효과를 체험합니다.

Sculpt Mode가 적합할 때

  • 캐릭터 얼굴/몸체 디테일
  • 유기적인 형태 (나무, 바위, 생물)
  • 옷 주름, 근육 표현
  • 지형(Terrain) 모델링
  • 기존 모델에 디테일 추가

Edit Mode가 더 나은 경우

  • 정확한 치수가 필요한 하드서피스
  • 건축물, 기계 부품
  • 깔끔한 토폴로지가 필요할 때
  • 정밀한 정점 위치 조절
팁: Sculpt를 시작하기 전에 메쉬의 폴리곤 수를 충분히 늘려야 합니다. Ctrl+3 등으로 Subdivision Surface를 추가하거나, Remesh를 사용하세요. 폴리곤이 적으면 디테일 표현이 어렵습니다.

작업 흐름

Base Mesh → Sculpt
Edit Mode에서 기본 형태를 잡고, Sculpt Mode에서 디테일을 추가하는 일반적인 흐름입니다.
Sculpt → Retopology
Sculpt로 만든 고해상도 모델을 낮은 폴리곤으로 다시 만들어 애니메이션에 적합하게 합니다.
