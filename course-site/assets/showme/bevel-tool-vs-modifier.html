Tool 지금 바로 깎기

Bevel Tool은 편집 모드에서 선택한 모서리를 즉시 깎아 실제 메시 구조를 바꿔요. 한번 적용하면 결과가 모델 안에 직접 새겨집니다.

Modifier 나중에도 조절하기

Bevel Modifier는 원본 메시를 유지한 채, 렌더/표시 단계에서 비파괴로 모서리를 만들어 줘요. Width와 Segments를 계속 바꿀 수 있습니다.

핵심 직접 편집 vs 비파괴

둘 다 모서리를 둥글리지만 목적이 달라요. 형태를 손으로 다듬는 순간에는 Tool, 결과를 나중까지 조정하고 싶으면 Modifier가 더 잘 맞아요.

주요 단축키

Ctrl + B Bevel Tool — 선택한 엣지를 즉시 베벨
Ctrl + A Apply Scale — Modifier가 예상대로 먹게 기준값 정리
Modifier Tab Bevel Modifier 추가 — 비파괴 방식으로 모서리 조절
기억할 한 줄: 지금 형태를 손으로 깎으면 Tool, 나중까지 수치를 바꾸고 싶으면 Modifier예요.

작업 전
Tool 적용 후
Tool은 결과가 메시 안에 고정되고, Modifier는 Width·Segments를 계속 바꿀 수 있다는 차이에 집중해 보세요.

Bevel Tool이 잘 맞는 때

하나의 엣지만 빠르게 깎고 싶을 때, 특정 부분만 손으로 조형하듯 수정할 때, 수정 범위가 작고 직접 결정하고 싶을 때 좋아요.

Bevel Modifier가 잘 맞는 때

제품 모형처럼 일관된 모서리 처리가 필요할 때, Width를 계속 바꿔 보며 찾고 싶을 때, 비파괴 워크플로를 유지하고 싶을 때 좋아요.

Tool 선택 기준

형태를 이미 확정했고, 특정 모서리만 손으로 다듬는 상황이라면 Tool이 더 빠릅니다.

Modifier 선택 기준

결과를 끝까지 비교하며 조정해야 하거나, 하드서피스 작업에서 여러 엣지를 같은 기준으로 관리해야 할 때 Modifier가 유리합니다.

주의: Modifier가 이상하게 먹으면 먼저 Ctrl + A로 Scale을 정리해 보세요. 겉보기 크기와 실제 데이터가 다르면 베벨 폭도 어긋납니다.
