O Proportional Editing이란?

하나의 버텍스를 움직일 때 주변 버텍스들도 같이 따라오는 기능이에요. 영향 반경을 조절해서 부드러운 언덕이나 파도 모양을 만들 수 있어요.

물에 돌을 던지면 물결이 퍼지듯이, 하나를 건드리면 주변도 서서히 따라와요

Shift+O Falloff 종류

  • Smooth: 가장 자연스럽게 감소 (기본, 언덕 모양)
  • Sphere: 구형으로 균일하게 영향
  • Root: 중앙이 더 강하게
  • Sharp: 경계가 뚜렷하게
  • Linear: 선형으로 균일하게 감소

Shift+O로 순환 전환 가능

Scroll 반경 조절이 핵심

O를 켜고 G를 누르면 이동 중에 스크롤로 영향 범위를 바꿀 수 있어요. 원이 크면 더 넓은 범위가, 작으면 좁은 범위만 따라와요. 반경을 조절하는 게 가장 중요해요.

파급 범위를 내 손안에서 조절하는 거예요

주요 단축키

O Proportional Editing 토글 (Object/Edit Mode 모두)
Shift + O Falloff 타입 순환
Scroll up/down 영향 반경 확대/축소 (변형 중에만)
Header icon 3D Viewport 헤더의 동심원 아이콘 클릭

중앙 포인트를 위아래로 드래그하세요. 점선 원이 영향 반경이에요.

필요할 때

  • 부드러운 언덕/파도/지형 만들기
  • 캐릭터 얼굴의 부드러운 변형
  • 여러 버텍스를 자연스럽게 늘리기

안 쓰는 경우

  • 딱딱한 하드서피스 모델링
  • 정확한 형태가 필요한 기계류
  • Boolean 작업

자주 쓰는 조합

Proportional + Sculpt 준비
기본 메쉬 변형 후 Sculpt Mode로 다듬기 전 큰 형태 잡기
Sphere Falloff + Terrain
지형이나 부드러운 오가닉 형태에서 자연스러운 굴곡 만들기
팁: Proportional Editing은 Object Mode에서도 작동해요. 여러 오브젝트 중 하나를 이동할 때 주변 오브젝트들도 따라오게 할 수 있어요. 단, 오브젝트 모드에서는 Smooth가 아닌 Connected 옵션도 확인해 보세요.
