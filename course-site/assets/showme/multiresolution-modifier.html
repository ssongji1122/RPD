Core Multiresolution이란?

Subdivision처럼 메쉬를 세분화하되, 각 레벨에서 독립적으로 Sculpt할 수 있는 모디파이어입니다. 낮은 레벨에서는 큰 형태를, 높은 레벨에서는 세밀한 디테일을 작업합니다.

지도의 축척 — 전국 지도(큰 형태)와 동네 지도(세부 디테일)를 오가며 편집하는 것과 같습니다

Setting Subdivide / Levels

Subdivide 버튼으로 세분화 레벨을 추가합니다. Sculpt, Viewport, Render 레벨을 독립적으로 설정할 수 있어 작업 상황에 맞게 조절합니다.

  • Sculpt Level — Sculpt 모드에서 사용하는 레벨
  • Viewport Level — 3D 뷰포트에 표시되는 레벨
  • Render Level — 렌더링 시 사용되는 레벨

vs Subdivision과의 차이

Subdivision Surface는 메쉬를 부드럽게만 만들지만, Multiresolution은 각 레벨에서 Sculpt한 디테일을 독립적으로 저장합니다. 레벨을 오가며 편집이 가능합니다.

  • Subdivision — 부드럽게만, 디테일 저장 불가
  • Multiresolution — 각 레벨별 Sculpt 디테일 저장

상태 전환 버튼으로 세분화 레벨을 바꿔서, 기본 형태와 고해상도 디테일이 어떻게 분리되는지 단계별로 비교합니다.
Multiresolution은 단순히 더 부드럽게 만드는 게 아니라, 레벨마다 다른 양의 Sculpt 정보를 저장하는 구조입니다. 낮은 레벨은 큰 형태, 높은 레벨은 세부 디테일을 담당합니다.

Single Level Thinking

모든 디테일을 한 레벨에서만 해결하려고 하면, 큰 형태를 고치려는 순간 미세 주름과 세부 조각이 같이 깨지기 쉽습니다.

Level Separation

낮은 레벨에서 덩어리를, 높은 레벨에서 주름과 표면 질감을 다루면 Sculpt가 훨씬 안정적이고 되돌리기 쉬워집니다.

Level 0: 기본 형태 (8면) → Level 1: 중간 디테일 (32면) → Level 2: 세밀한 디테일 (128면)

Multiresolution이 필요할 때

  • 캐릭터 스컬핑 (큰 형태 → 세부 주름)
  • 지형 모델링
  • 유기적 모델 디테일링
  • 다단계 Sculpt 작업

Multiresolution을 안 쓰는 경우

  • 하드서페이스 모델링
  • 저폴리(Low-Poly) 스타일
  • 일반 모델링 (Subdivision 사용)
  • 단순한 부드러움만 필요할 때
팁: Multires는 Sculpt 전용입니다! 일반 모델링에는 Subdivision Surface를 사용하세요. Multires 적용 전 반드시 Apply Scale(Ctrl+A)을 먼저 하세요.

자주 쓰는 조합

Multires + Sculpt Mode
필수 조합. Multiresolution은 Sculpt 모드에서 각 레벨별로 디테일을 조각할 때 진가를 발휘합니다.
Bake Normal Map
고레벨의 Sculpt 디테일을 저레벨 메쉬에 Normal Map으로 전달. 게임 에셋 제작의 핵심 워크플로우입니다.
