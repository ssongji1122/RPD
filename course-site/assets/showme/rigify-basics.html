Rigify는 Blender에 내장된 자동 리깅 애드온이에요. AI가 아니라 미리 정해 놓은 본 템플릿(메타리그)을 가지고 IK·FK 같은 컨트롤 본을 통째로 만들어 줘요.

Core Rigify란?

Blender 안에 들어 있는 자동 리깅 도구예요. 사람·동물 같은 흔한 캐릭터의 뼈 구조를 템플릿으로 갖고 있어서, 비율만 맞추면 컨트롤 본까지 한 번에 만들어 줘요.

옷 패턴지를 골라서 내 몸에 맞게 줄이고 늘리는 거예요. 처음부터 재단할 필요가 없어요.

구조 두 단계 워크플로

Meta-Rig 배치 → Generate Rig 두 단계로 끝나요. 메타리그는 비율 맞추기용 본, Generate Rig 버튼이 컨트롤 본까지 풀세트로 만들어 줘요.

  • Meta-Rig: 캐릭터 비율 맞추는 본 템플릿
  • Generate Rig: IK·FK·Pole·Tweak 본 자동 생성

템플릿 Meta-Rig 종류

Human이 가장 자주 쓰여요. Basic은 단순한 팔다리만 있는 가벼운 버전이고, Animals 카테고리에는 Cat·Wolf·Bird 같은 사족·조류 템플릿이 들어 있어요.

  • Human (Meta-Rig): 사람형 표준
  • Basic: 팔/다리/꼬리 단순 버전
  • Animals: Cat, Wolf, Bird 등

결과 Generate Rig가 만드는 것

버튼 하나로 컨트롤 본(빨강 IK, 초록 FK, 파랑 Tweak)이 한꺼번에 만들어져요. 학생이 직접 만들면 며칠 걸리는 리그가 몇 초면 끝나요.

레고 세트의 완성품 박스를 받는 느낌이에요. 부품만 있던 게 갑자기 조립품으로 바뀌어요.

주요 단축키 · 경로

Shift + A Add → Armature → Rigify Meta Rigs → Human
Tab Edit Mode 진입 (메타리그 비율 맞추기)
(클릭) Properties > Armature > Rigify Buttons > Generate Rig
Ctrl + P 메쉬 → Rig 순으로 선택 후 → With Automatic Weights
Blender 공식 문서 — Rigify Add-on

실제 작업 순서를 5단계로 나눠 봤어요. 위쪽 단계 버튼을 눌러서 넘기거나, 아래쪽 본 색상 칩을 눌러 어떤 본이 어떤 역할인지 확인해 보세요.

1 Rigify Add-on 켜기

Rigify는 기본 내장이지만 처음엔 꺼져 있어요. Edit > Preferences > Add-ons에서 "Rigify"를 검색해서 체크박스를 켜주세요. 한 번만 켜면 다음부터는 자동으로 적용돼요.

Edit > Preferences > Add-ons
Search: rigify
Rigify Enabled
Generate and edit advanced rigs (humanoid, quadrupeds...)
2 Meta-Rig 추가하기

Object Mode에서 Shift+A로 추가 메뉴를 열고 Armature 아래에서 Rigify Meta Rigs를 따라가요. 사람 캐릭터면 Human이에요.

Add → Armature → Rigify Meta Rigs → Human (Meta-Rig)
그 외 옵션: Basic Human / Basic Quadruped / Cat / Wolf / Bird ...

메타리그가 씬에 추가되면 보통 캐릭터 메쉬 옆에 사람 비율의 본 묶음이 보여요.

3 캐릭터 비율에 맞추기

Tab으로 Edit Mode 진입한 다음, 각 본의 Head·Tail을 잡고 캐릭터 관절 위치로 옮겨요. 골반·무릎·팔꿈치·손목 같은 관절이 캐릭터 메쉬 안에 정확히 들어가게 맞추는 게 핵심이에요.

Header — Mode
Object Mode Edit Mode Pose Mode
관절 위치 정밀도가 변형 품질을 결정해요. 어긋나면 Generate Rig 결과도 어긋나요.

팁: 좌우 대칭은 X-Axis Mirror 옵션을 켜면 한쪽만 만져도 반대쪽이 따라와요.

4 Generate Rig 클릭

Object Mode로 돌아와서 메타리그가 선택된 상태에서 Properties > Armature 탭으로 가요. Rigify Buttons 패널 안의 Generate Rig 버튼을 누르면 끝이에요.

Properties > Armature Data
Rigify Buttons
Generate Rig
누르면 별도의 새 아마추어 오브젝트(보통 "rig")가 생성돼요. 메타리그는 그대로 남아 있어요.
5 결과 리그 확인 + 메쉬 연결

완성된 리그는 색상별로 역할이 달라요. 빨강은 IK, 초록은 FK, 파랑은 Tweak 본이에요. 메쉬에 적용하려면 메쉬를 먼저 선택하고 Shift+클릭으로 리그를 추가 선택한 뒤 Ctrl+P → With Automatic Weights를 골라요.

Generate Rig 결과 — 컨트롤 본이 색상별로 표시돼요
본 색상 가이드 (생성된 리그 기준)

칩을 눌러서 어떤 역할인지 확인해 보세요.

왼쪽 칩을 눌러 본 색상별 역할을 확인하세요.

Rigify가 잘 맞는 경우

  • 인체형 캐릭터 (1순위)
  • 사족 동물 (Cat / Wolf 메타리그)
  • 새 (Bird 메타리그)
  • IK/FK 전환이 필요한 캐릭터 애니메이션

Rigify를 피해야 하는 경우

  • 로봇·기계처럼 관절 구조가 특이한 모델
  • 사물·식물 같은 비생물체
  • 표준 인체 비율을 크게 벗어난 캐릭터
  • 이런 경우는 수동 리깅이 더 깔끔해요
