Core Armature란?

Armature는 뼈대(skeleton)를 담는 오브젝트입니다. 하나의 Armature 안에 여러 Bone이 들어가고, 이 뼈대를 메쉬에 연결(Skinning)하면 캐릭터를 움직일 수 있습니다.

사람의 골격처럼, 뼈가 모여서 하나의 뼈대(Armature)를 이루는 거예요

구조 Bone: Head와 Tail

각 Bone에는 Head(시작점)와 Tail(끝점)이 있습니다. Head는 굵은 쪽, Tail은 가는 쪽입니다. Bone의 방향은 Head에서 Tail을 향합니다.

뼈의 관절 쪽이 Head, 반대 끝이 Tail이라고 생각하세요

계층 Parent-Child 관계

Bone끼리 부모-자식 관계를 맺으면 부모 Bone이 움직일 때 자식도 따라갑니다. 팔 위쪽(Upper Arm)이 움직이면 아래쪽(Forearm)과 손(Hand)도 함께 이동합니다.

팔꿈치를 구부리면 손도 따라오는 것과 같아요

모드 Edit / Pose Mode

Edit Mode에서 뼈를 추가·삭제·위치 조정하고, Pose Mode에서 뼈를 회전·이동하여 포즈를 잡습니다. Rest Position은 편집 기본 자세, Pose Position은 애니메이션 자세입니다.

주요 단축키

Shift + A Armature 추가 (Add → Armature)
E Edit Mode에서 Bone 연장 (Extrude)
Ctrl + P Parent 설정 (Bone 간 또는 메쉬-Armature)
Alt + P Parent 해제
Ctrl + Tab Pose Mode 전환

인터랙티브 데모 — Bone 끝(원)을 드래그해 보세요:

부모 Bone을 움직이면 자식 Bone이 따라옵니다.

Armature가 필요할 때

  • 캐릭터 애니메이션 (걷기, 뛰기)
  • 로봇/기계 관절 움직임
  • 동물 리깅
  • 얼굴 표정 (Facial Rig)

Armature가 불필요한 경우

  • 단순 위치 이동 (Transform만으로 충분)
  • Shape Key로 충분한 변형
  • 정적(Static) 오브젝트
팁: Armature를 메쉬에 연결할 때 Ctrl+P → "Armature Deform → With Automatic Weights"를 선택하면 Blender가 자동으로 Weight를 계산해 줍니다. 복잡한 모델은 이후 Weight Paint로 보정하세요.

자주 쓰는 조합

Armature + Automatic Weights
뼈대 설정 후 자동 Weight로 빠르게 리깅. 간단한 캐릭터에 효과적입니다.
Armature + IK Constraint
Inverse Kinematics로 자연스러운 팔다리 움직임. 발이 땅에 고정된 채 몸이 움직이는 표현에 유용합니다.
