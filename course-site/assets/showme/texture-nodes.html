Texture Node란? 기초

Image Texture(사진), Noise(잡음), Checker(체크무늬), Wave(파도) 등 이미지/절차적 패턴을 생성하는 노드입니다. Shader Editor에서 Material의 색상, 거칠기 등에 연결하여 표면에 패턴을 입힙니다.

Mapping Node 제어

Texture의 위치(Location), 크기(Scale), 회전(Rotation)을 제어하는 노드입니다. Texture Coordinate 노드와 함께 사용하여 패턴의 배치를 정밀하게 조절합니다.

UV vs Generated 좌표계 좌표

  • UV — 수동으로 펼친 좌표 (정밀한 제어, UV Unwrap 필요)
  • Generated — 오브젝트 바운딩 박스 기반 자동 좌표 (빠르지만 변형 시 제한적)

Image Texture

  • 사진 기반 — 실제 이미지 파일 사용
  • 리얼리즘 표현에 최적
  • UV Unwrap 필요
  • 해상도에 제한됨 (확대 시 흐려짐)

Procedural Texture

  • 수학 기반 — 알고리즘으로 패턴 생성
  • 무한 해상도 (확대해도 선명)
  • UV 없이도 사용 가능
  • 파라미터로 자유롭게 커스터마이즈

자주 쓰는 조합

Noise + ColorRamp
Noise 출력을 ColorRamp로 매핑하여 자연스러운 얼룩, 먼지, 산화 효과를 만듭니다.
Checker + Mapping
Mapping으로 Scale/Rotation을 조절해 바닥 타일, 체스보드 패턴을 정밀 배치합니다.
Image + Mapping
이미지 텍스처의 위치와 크기를 Mapping 노드로 미세 조정합니다.
팁: Procedural Texture는 해상도 걱정 없이 클로즈업에서도 선명하게 유지됩니다. 반면 Image Texture는 실제 사진의 디테일을 그대로 살릴 수 있어 리얼리즘에 강합니다. 둘을 혼합하면 최상의 결과를 얻을 수 있습니다.
