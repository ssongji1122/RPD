Color Ramp란? 기초

입력값(0~1 흑백)을 원하는 색상 그라데이션으로 변환하는 노드입니다. "흑백 데이터를 컬러로 바꾸는 번역기"예요.

온도계를 생각해보세요 — 0°C는 파랑, 50°C는 노랑, 100°C는 빨강. Color Ramp가 바로 이 매핑을 합니다

핵심 구성 요소 필수

  • Color Stop (색상 정지점) — 특정 위치에 색상을 지정. 여러 개 추가 가능
  • Fac (입력) — 0~1 사이 값을 받음 (보통 Noise 등 텍스처에서 연결)
  • Color (출력) — 매핑된 색상 결과
  • Interpolation — Linear(부드러운) / Constant(딱딱 끊기는) 등 보간 방식

노드 조작법

Shift+A → Color → Color Ramp Color Ramp 노드 추가
Color Stop 클릭 색상 변경 (하단 색상 바 클릭)
+ 버튼 새 Color Stop 추가
- 버튼 선택된 Color Stop 삭제

🎨 색상 매핑

  • Noise Texture → Color Ramp로 대리석, 나무결 표현
  • 높이 데이터 → 색상으로 지형 그라데이션
  • Fresnel → Color Ramp로 림라이트 효과

⚫ 대비 조절

  • 흑백 텍스처의 명암 대비 강화
  • Roughness 맵 세밀 조정
  • 마스크 경계를 부드럽게/날카롭게

✂️ Constant = 셀 셰이딩

  • Interpolation을 Constant로 바꾸면 딱딱 끊기는 효과
  • 만화풍, 로우폴리 스타일
  • 단계별 색상 분리

🔗 자주 쓰는 연결 조합

  • Noise → Color Ramp → Base Color
  • Noise → Color Ramp → Roughness
  • Layer Weight → Color Ramp → Mix Shader
