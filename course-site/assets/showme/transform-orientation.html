개념 Transform Orientation이란?

G/R/S로 변형할 때 어느 방향이 기준이 될지 정하는 설정이에요. 같은 G+X를 눌러도 Global이냐 Local이냐에 따라 이동 방향이 달라져요.

나침반처럼, 어느 방향이 '앞'인지 기준을 정해줘요

종류 5가지 Orientation

  • Global: 항상 세계 좌표 기준 (X=오른쪽, Z=위)
  • Local: 오브젝트가 기울어진 방향 기준
  • Normal: 선택한 면/엣지의 법선 방향 기준
  • View: 현재 보고 있는 화면 기준
  • Cursor: 3D Cursor 방향 기준

주의 언제 헷갈리나요?

오브젝트를 회전시킨 뒤 G+Z로 올리려 하면 이상한 방향으로 가요. 이게 Global과 Local의 차이예요. Local로 바꾸면 오브젝트의 '위 방향'으로 움직여요.

기울어진 테이블 위에서 '위로 올리기'는 Global 기준 Z가 아니라 Local 기준이에요

필요할 때

  • 오브젝트가 기울어진 방향으로 정확히 이동할 때 (Local)
  • 법선 방향으로 면을 밀 때 (Normal)
  • 현재 화면에서 위아래 이동 (View)

안 쓰는 경우

  • 기본 형태 작업 시 Global로 충분
  • 오브젝트가 회전 없이 기본 배치일 때

자주 쓰는 조합

Local + G/R/S
기울어진 오브젝트를 자기 방향으로 자연스럽게 이동
Normal + Extrude
굴곡진 표면의 법선 방향으로 면 돌출
팁: Ctrl+X/Y/Z를 누르면 반대축으로 제한돼요. 그리고 두 번 눌러서 (예: X X) Local 모드로 빠르게 전환할 수 있어요.
