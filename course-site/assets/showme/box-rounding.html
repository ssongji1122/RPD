베벨 너비 컷 위치가 둥글기를 결정

베벨(Ctrl+B)의 너비(Width)는 원래 모서리에서 얼마나 멀리 잘라낼지를 결정합니다. 너비가 넓을수록 라운딩이 크고 완만해지고, 좁을수록 작고 날카로운 모서리가 됩니다.

나무를 깎을 때 모서리를 많이 깎을수록 더 둥근 것처럼, 베벨 너비 = 깎는 양

R 값 단면 곡선의 모양

같은 너비라도 Profile(R) 값에 따라 단면 곡선의 모양이 달라집니다.

  • R = 0.0 — 안쪽으로 오목하게 파임 (凹)
  • R = 0.5 — 평평한 직선 (기본값)
  • R = 0.75 — 약간 볼록하게
  • R = 1.0 — 완전 원형 볼록 (凸)

세그먼트 수가 많을수록 R 값의 곡선이 더 부드럽게 보입니다.

R 값 = 잘린 단면이 얼마나 부풀거나 꺼지는가

지지 루프컷 Subdivision 고정

Subdivision Surface 모디파이어를 쓸 때, 날카롭게 유지하고 싶은 모서리 가까이 루프컷을 추가하면 해당 영역이 덜 둥글어집니다. 루프컷이 모서리에 가까울수록 더 날카롭고, 멀수록 더 둥글어집니다.

지지 루프컷 = 고무줄이 튀어나가지 않도록 잡아주는 핀

주요 단축키

Ctrl + B 베벨 시작 (드래그로 너비 조절)
스크롤 (베벨 중) 세그먼트 수 조절 (많을수록 부드러운 곡선)
P (베벨 후 패널) Profile(R) 값 변경 — 0.0 ~ 1.0
왼쪽 아래 Operator Panel → Shape 베벨 확정 후 R 값 수치 직접 입력
Ctrl + R 지지 루프컷 추가 (모서리 가까이 배치)
📖 Blender 공식 문서 — Bevel (Profile / Shape 값 포함)

베벨 너비와 R 값을 바꿔가며 모서리 단면이 어떻게 달라지는지 확인하세요.

너비: R 값:

너비 버튼 → 모서리 크기 / R 값 버튼 → 단면 곡선 모양. 4개 코너를 동시에 비교합니다.

지지 루프컷 위치에 따라 Subdivision Surface 결과가 어떻게 달라지는지 확인하세요.

루프컷 위치:

주황 점 = 지지 루프컷 위치. 가까울수록 모서리가 날카롭게, 멀수록 더 둥글어집니다.

베벨 라운딩이 필요할 때

  • 제품/기기 모서리 둥글게 표현
  • 로봇 헤드처럼 딱딱하지만 부드러운 형태
  • 건축 기둥·벽 모서리
  • 렌더링 시 하이라이트 라인 표현

지지 루프컷이 필요할 때

  • Subdivision 쓰면서 모서리를 유지하고 싶을 때
  • 박스 전체는 둥글게, 특정 면만 날카롭게
  • 캐릭터 눈 주변처럼 부분 유지
  • Crease 대신 토폴로지로 해결할 때
R 값 선택 팁: 딱딱한 제품엔 R=1.0 + 세그먼트 3~4가 자연스러운 광택 하이라이트를 만들어 줍니다. 유기적 형태나 캐릭터엔 지지 루프컷 + Subdivision Surface 조합이 더 자연스럽습니다.

자주 쓰는 조합

Bevel R=1.0 + 세그먼트 4
딱딱한 제품·로봇 모서리. 세그먼트를 늘려 둥근 하이라이트 라인을 만드는 기본 방법.
지지 루프컷 + Subdivision Surface
전체를 부드럽게 만들되 특정 모서리만 날카롭게. 루프컷이 가까울수록 더 날카로운 결과.
Bevel Modifier (자동 각도 기반)
특정 각도 이상의 엣지에만 자동으로 베벨 적용. 매번 수동으로 베벨하지 않아도 됨.
Bevel + Mirror + Subdivision
절반만 모델링하면서 모서리 라운딩까지 적용. 순서: Mirror → Bevel → Subdivision.
