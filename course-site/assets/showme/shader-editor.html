Shader Editor란?

노드를 연결해서 재질을 만드는 비주얼 에디터예요. 레고 블록처럼 각 노드가 하나의 기능을 담당하고, 선으로 연결하면 복잡한 재질이 완성됩니다.

레고 블록을 연결하듯 — 색상 블록 + 텍스처 블록 + 출력 블록을 이어서 재질을 조립해요

핵심 노드 3가지

  • Principled BSDF — 재질의 핵심 엔진 (색, 금속, 거칠기 등)
  • Image Texture — 이미지 파일을 재질에 입히기
  • Material Output — 최종 출력 (반드시 필요)

주요 단축키

Shift + A 노드 추가 메뉴
Ctrl + Shift + Click Viewer Node 연결 (노드 출력 미리보기)
Ctrl + T Texture Mapping 노드 자동 연결
M 선택한 노드를 Frame으로 묶기
H 노드 숨기기/접기

노드를 드래그하여 연결 관계를 확인해보세요. 선을 따라가면 데이터 흐름을 이해할 수 있어요.

노드를 드래그해서 위치를 변경할 수 있어요. 각 구성 버튼으로 다른 노드 연결 패턴을 확인하세요.

Shader Editor가 필요할 때

  • 텍스처 이미지를 재질에 입힐 때
  • 복잡한 그라데이션/패턴 재질
  • 투명, 발광, 유리 등 특수 재질
  • 여러 재질을 섞을 때 (Mix Shader)

안 써도 되는 경우

  • 단색 Base Color만 바꾸는 경우
  • 기본 Metallic/Roughness 조절
  • Properties 패널에서 충분한 경우
팁: Ctrl+Shift+Click으로 아무 노드나 클릭하면 그 노드의 출력만 미리 볼 수 있어요. 복잡한 노드 트리를 디버깅할 때 필수입니다.

자주 쓰는 노드 조합

Image Texture + Mapping + Tex Coord
Ctrl+T로 한 번에 자동 연결. UV 기반 텍스처 매핑의 기본 세트.
ColorRamp + Noise Texture
절차적 패턴 생성. 나무결, 대리석, 녹 등 자연스러운 재질 표현.
