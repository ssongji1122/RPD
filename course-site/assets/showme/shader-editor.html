노드란? 기초

Input → Processing → Output 구조의 기능 블록입니다. 소켓을 선으로 연결하면 데이터가 왼쪽에서 오른쪽으로 흐르며 재질이 만들어집니다.

레고 블록처럼 — 각 노드가 하나의 기능을 담당하고, 선으로 연결하면 복잡한 재질이 완성돼요

Socket 색상 규칙 필수

  • 노랑 — Color (색상 데이터)
  • 회색 — Value (숫자 1개)
  • 보라 — Vector (좌표/방향)
  • 초록 — Shader (셰이더 출력)

같은 색 소켓끼리 연결하는 것이 기본이며, 일부는 자동 변환됩니다.

필수 노드 3종 핵심

  • Principled BSDF — 메인 셰이더 (색, 금속, 거칠기 등 통합 제어)
  • Image Texture — 이미지 파일을 재질에 입력
  • Mapping — UV 좌표 변환 (위치, 회전, 크기)
Image Texture → Mapping → Principled BSDF → Material Output 노드 체인

주요 단축키

Shift + A 노드 추가 메뉴 (Add)
Ctrl + T 텍스처 매핑 노드 세트 자동 추가 (Texture Setup)
Ctrl + Shift + Click Viewer Node 연결 (노드 출력 미리보기)
M 선택한 노드를 Frame으로 묶기
H 노드 숨기기/접기

Shader Editor가 필요한 경우

  • 텍스처 이미지를 재질에 입힐 때
  • 복잡한 그라데이션/패턴 재질
  • 투명, 발광, 유리 등 특수 재질
  • 여러 재질을 섞을 때 (Mix Shader)
  • UV 좌표를 직접 조절해야 할 때

Material Properties만으로 충분한 경우

  • 단색 Base Color만 바꾸는 경우
  • 기본 Metallic/Roughness 조절
  • 간단한 Emission 색상 변경
  • Properties 패널 슬라이더로 해결 가능할 때

자주 쓰는 노드 조합

Mix Shader
두 셰이더를 Factor 비율로 혼합. 부식, 칠 벗김, 젖은 표면 등 2가지 재질 전환에 사용.
Color Ramp
그라데이션 커브로 색상 매핑. Noise/Wave 텍스처와 함께 사용하면 자연스러운 패턴 생성.
Mapping Node
텍스처 UV 좌표의 위치/회전/크기를 조절. Ctrl+T로 자동 추가되는 기본 세트에 포함.
Learning Path: 이미지 텍스처를 매핑하고 싶을 때 → Texture Nodes (Image Texture + Mapping + Texture Coordinate) 조합을 먼저 익히세요. Ctrl+T로 한 번에 추가할 수 있습니다.
