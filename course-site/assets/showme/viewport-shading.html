개념 Viewport Shading이란?

3D Viewport에서 모델을 어떻게 표시할지 결정하는 4가지 모드예요. 작업 종류에 따라 골라 쓰면 훨씬 편해요.

사진 편집 앱에서 원본 보기 / 필터 미리보기 / 최종 결과 보기를 전환하는 것처럼요

비교 4가지 모드 비교

  • 🔲 Wireframe (Alt+Z): 폴리곤 구조만 표시. 내부 구조 확인용
  • ⬜ Solid (Z → Solid): 기본 단색 표시. 모델링 작업 시 기본값
  • 🎨 Material Preview (Z → Material): HDRI 조명으로 재질 미리보기
  • 🖥️ Rendered (Z → Rendered): 실제 렌더 결과를 실시간 표시. 느림

설정 Solid 모드 서브 설정

Solid 모드에서도 헤더 오른쪽 구 아이콘을 클릭하면 Matcap 변경, Cavity 표시, X-Ray 켜기 등 세부 설정이 가능해요.

기본 작업등 위에 렌즈 필터를 갈아 끼우는 것처럼 형태 파악을 돕는 설정들이에요

주요 단축키

Z Shading Pie Menu 열기 (4가지 중 선택)
Alt + Z Solid ↔ Wireframe 빠른 전환
Header icon (구 모양) 현재 모드의 세부 설정 패널

버튼을 눌러 각 Shading 모드를 시뮬레이션해 보세요.

폴리곤 구조(와이어프레임)만 표시합니다. 내부 구조 확인이나 루프 파악에 유용해요.

필요할 때

  • Wireframe: 안쪽 폴리곤 확인, 겹치는 면 발견, 루프 구조 파악
  • Solid: 모델링 / 편집의 기본 작업 모드
  • Material Preview: 재질/UV 작업 시 색상 확인
  • Rendered: 최종 결과 확인 (느리므로 주의)

안 쓰는 경우

  • Rendered 모드는 모델링 중엔 성능이 느려서 비추
  • Wireframe만으로는 재질 확인 불가

자주 쓰는 조합

Solid + Cavity ON
면 사이 경계를 더 명확히 표시해서 폴리곤 구조 파악이 쉬워요
Material Preview + HDRI
재질 작업 시 반사/조명 효과를 사전에 확인
팁: Alt+Z가 Wireframe 토글이에요. 모델 안쪽에 버텍스가 겹치는지 확인하거나, 내부 구조를 볼 때 가장 빠른 방법이에요.
