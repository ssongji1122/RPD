Viewport Shading = 3D Viewport에서 오브젝트를 어떻게 표시할지 결정하는 4가지 모드. 작업 목적에 따라 골라 쓰면 생산성이 크게 향상됩니다.

개요 4가지 Shading 모드

  • Wireframe — 엣지만 표시, 토폴로지 확인용
  • Solid — 기본 단색 표시, 모델링 작업의 기본값
  • Material Preview — HDRI 조명으로 재질 미리보기
  • Rendered — 실제 렌더 결과 실시간 미리보기
사진 편집 앱의 원본 보기 / 필터 미리보기 / 최종 결과 보기 전환과 같습니다

단축키 빠른 전환

헤더 우측 4개 아이콘으로 클릭 전환하거나 단축키를 사용합니다.

  • Z — Shading Pie Menu 열기
  • Alt+Z — Wireframe 토글 (Solid ↔ Wire)
  • 헤더 우측 구(球) 아이콘 클릭 — 현재 모드 세부 설정

설정 Solid 모드 옵션

Solid 모드에서 헤더 오른쪽 드롭다운으로 세부 설정이 가능합니다.

  • MatCap — 재질감 시뮬레이션 (조명 없이)
  • Color: Material / Random / Object / Single
  • Cavity — 폴리곤 경계 강조 표시
  • X-Ray — 내부 구조 투과 보기

비교 Material Preview vs Rendered

두 모드는 외관이 비슷하지만 조명 방식이 다릅니다.

  • Material Preview: HDRI 환경 조명 사용 (씬 조명 무시)
  • Rendered: 실제 씬 조명/그림자/반사 계산
  • Material Preview가 훨씬 빠르고 재질 작업에 충분

Wireframe 모드

  • N-Gon(5각형 이상 면) 탐지
  • 메쉬 관통 선택 (뒤쪽 버텍스도 선택)
  • 토폴로지 / 엣지 루프 리뷰
  • 겹치는 버텍스 발견

Solid 모드

  • 모델링 / 편집 작업의 기본 모드
  • 빠른 성능, 형태 명확히 확인
  • MatCap으로 볼륨감 시뮬레이션
  • Cavity ON으로 폴리곤 경계 강조

Material Preview 모드

  • 텍스처 UV 배치 실시간 확인
  • 재질 색상 / 거칠기 빠른 체크
  • HDRI 환경 조명 선택 가능
  • Shader 노드 작업 시 기본 모드

Rendered 모드

  • 최종 조명 / 그림자 / 반사 확인
  • 렌더 전 결과 예측
  • Compositor / 후처리 효과 미리보기
  • 성능 주의 — 간헐적으로만 사용 권장

자주 쓰는 조합

Solid + Cavity ON
면 사이 경계를 강조해 폴리곤 구조 파악이 훨씬 쉬워집니다. Solid 세부 설정에서 Cavity 체크박스 활성화.
Material Preview + HDRI
재질 작업 시 반사/조명 효과를 사전에 확인합니다. 헤더 드롭다운에서 HDRI 환경 이미지를 교체할 수 있습니다.

흔한 실수

실수 1: Rendered 모드인데 아무것도 안 보임
씬에 카메라 또는 조명이 없거나, World 설정에 배경 색상이 없는 경우입니다. 조명과 World Background 색상을 먼저 확인하세요.
실수 2: Material Preview와 실제 렌더가 다름
LookDev(Material Preview)는 HDRI 환경 조명을 사용하고, Rendered는 실제 씬 조명을 사용합니다. 최종 결과는 반드시 Rendered 모드로 확인하세요.
팁: Alt+Z가 Wireframe 토글입니다. 모델 안쪽에 버텍스가 겹치는지 확인하거나, 내부 구조를 볼 때 가장 빠른 방법입니다.
