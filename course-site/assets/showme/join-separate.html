Core Join (Ctrl+J)

여러 오브젝트를 하나로 합치는 기능입니다. Active 오브젝트(마지막으로 선택한 오브젝트)를 기준으로 합쳐지며, 합친 후에는 한 오브젝트로 함께 움직입니다.

Core Separate (P)

Edit Mode에서 선택한 부분을 별도 오브젝트로 분리하는 기능입니다. 3가지 방식을 제공합니다.

  • Selection — 현재 선택한 메시만 분리
  • By Material — 머티리얼 기준으로 분리
  • By Loose Parts — 연결되지 않은 파츠별 분리
레고 블록을 붙이는 게 Join, 떼어내는 게 Separate. 같은 세트지만 따로 움직일 수 있어요

주요 단축키

Ctrl + J Join — Object Mode에서 선택한 오브젝트들을 하나로 합침
P Separate — Edit Mode에서 선택한 부분을 별도 오브젝트로 분리
Selection / By Material / By Loose Parts Separate 실행 시 나타나는 3가지 분리 방식 선택
📖 Blender 공식 문서 — Join

Join과 Separate 버튼을 눌러 오브젝트가 합쳐지고 분리되는 과정을 확인해 보세요.

왼쪽: 분리된 3개의 오브젝트 (각각 주황 아웃라인). 오른쪽: 하나로 합쳐진 오브젝트 (하나의 아웃라인).

Join이 필요할 때

  • 파츠를 하나의 메시로 통합할 때 (최종 정리)
  • 같이 움직여야 할 부품들
  • Export 전 오브젝트 수 정리

Separate가 필요할 때

  • 관절/움직이는 부품 분리
  • UV/Material을 따로 관리할 때
  • 파츠별 Modifier 적용 시

자주 쓰는 조합

Join + Ctrl+L (Select Linked)
합친 후 원래 파츠를 빠르게 재선택할 수 있습니다. Edit Mode에서 Ctrl+L로 연결된 메시를 한번에 선택합니다.
Separate by Loose Parts
연결되지 않은 메시를 한번에 전부 분리합니다. 여러 오브젝트를 Join한 후 다시 개별 분리할 때 유용합니다.
팁: Join하면 Active 오브젝트의 Origin이 기준이 돼요. 합치기 전에 Origin 위치를 확인하세요!
