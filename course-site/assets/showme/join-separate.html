Join은 여러 오브젝트를 하나로 합치고, Separate는 하나의 오브젝트를 여러 개로 분리합니다. 메쉬 데이터 관리의 핵심입니다.

Join Ctrl+J 합치기

Object Mode에서 여러 오브젝트를 선택 후 Ctrl+J로 단일 메쉬 오브젝트로 합칩니다.

  • Origin: Active Object(마지막 선택)의 Origin이 기준
  • Materials: 각 오브젝트의 Material 슬롯이 합쳐짐
  • 결과: 하나의 오브젝트, 내부적으로 여러 Loose Part

Separate P 분리하기

Edit Mode에서 선택 후 P를 눌러 분리 방식을 선택합니다.

  • Selection: 선택한 버텍스/페이스만 분리
  • By Material: Material 기준으로 분리
  • By Loose Parts: 연결되지 않은 파트 전부 자동 분리
레고 블록을 붙이는 것이 Join, 떼어내는 것이 Separate — 합치든 나누든 블록 자체 모양은 변하지 않아요

Join이 필요할 때

  • 최종 에셋으로 파츠 통합 시
  • 퍼포먼스 최적화 (Draw Call 감소)
  • 공통 Material 적용 전
  • Export 전 오브젝트 수 정리
  • 같이 움직여야 할 부품 묶기

Separate가 필요할 때

  • 모델링 중 일부를 독립 오브젝트로
  • 파츠별 Physics 개별 적용 시
  • 관절/움직이는 부품 분리
  • UV / Material을 따로 관리할 때
  • Modifier를 파츠별로 다르게 적용 시

흔한 실수 & 해결법

Join 후 Origin이 엉뚱한 위치
마지막으로 선택(밝게 빛나는 오브젝트)이 Active Object입니다. Join 전에 원하는 오브젝트를 마지막으로 클릭하세요.
Separate 후 오브젝트가 너무 많이 생김
Loose Parts가 예상보다 많은 경우입니다. Edit Mode에서 Mesh → Merge by Distance로 떨어진 버텍스를 먼저 연결하세요.
Join 후 Material이 두 배로 늘어남
각 오브젝트의 Material 슬롯이 합산됩니다. 중복 Material은 Material Properties에서 정리하세요.
원하는 파트만 Separate하고 싶을 때
Edit Mode에서 해당 파트 위에 마우스를 올리고 Ctrl+L로 연결 메쉬 전체 선택 후 P → Selection.
팁: Join 전에는 항상 어떤 오브젝트가 Active(밝게 빛나는 오브젝트)인지 확인하세요. 마지막으로 클릭한 오브젝트의 Origin이 기준이 됩니다.
