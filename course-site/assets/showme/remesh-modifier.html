Core Remesh란?

메쉬의 토폴로지를 완전히 재구성하는 모디파이어입니다. 지저분한 메쉬를 균일한 사각형 또는 삼각형 그리드로 바꿔줍니다.

구겨진 종이를 펴서 새 격자무늬 종이로 바꾸는 것과 같습니다

Modes 4가지 모드

  • Voxel — 가장 많이 사용. 균일한 복셀 크기로 재구성
  • Smooth — 부드럽게 재구성, 형태 유지
  • Sharp — 날카로운 모서리를 유지하며 재구성
  • Blocks — 마인크래프트 스타일 복셀화

Setting Voxel Size

Voxel 크기가 작을수록 원본 디테일을 더 잘 보존하지만, 면 수가 크게 증가합니다. 클수록 단순화됩니다.

  • 작은 값 — 디테일 보존, 면 수 증가
  • 큰 값 — 단순화, 면 수 감소

왼쪽: 원본 메쉬 (지저분한 토폴로지) → 오른쪽: Remesh 결과 (깨끗한 토폴로지)

Remesh가 필요할 때

  • Sculpt 후 리토폴로지
  • Boolean 연산 후 메쉬 정리
  • AI 생성 모델 토폴로지 정리
  • 3D 스캔 데이터 정리

Remesh를 안 쓰는 경우

  • 이미 깨끗한 토폴로지
  • UV가 필요한 모델 (UV 파괴됨)
  • 정밀한 Edge Flow가 필요할 때
  • 애니메이션용 토폴로지
주의: Remesh는 UV 맵을 파괴합니다! UV 작업 전에 사용하세요. Remesh 후에는 반드시 UV를 새로 펴야 합니다.

자주 쓰는 조합

Boolean → Remesh
Boolean 연산 후 지저분해진 메쉬를 Remesh로 깨끗하게 정리합니다.
Sculpt → Remesh → Decimate
Sculpt 결과를 균일하게 정리한 뒤 Decimate로 면 수를 줄여 최적화합니다.
