Solidify란?

평면(면)에 두께를 추가하는 모디파이어입니다. 종이처럼 얇은 메쉬를 실제 물체처럼 두께 있는 3D로 만들어 줍니다.

종이에 두께를 입혀 판자로 만드는 것

Offset 방향

Offset 값으로 두께가 추가되는 방향을 제어합니다.

  • -1 : 안쪽(노멀 반대 방향)
  • 0 : 양쪽 균등
  • +1 : 바깥쪽(노멀 방향)

Even Thickness를 켜면 경사면에서도 균일한 두께를 유지합니다.

Thickness

두께 값을 직접 지정합니다. 양수가 기본이며, 음수를 입력하면 방향이 반전됩니다.

더보기
입문 단계에서는 `Thickness = 두께`, `Offset = 어느 쪽으로 생기는지`, `Even Thickness = 코너 보정` 이 세 가지만 먼저 기억하면 충분합니다.
개념 이해 시각화

Solidify는 같은 평면이라도 Thickness와 Offset 조합에 따라 “종이”, “얇은 패널”, “중앙 기준 두께”처럼 전혀 다른 결과를 만듭니다.

Thickness 0.00

면만 있는 상태

아직 두께가 없어서 한쪽 면처럼 보입니다. 렌더나 출력에서 실제 부피가 없는 상태예요.

Thickness 0.04 / Offset +1

바깥으로 두께 추가

원본 면을 기준으로 바깥쪽으로만 벽이 자랍니다. 외곽 실루엣을 유지해야 할 때 직관적입니다.

Thickness 0.04 / Offset 0

중앙 기준 확장

기존 면을 중심으로 양쪽에 반씩 두께가 생깁니다. 기준선을 유지하며 패널을 만들 때 유용합니다.

Even Thickness On

모서리 두께 보정

굽은 부분이나 경사면에서 두께가 찌그러지는 걸 줄여 줍니다. 코너를 만질 때 더 “실제 벽”처럼 보입니다.

조작 방법

Add Modifier Properties → Modifiers → Add Modifier → Solidify
Thickness 모디파이어 패널에서 직접 입력
Even Thickness 경사면 두께 보정 체크박스

Solidify가 필요할 때

  • 갑옷 / 헬멧
  • 그릇 / 컵
  • 유리창
  • 건축 벽체
  • 기계 패널

Solidify를 안 쓰는 경우

  • 이미 닫힌 메쉬 (Solid)
  • 솔리드 오브젝트
팁: Solidify는 열린 메쉬(Open Mesh)에 사용합니다. 이미 닫힌 메쉬에는 효과가 이상해질 수 있어요.

자주 쓰는 조합

Solidify + Subdivision
부드러운 두께감. Solidify가 위, Subdivision이 아래 순서로 배치합니다.
Solidify + Mirror
대칭 형태에 두께 추가. Mirror가 위, Solidify가 아래 순서로 배치합니다.
