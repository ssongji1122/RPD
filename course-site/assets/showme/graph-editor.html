Dope Sheet "언제"

Dope Sheet는 키프레임의 타이밍을 관리하는 에디터입니다. 키프레임이 어느 프레임에 있는지 한눈에 보여주며, 마름모를 드래그하여 타이밍을 조절합니다.

악보에서 각 음표가 언제 연주되는지 보여주는 것과 같아요

Graph Editor "어떻게"

Graph Editor는 키프레임 사이의 보간(Interpolation)을 F-Curve로 보여줍니다. 커브의 기울기가 속도를 나타내며, 핸들을 조작하여 움직임의 느낌을 정교하게 제어합니다.

자동차 속도계처럼, 같은 거리를 이동해도 어떻게 가속/감속하는지 보여줍니다

Interpolation 종류 곡선

키프레임 사이의 보간 방식을 바꾸면 동작의 느낌이 완전히 달라집니다:

  • Linear: 일정한 속도 (로봇 같은 움직임)
  • Bezier: 부드러운 가감속 (기본값)
  • Constant: 보간 없음 (깜빡이는 효과)
  • Bounce: 바운스 효과

주요 단축키 (Graph Editor)

T Interpolation 타입 변경
V Handle 타입 변경 (Free, Aligned, Vector, Auto)
G 키프레임/핸들 이동
Home 전체 F-Curve 보기
Shift + E Extrapolation 모드 (Make Cyclic 등)

세 가지 Interpolation 커브(Linear, Ease, Bounce)를 비교하세요. 위쪽은 F-Curve, 아래쪽은 실제 공의 움직임입니다.

각 Interpolation 타입을 선택하면 커브 모양과 공의 움직임이 달라집니다.

Dope Sheet 사용 시점

  • 키프레임 타이밍 한눈에 확인
  • 여러 채널의 키프레임 동시 이동
  • 액션(Action) 관리
  • 전체 애니메이션 흐름 파악

Graph Editor 사용 시점

  • 움직임의 가감속 조절
  • 바운스, 탄성 같은 효과 추가
  • F-Curve 핸들 미세 조정
  • Interpolation/Extrapolation 설정
팁: Dope Sheet에서 타이밍을 먼저 잡고, Graph Editor에서 곡선을 미세 조정하는 것이 효율적인 워크플로우입니다. T로 Interpolation 타입을 빠르게 전환하세요.

Graph Editor 활용 기법

Ease In / Ease Out
시작과 끝을 부드럽게. 자연스러운 동작의 기본입니다. Bezier 핸들의 길이로 가감속 정도를 조절합니다.
Make Cyclic (Shift+E)
F-Curve를 반복 재생합니다. 걷기, 호흡, 진동 같은 반복 동작에 유용합니다.
