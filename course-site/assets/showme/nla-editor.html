NLA Editor

Non-Linear Animation Editor의 줄임말. 애니메이션 액션을 영상 편집처럼 트랙 위에 클립으로 배치하고 순서·겹침을 자유롭게 조합하는 편집기예요.

Premiere Pro의 타임라인과 동일한 개념 — 클립을 트랙에 올려 순서와 겹침을 편집해요.

Action

걷기, 점프, 손흔들기처럼 하나의 완결된 애니메이션 동작이에요. Action Editor에서 만들고, NLA 트랙에 클립(Strip)으로 꺼내 씁니다.

  • 오브젝트당 여러 Action 저장 가능
  • Action은 키프레임 묶음

Action Strip

Action을 NLA 트랙 위에 올린 것. 시작·종료 프레임, 반복 횟수(Repeat), 블렌드 인/아웃, 스케일을 개별 설정할 수 있어요.

  • G 이동 / S 스케일
  • 스트립 우클릭 → Duplicate로 복제

트랙 블렌딩

여러 트랙을 쌓으면 아래에서 위 순서로 블렌드되어 합산됩니다. 블렌드 모드(Replace / Add / Multiply 등)로 합성 방식을 조절해요.

  • 위 트랙이 아래 트랙을 덮어씀(Replace 기본)
  • Add 모드로 동작을 더할 수 있음

Push Down

Action Editor에서 현재 액션을 NLA 트랙으로 내리는 작업이에요. Push Down하지 않으면 NLA에서 스트립이 보이지 않습니다.

  • Action Editor 헤더의 Push Down 버튼 클릭
  • Push Down 후 Tweak Mode로 재편집 가능

Tweak Mode

스트립 내부 키프레임을 직접 편집하는 모드예요. Tab으로 진입/탈출합니다. Tweak Mode 밖에서 키프레임을 수정하려 해도 반영되지 않아요.

  • 편집 중 스트립 테두리가 밝게 강조됨
  • 탈출 시 변경 내용 자동 저장

주요 단축키

Tab Tweak Mode 진입 / 탈출 (스트립 내부 편집)
G 선택한 스트립 이동
S 스트립 스케일 (길이 조정)
N 스트립 속성 패널 열기
Shift+A 새 스트립 추가
Alt+A 전체 선택 해제
Blender 공식 문서 — NLA Editor

스트립을 드래그해서 타임라인 위에서 이동해 보세요. 겹치는 구간은 노란 블렌드 영역으로 표시됩니다.

Mixamo 애니메이션 조합

  • Mixamo에서 가져온 Walk, Run, Jump를 하나의 캐릭터에 합칠 때
  • 각 동작을 NLA 트랙에 순서대로 배치해 시퀀스 구성
  • 반복(loop) 횟수를 Repeat로 설정

레이어 블렌드 (동작 합산)

  • 걷기 + 손흔들기처럼 두 동작을 위·아래 트랙에 올려 합산
  • Add 블렌드 모드로 동작이 서로 더해짐
  • Influence 값으로 합산 비율 조절 가능

타이밍 오프셋 & 반복

  • 같은 액션을 여러 스트립으로 복제해 타이밍을 어긋나게 배치
  • 스케일(S)로 스트립 길이 조정 — 동작 속도 변경
  • Strip Offset으로 시작 위치 세밀 조정

게임 엔진 내보내기 준비

  • Unity / Unreal용 FBX 내보내기 전 액션 정리
  • 각 액션을 NLA 트랙으로 정리하면 별도 클립으로 분리됨
  • Action 이름을 명확하게 지정해야 엔진에서 인식
자주 하는 실수
Action을 만들고 Push Down하지 않으면 NLA Editor에서 스트립이 보이지 않아요. Action Editor 헤더에서 Push Down 버튼을 반드시 눌러야 해요.

또한 스트립 내 키프레임을 수정하려면 반드시 Tab으로 Tweak Mode에 진입해야 합니다. 트랙의 위·아래 순서가 블렌드 결과에 영향을 주므로, 아래 트랙이 기본(Base) 동작이 되도록 배치하세요.

워크플로 흐름

① Action 제작
Action Editor에서 키프레임 설정해 개별 동작 완성
② Push Down

③ 스트립 배치
NLA Editor에서 스트립 이동(G), 스케일(S), 복제로 배치
④ 블렌드 설정
N 패널에서 Blend Mode / Influence / Repeat 조정
