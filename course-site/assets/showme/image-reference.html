모델링용 Reference

뷰포트에 반투명하게 떠 있는 가이드 이미지입니다. 편집에 전혀 영향을 주지 않으며, 렌더링에 포함되지 않습니다.

  • 선택은 되지만 메쉬가 아님
  • Properties Data > Opacity 활성화 후 조절 가능
  • Outliner 필터에서 Selectable 활성화 후 선택 방지 가능
  • 로봇 모델링 가이드로 가장 적합
스케치북 위에 트레이싱지 올려놓은 것처럼, 밑그림만 보이고 건드릴 수 없어요

카메라 배경 Background

카메라 배경에 딱 붙는 이미지입니다. Camera View(Numpad 0)에서만 보입니다.

  • 배경 합성, VFX 매핑에 사용
  • 일반 뷰포트에서는 보이지 않음
  • 카메라와 연동되어 항상 정렬됨
  • 렌더링에 포함되지 않음
카메라 뷰파인더 뒤에 붙여 놓은 배경 사진이에요

실제 메쉬 Mesh Plane

이미지가 텍스처로 입혀진 실제 메쉬 면입니다. Edit Mode에서 편집할 수 있습니다.

  • 렌더링에 포함됨
  • UV 자동 설정
  • 나무·잎사귀 등 실제 오브젝트로 활용
  • Solidify, Array 등 Modifier 적용 가능
이미지를 종이처럼 오려서 3D 공간에 세워 놓은 거예요

빈 슬롯 Empty Image

이미지가 없는 빈 Empty 오브젝트입니다. 나중에 이미지를 할당할 수 있는 자리만 만들어 둡니다.

  • 이미지 플레이스홀더로 사용
  • Properties에서 이미지 나중에 할당
  • 파이프라인 초기 구조 잡을 때 유용
  • Reference와 동일한 방식으로 동작
액자 틀만 걸어 두고 사진은 나중에 끼워 넣는 거예요

주요 단축키

Shift + A → Image → Reference 레퍼런스 이미지 추가
Numpad 1 / 3 / 7 Front / Right / Top 직교뷰 전환
N 패널 → Item 위치·크기 조절 (투명도는 Properties Data > Opacity 활성화)
H / Alt+H 레퍼런스 숨기기 / 다시 보이기

렌더 결과

Reference를 쓸 때

  • 로봇 모델링 가이드 이미지
  • 캐릭터 전면/측면 ortho 레퍼런스
  • 비율을 맞춰야 하는 모든 모델링
  • 건축 도면 tracing

다른 타입을 쓸 때

  • Background: VFX, 카메라 매핑, 배경 합성
  • Mesh Plane: 실제 오브젝트로 필요할 때 (나무, 잎사귀 등)
  • Empty Image: 나중에 교체할 이미지 플레이스홀더
팁: 레퍼런스 추가 후 Properties(Object Data Properties) > Opacity를 활성화하고 0.3~0.5로 낮추면 모델링 도중 메쉬와 구분이 쉬워요. Outliner 필터(깔대기 아이콘)에서 Selectable을 활성화한 뒤 레퍼런스 옆 셀렉터블 아이콘을 끄면 실수로 이동시키는 걸 막을 수 있어요.
주의: 레퍼런스 이미지는 특정 뷰(Numpad 1=정면, Numpad 3=우측면)에서 추가해야 해당 뷰에 정렬됩니다. Object Mode에서 추가하세요.

레퍼런스 세팅 흐름

정면 + 측면 두 장
Numpad 1에서 정면 Reference 추가 → Numpad 3에서 측면 Reference 추가 → 크기 맞춰 정렬
Reference + Mirror
정면 레퍼런스로 비율 잡기 → Mirror Modifier로 한쪽만 편집해 대칭 완성
