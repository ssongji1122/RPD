Mirror Modifier란?

거울처럼 한쪽만 편집하면 반대쪽이 자동으로 따라오는 Modifier입니다. 좌우 대칭 모델을 만들 때 작업량을 절반으로 줄여줍니다.

거울 앞에서 한쪽을 바꾸면 반대쪽이 자동으로 따라와요

Clipping이란?

중심선을 넘지 못하게 하는 옵션입니다. ON이면 중심 정점이 정확히 중앙에 고정되어, 양쪽이 깔끔하게 이어집니다.

종이를 반으로 접으면 접힌 선(중심선)이 경계가 돼요. Clipping은 그 선 너머로 펜이 못 넘어가게 막는 역할이에요

Origin(원점)이 핵심이에요

Mirror는 오브젝트의 원점(Origin)을 기준으로 반전합니다. 원점이 메시 중앙에 없으면 미러가 엉뚱한 곳에 나타나요. 가장 흔한 실수

Mirror 추가 전 반드시: Ctrl+A → All Transforms 적용 → 원점 확인 순서로 시작하세요

주요 단축키

Ctrl + A All Transforms 적용 — Mirror 추가 전 반드시 실행 필수
Shift + S → Cursor to World Origin 3D 커서를 월드 원점으로 — 원점 리셋 시작점
우클릭 → Set Origin → Origin to 3D Cursor 오브젝트 원점을 중앙으로 이동
S X 0 중심선 정렬 (선택한 정점의 X좌표를 0으로)

왼쪽 반의 꼭짓점을 드래그하면 오른쪽이 자동으로 미러됩니다.

꼭짓점을 드래그해 보세요. Clipping을 토글하면 중심선 넘기 동작이 달라집니다.

Mirror가 필요할 때

  • 좌우 대칭 로봇
  • 자동차
  • 캐릭터 얼굴
  • 가구

Mirror를 안 쓰는 경우

  • 비대칭 디자인
  • 유기적 형태
  • 한쪽만 다른 디테일
팁: Mirror는 원점(Origin)이 중심에 있어야 해요. S+X+0으로 중심선을 맞추고, 오브젝트를 먼저 중앙에 놓은 뒤 Mirror를 추가하세요.

자주 쓰는 조합

Mirror + Subdivision Surface
Mirror로 대칭을 잡고 Subdivision으로 부드럽게. Mirror가 위, Subdivision이 아래 순서로 배치합니다.
Mirror + Solidify
대칭 형태에 두께를 추가할 때. 얇은 판이나 갑옷 같은 모델에 유용합니다.
