Simple Deform이란? Modifier

메쉬를 간단한 수학적 변형으로 왜곡하는 Modifier입니다. 4가지 모드가 있으며, 각각 다른 방식으로 오브젝트를 변형합니다.

점토를 손으로 비틀고, 구부리고, 늘리는 것과 같아요

4가지 모드 핵심

  • Twist — 축을 중심으로 비틀기 (나선형)
  • Bend — 축을 기준으로 구부리기 (아치형)
  • Taper — 한쪽 끝을 좁히기 (뾰족하게)
  • Stretch — 축 방향으로 늘리기/압축

주요 단축키

Ctrl + 1 Modifier 추가 패널
Properties → Modifier Angle/Factor 슬라이더로 변형 강도 조절
Origin 오브젝트 Empty를 Axis Origin으로 지정해 변형 기준점 변경
📖 Blender 공식 문서 — Simple Deform Modifier
상태 전환 버튼으로 변형 모드를 바꿔서, 같은 실린더가 비틀림·굽힘·테이퍼·스트레치 중 어떤 방식으로 바뀌는지 비교합니다.
Simple Deform은 같은 메시를 다른 수학적 규칙으로 변형하는 모디파이어입니다. 어떤 모드를 고르느냐에 따라 실린더는 나선, 아치, 뿔, 압축된 기둥처럼 완전히 다르게 읽힙니다.

Wrong Mode

비슷해 보여도 모드마다 변형 규칙이 달라서, Bend로 해결할 문제를 Twist로 접근하면 형태가 쉽게 꼬여 보이거나 의도와 멀어집니다.

Right Deform

각 모드를 “회전”, “굽힘”, “끝 좁히기”, “축 늘리기”로 나누어 보면 원하는 실루엣을 가장 빠르게 만드는 변형 방식을 고르기 쉬워집니다.

버튼을 눌러 각 모드의 변형 효과를 확인하세요. 왼쪽이 원본, 오른쪽이 변형 결과입니다.

Simple Deform이 유용할 때

  • 나선형 기둥/계단 (Twist)
  • 아치/다리/터널 (Bend)
  • 뿔/송곳/탄환 (Taper)
  • 카툰 스타일 늘리기 (Stretch)

다른 방법이 나을 때

  • 복잡한 유기체 변형 → Sculpt
  • 경로 따라 변형 → Curve Modifier
  • 정밀한 로컬 변형 → Lattice
팁: Simple Deform은 세분화(Subdivision)가 충분해야 부드럽게 변형됩니다. 실린더라면 최소 32개 이상의 세그먼트를 권장합니다. 변형 축은 Axis Origin (Empty)으로 자유롭게 조절할 수 있어요.

자주 쓰는 조합

Simple Deform + Array
Twist로 나선을 만들고 Array로 반복하면 스프링이나 DNA 구조를 쉽게 만들 수 있습니다.
Simple Deform + Subdivision
Subdivision으로 메쉬를 촘촘하게 한 뒤 Bend를 적용하면 더 부드러운 곡선이 됩니다.
