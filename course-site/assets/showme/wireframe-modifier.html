Core Wireframe이란?

메쉬의 모서리를 실제 두께가 있는 와이어프레임으로 변환하는 모디파이어입니다. 모든 면을 모서리 골격만 남기는 효과를 줍니다.

종이 상자에서 면을 다 뜯어내고 뼈대만 남기는 것

Setting Thickness

와이어의 두께를 조절합니다. Even Thickness 옵션을 켜면 모서리 각도에 관계없이 균일한 두께를 유지합니다.

  • Thickness 작음 → 가느다란 와이어
  • Thickness 큼 → 두꺼운 와이어 (겹칠 수 있음)

Option Replace Original

ON이면 원본 면을 제거하고 와이어만 남깁니다. OFF면 원본 면 위에 와이어를 추가합니다.

  • ON → 순수 와이어프레임 (골격만)
  • OFF → 원본 면 + 와이어프레임 동시
📖 Blender 공식 문서 — Wireframe Modifier
상태 전환 연속 조절 모드 버튼은 원본 유지 여부를 바꾸고, Thickness 슬라이더는 와이어의 존재감을 연속적으로 조절합니다.
Wireframe Modifier는 선을 보여 주는 기능이 아니라, 메쉬의 엣지를 실제 두께가 있는 구조물로 바꾸는 작업입니다. Replace Original과 Thickness 조합에 따라 골격만 남길지, 원본 위에 구조를 덧씌울지가 갈립니다.

Collapsed Grid

두께를 무작정 키우거나 원본을 함께 두는 이유를 모르면, 구조가 읽히기보다 면과 와이어가 서로 겹쳐 보여 결과가 탁해집니다.

Structural Read

모드로 원본 유지 여부를, Thickness로 골격의 존재감을 나눠 보면 홀로그램, 랜턴, 프레임 구조를 훨씬 명확하게 설명할 수 있어요.

0.06

모드를 전환하면 Replace Original ON/OFF 효과를 확인할 수 있습니다.

Wireframe이 필요할 때

  • 건축 와이어프레임 표현
  • 홀로그램 효과
  • 장식용 패턴 (랜턴, 조명)
  • 3D 프린팅 구조물

Wireframe을 안 쓰는 경우

  • 일반 솔리드 렌더링
  • 유기적 형태 모델링
  • 솔리드 모델이 필요한 경우

자주 쓰는 조합

Wireframe + Subdivision Surface
Subdivision으로 메쉬를 세분화한 뒤 Wireframe을 적용하면 부드럽고 촘촘한 와이어 패턴이 만들어집니다.
Wireframe + Emission Material
와이어프레임에 Emission(발광) 재질을 적용하면 SF 홀로그램이나 글로우 효과를 연출할 수 있습니다.
