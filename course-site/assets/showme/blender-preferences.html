설정 Preferences란?

Blender 전체 동작 방식을 내 환경에 맞게 조정하는 설정 창이에요. 마우스 입력 방식, 자동 저장 주기, 테마, 단축키까지 모두 여기서 바꿀 수 있어요.

스마트폰의 '설정' 앱과 똑같아요. 기기를 처음 받으면 언어·알림·밝기를 먼저 맞추듯, Blender도 Preferences를 먼저 설정해요.

입력 Emulate 3 Button Mouse

마우스 휠 버튼(중간 버튼)이 없을 때 켜는 옵션이에요. ON으로 설정하면 Alt + 좌클릭으로 화면을 돌릴 수 있어요.

세 버튼짜리 마우스를 흉내내는 옵션이에요. 노트북 트랙패드나 일반 마우스로도 Blender를 쓸 수 있게 해줘요.

저장 Auto-Save

일정 시간마다 자동으로 임시 파일을 저장해요. 갑자기 프로그램이 꺼져도 최근 작업을 복구할 수 있어요. 기본값은 2분 간격이에요.

구글 독스가 글을 쓸 때마다 자동 저장하는 것과 같아요. Blender도 내 작업을 몰래 백업해 둬요.

주요 단축키 · 경로

Ctrl + , Preferences 창 열기 (쉼표)
Edit → Preferences 메뉴에서 직접 접근
Input 탭 Emulate 3 Button Mouse 위치
Save & Load 탭 Auto-Save 간격 설정 위치
Themes 탭 UI 색상 테마 변경

Blender를 처음 켰을 때 해두면 좋은 설정 순서예요.

① Preferences 열기

상단 메뉴 Edit → Preferences 또는
Ctrl + , 단축키로 열어요.

② Input 탭 이동

왼쪽 사이드바에서 Input을 클릭해요.
마우스 입력 관련 설정이 여기 있어요.

③ Emulate 설정 (마우스 없으면)

마우스 중간 버튼(휠 클릭)이 없으면
Emulate 3 Button Mouse를 체크하세요.

이걸 켜면 Alt+좌클릭으로 화면을 돌릴 수 있어요.

④ Numpad 없으면

노트북처럼 Numpad가 없으면
Emulate Numpad도 같이 체크해요.
일반 숫자키 1~9로 뷰 전환이 돼요.

⑤ Save & Load 탭

Auto Save가 켜져 있는지 확인만 해두세요.
기본값(2분 간격)이면 충분해요.

⑥ 설정 저장

창 하단 Save Preferences를 눌러야
다음에 Blender를 켜도 설정이 유지돼요.

저장 안 하면 다음에 또 설정해야 해요!
처음엔 이 2가지만: Input → Emulate 3 Button Mouse 켜기 (마우스 없을 때), Save & Load → Auto Save ON 확인. 나머지는 기본값으로 충분해요.

Preferences가 필요할 때

  • 마우스 중간 버튼이 없을 때
  • Numpad가 없는 노트북
  • UI가 너무 작거나 클 때
  • 자동 저장 간격을 바꿀 때
  • 테마 색상을 바꿀 때
  • 유용한 Add-on을 켤 때

꼭 확인할 2가지

  • Emulate 3 Button Mouse — 마우스 중간 버튼 없으면 켜기
  • Auto Save — 켜져 있는지 확인만 해두기
  • 나머지는 기본값으로 충분해요

유용한 Add-on 조합

Node Wrangler
셰이더 노드 작업을 훨씬 빠르게 해줘요. 6주차 이후에 꼭 활성화하세요.
Looptools
원형 정렬, 브릿지 등 Edit Mode 보조 도구를 추가해 줘요.
Import-Export: Images
다양한 이미지 포맷을 텍스처로 임포트할 때 유용해요.
Rigify
캐릭터 리깅을 위한 자동 뼈대 생성 Add-on. 11주차 리깅 때 필요해요.
팁: Preferences 설정은 자동으로 저장돼요. 한 번 설정해두면 Blender를 다시 켜도 유지됩니다. 단, 수동으로 Save Preferences 버튼을 누르거나 Auto-Save Preferences가 켜져 있어야 해요.
