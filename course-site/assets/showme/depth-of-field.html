🎥 Depth of Field란?

카메라의 초점이 맞는 거리 범위입니다. 이 범위 안에 있는 피사체는 선명하게, 범위 밖은 흐릿하게(보케) 렌더됩니다.

DSLR 카메라의 조리개 설정과 동일한 원리입니다. f-Stop = 조리개 크기, Focus Distance = AF 초점거리.

Focus Distance

카메라에서 초점이 맞는 거리(m 단위)입니다. 이 거리에 있는 피사체만 완전히 선명하게 렌더됩니다.

  • 숫자가 클수록 먼 곳에 초점
  • 숫자 직접 입력 또는 슬라이더로 조정
  • Focus Object를 지정해 동적 추적도 가능

f-Stop (조리개)

낮을수록 초점 범위가 좁아집니다(심도가 얕아집니다). 강한 아웃포커싱 효과를 줍니다.

  • f/1.4 — 배경 거의 다 흐림 (인물/제품 촬영)
  • f/8 — 어느 정도 심도 있음 (풍경 사진)
  • f/22 — 전체가 대부분 선명

Blades (보케 모양)

보케의 형태를 결정합니다. 실제 카메라 조리개 날 수를 시뮬레이션합니다.

  • 0 — 원형 보케
  • 6 — 육각형 보케
  • 8 — 팔각형 보케

렌더 엔진별 차이

Cycles — 물리 기반 DOF. 사실적이지만 f-Stop이 낮으면 노이즈 증가 → 샘플 수를 늘려야 합니다.

EEVEE — 후처리(Post Processing) 방식. 빠르지만 픽셀리(pixelated)할 수 있음 → DOF 샘플 수 증가로 해결.

활성화 방법

Properties 녹색 카메라 아이콘 패널 열기
Depth of Field 체크박스로 활성화
Focus Distance 초점 거리 입력 (m)
Blender 공식 문서 — Depth of Field
슬라이더를 움직여 초점 거리와 f-Stop에 따른 심도 변화를 확인하세요.

제품 / 인물 촬영 연출

  • 주인공 오브젝트만 선명하게
  • 배경을 보케로 흐릿하게 처리
  • 낮은 f-Stop(f/1.4~f/2.8) 사용

시네마틱 장면 연출

  • 영화 같은 피사계 심도 표현
  • 카메라 이동에 따른 Focus Pull 애니메이션
  • Cycles 렌더로 사실적인 보케

배경 강조 분리

  • 복잡한 배경에서 주요 오브젝트 돋보이게
  • 스토리 포인트를 시선으로 유도
  • 조명과 함께 사용 시 효과 극대화

주의사항 / 흔한 실수

  • EEVEE 픽셀리 → DOF 샘플 수 증가
  • Focus Distance 미확인으로 엉뚱한 초점
  • Cycles f-Stop 낮음 → 샘플 수 늘려야

함께 쓰면 좋은 설정

Focus Object
특정 오브젝트를 지정해 자동으로 초점 추적. 애니메이션에 유용.
Camera > Lens
Focal Length(mm)를 높이면 압축 효과와 함께 DOF가 더 강조됨.
Compositor > Defocus
후처리 단계에서 DOF를 더 세밀하게 제어할 수 있음.
EEVEE — Post Processing
Render Properties > Post Processing에서 DOF 샘플 수 조정.
팁: 애니메이션에서 Focus Distance를 키프레임으로 등록하면, 피사체가 이동할 때 초점이 따라가는 Focus Pull 효과를 연출할 수 있습니다.
