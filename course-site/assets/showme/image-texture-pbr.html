PBR 맵이란? 기초

Physically Based Rendering — 물리 기반 렌더링에 필요한 이미지 세트입니다. 하나의 재질을 표현하기 위해 여러 장의 이미지를 사용해요.

화장에 비유하면: Base Color = 파운데이션(피부색), Roughness = 파우더(광택 조절), Normal = 콘투어링(입체감 추가)

PBR 맵 3종 필수

  • Diffuse (Base Color) — 물체의 기본 색상. 컬러 이미지
  • Roughness — 표면 거칠기. 흑백 이미지 (검=매끄러움, 흰=거침)
  • Normal — 표면 요철/디테일. 파란색 이미지. 실제 메쉬 변형 없이 입체감

Color Space 주의! 꼭 기억

Blender에서 이미지를 로드할 때 Color Space 설정이 중요합니다:

  • Diffuse → sRGB (기본값, 변경 불필요)
  • Roughness → Non-Color (필수 변경!)
  • Normal → Non-Color (필수 변경!)
sRGB = "사람 눈이 보는 색" / Non-Color = "데이터 값". 거칠기나 요철은 색이 아니라 데이터이므로 Non-Color로 해야 정확해요
📖 Blender Manual — Image Texture Node

PBR 3종 맵 노드 연결 다이어그램

Image Texture
Color Space: sRGB → Color 출력 → Base Color 🎨 Diffuse 맵
Image Texture
Color Space: Non-Color → Color 출력 → Roughness ⬛ Roughness 맵
Image Texture
Color Space: Non-Color → Normal Map → Normal 🔵 Normal 맵
주의: Normal 맵은 직접 Normal에 연결하면 안 돼요! 반드시 중간에 Normal Map 노드를 거쳐야 합니다. 또한 Roughness와 Normal 모두 Color Space를 Non-Color로 바꿔야 정상 작동합니다.

유용한 단축키

Ctrl+T Image Texture에 Texture Coordinate + Mapping 자동 연결
Ctrl+Shift+Click 노드 프리뷰 (Viewer 연결)
Shift+A → Texture → Image Texture Image Texture 노드 추가

Poly에서 PBR 맵 다운로드

withpoly.com에서 프롬프트 입력 (예: "rusty painted metal"). Color/Roughness/Normal 3종 다운로드

Shader Editor에 Image Texture 3개 추가

Shift+A → Texture → Image Texture를 3번. 각각에 다운받은 이미지 할당 (Open 클릭)

Color Space 설정

Diffuse는 sRGB(기본값). Roughness와 Normal은 반드시 Non-Color로 변경!

연결: Diffuse → Base Color

첫 번째 Image Texture의 Color 출력 → Principled BSDF의 Base Color 입력

연결: Roughness → Roughness

두 번째 Image Texture의 Color 출력 → Principled BSDF의 Roughness 입력

연결: Normal → Normal Map → Normal

Shift+A → Vector → Normal Map 추가. Image Texture Color → Normal Map Color, Normal Map Normal → BSDF Normal

결과 확인

Z → Rendered 모드에서 확인. 이미지가 늘어나 보이면 정상 — UV가 없어서예요 (Week 7에서 해결)
