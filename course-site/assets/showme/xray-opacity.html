X-Ray X-Ray 모드란?

오브젝트를 반투명하게 보여주는 뷰포트 표시 모드입니다. 내부 구조, 뒤쪽 폴리곤, 겹친 엣지까지 모두 보이게 해줍니다.

마치 X선 촬영처럼 — 표면 뒤에 있는 것까지 비쳐 보여요

Opacity 투명도 슬라이더

X-Ray 모드 ON 시 활성화되는 0.0 ~ 1.0 값입니다. 값이 낮을수록 더 투명해지고, 1.0이면 완전 불투명(X-Ray 효과 없음)에 가깝습니다.

유리의 두께 조절 — 얇을수록 더 많이 비쳐요

위치 어디서 조절하나요?

3D 뷰포트 우측 상단 Viewport Shading 아이콘(●) 클릭 → X-Ray 체크박스 옆 숫자 슬라이더를 드래그하거나 클릭해서 값을 입력합니다.

  • 토글: 아이콘 영역 우측의 체크 ON/OFF
  • 슬라이더: 체크 옆 파란 숫자 드래그
  • 정확한 값: 숫자 더블클릭 후 직접 입력

Opacity 값 의미

0.1 — 거의 투명 내부 구조가 매우 선명하게 보임
0.5 — 반투명 표면과 내부를 동시에 파악 가능
0.7 — 기본값 Blender 기본 설정 (0.723)
1.0 — 불투명 X-Ray 효과 없음, 일반 Solid처럼 보임

0.72

X-Ray 토글과 슬라이더를 조작하면 캔버스가 즉시 반응합니다.

X-Ray Opacity가 필요할 때

  • 내부 Edge Loop 위치 확인
  • 뒤쪽 정점/면 선택
  • 오브젝트 내부에 파묻힌 메쉬 점검
  • 두 오브젝트의 겹침 여부 확인
  • Armature 본이 메쉬 안에 맞는지 확인
  • UV 펼침 전 메쉬 이음새 확인

X-Ray를 끄는 경우

  • 표면 디테일 집중 편집
  • 재질·색상 결과 확인
  • 앞쪽 요소만 선택해야 할 때
  • 최종 결과물 미리보기
  • Sculpt Mode에서 브러쉬 작업
팁: Edit Mode에서 Alt+Z로 X-Ray를 켜면 뒤쪽 정점까지 한 번에 박스 선택(B)할 수 있어요. 반대로 표면만 선택할 때는 꼭 X-Ray를 끄세요.
주의: X-Ray 모드는 렌더링과 무관합니다. 뷰포트 표시 설정일 뿐이므로 최종 렌더 결과에는 영향을 주지 않아요.

함께 쓰면 좋은 조합

X-Ray + 박스 선택 (B)
X-Ray ON 상태에서 B로 드래그하면 앞뒤 정점을 모두 선택. 빠른 전체 영역 선택에 최적.
X-Ray + Mirror Modifier
대칭 모델에서 내부 중심선 정점 확인. Clipping 적용 여부를 투명하게 시각화.
X-Ray + Weight Paint
Armature 본이 메쉬 내부에서 올바른 위치에 있는지 확인하며 웨이트 조절.
낮은 Opacity + 레퍼런스 이미지
배경 이미지와 메쉬 내부 구조를 동시에 확인하여 정밀한 형태 맞추기에 유리.
