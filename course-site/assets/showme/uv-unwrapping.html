UV Unwrapping이란?

3D 메쉬의 표면을 2D 평면으로 펼치는 과정입니다. 텍스처(이미지)를 3D 모델에 올바르게 입히려면, 각 면이 2D 이미지의 어느 부분에 대응하는지 정의해야 합니다. 이때 U, V 좌표계를 사용합니다.

종이 상자를 뜯어서 평면으로 펼치는 것과 같습니다 — 3D를 2D로 만드는 과정이에요

Seam(재단선)이란?

UV를 펼칠 때 "어디를 잘라야 하는지" 알려주는 선입니다. Seam의 위치에 따라 UV Island(섬)의 모양과 개수가 달라집니다. 적절한 Seam 배치가 깨끗한 UV의 핵심입니다.

옷을 만들 때 천을 자르는 재단선 — 어디를 자르느냐에 따라 결과물이 달라져요

Seam 배치 방식을 선택하여 UV 펼침 결과가 어떻게 달라지는지 확인하세요.

UV Unwrapping이 필요할 때

  • 텍스처 페인팅 (Texture Paint)
  • 이미지 텍스처 매핑
  • 게임 에셋 제작
  • Baking (Normal Map, AO 등)

UV가 덜 중요한 경우

  • Procedural 텍스처만 사용
  • 단색 재질 (Color만 적용)
  • 간단한 프리뷰/스케치
팁: Smart UV Project는 빠르지만 결과가 깔끔하지 않을 수 있습니다. 중요한 모델은 직접 Seam을 잡고 U → Unwrap을 사용하세요. Seam은 눈에 잘 안 보이는 곳(겨드랑이, 뒤통수 등)에 배치하는 것이 좋습니다.

자주 쓰는 워크플로우

Mark Seam → Unwrap → UV Editor 정리
Edit Mode에서 Seam을 잡고, U키로 Unwrap한 뒤, UV Editor에서 Island를 정렬합니다.
Smart UV Project → 수동 보정
복잡한 모델은 먼저 Smart UV Project로 대략 펼친 뒤, 문제 부분만 수동으로 Seam을 잡아 다시 Unwrap합니다.
