Core Weld란?

가까운 꼭짓점들을 자동으로 병합하는 모디파이어입니다. 중복 정점 제거, 이음새 연결에 사용합니다.

가까이 있는 점들을 자석처럼 합치는 것

Setting Distance

이 거리 이내의 꼭짓점들이 하나로 병합됩니다. 너무 크게 설정하면 의도치 않게 형태가 무너질 수 있습니다.

  • Distance 작음 → 아주 가까운 점만 병합
  • Distance 큼 → 더 먼 점까지 병합 (주의!)

Option Vertex Group

특정 Vertex Group의 꼭짓점만 Weld를 적용할 수 있습니다. 부분적으로 중복 정점을 정리할 때 유용합니다.

관련 단축키

M → D Merge by Distance (파괴적 방식)
Ctrl + 1 Modifier 추가 패널
📖 Blender 공식 문서 — Weld Modifier
연속 조절 Distance 슬라이더는 병합 임계값을 연속적으로 바꾸고, 초기화 버튼은 기준 거리로 되돌립니다.
Weld는 면을 새로 만드는 기능이 아니라, 일정 거리 안에 있는 꼭짓점을 같은 점으로 간주해 합치는 기능입니다. Distance가 커질수록 더 공격적으로 병합됩니다.

Too Much Weld

거리를 너무 크게 올리면 의도한 구조까지 한 점으로 합쳐져서 실루엣이 무너지거나 구멍이 막히는 문제가 생깁니다.

Clean Seam

거리를 아주 조금씩 올리며 이음새만 정리하면 Mirror 중심선, Array 연결부, Boolean 후처리를 비파괴적으로 깔끔하게 만들 수 있어요.

0.25

Distance를 키우면 더 먼 점들도 병합됩니다. 너무 크면 형태가 무너져요!

Weld가 필요할 때

  • Mirror 이음새 정리
  • Boolean 후 중복 정점 제거
  • Import 모델 정리
  • Array 연결부 병합

Weld를 안 쓰는 경우

  • 의도적으로 분리된 메쉬
  • 거리 설정 실수 시 형태 붕괴 위험
  • 이미 깨끗한 토폴로지

자주 쓰는 조합

Boolean → Weld
Boolean 연산 후 교차 부분에 생기는 중복 정점과 이음새를 Weld로 정리합니다.
Array → Weld
Array로 반복 생성된 오브젝트의 연결부에서 겹치는 꼭짓점을 병합합니다.
팁: Merge by Distance(M→D)와 비슷하지만, Weld는 비파괴적! 언제든 Distance를 조절할 수 있어요.
